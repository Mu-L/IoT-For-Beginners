# தொடக்கர்களுக்கான IoT - பாடத்திட்டம்

Microsoft இல் Azure Cloud Advocates IoT அடிப்படைகள் பற்றிய 12 வாரம், 24 பாடம் கொண்ட பாடத்திட்டத்துடன் மகிழ்ச்சியுடன் வழங்குகின்றனர். ஒவ்வொரு பாடத்திலும் முன் மற்றும் பிந்தைய வினா பதில் பாடவினாக்கள், பாடத்தை முடிக்க எழுதப்பட்ட வழிகாட்டுதல்கள், ஒரு தீர்வு, ஒரு பணிப்பணி மற்றும் மேலும் உட்பட உள்ளது. எங்கள் திட்டவியல் வழிமுறை நீங்கள் உருவாக்கும் போது கற்றுக்கொள்ள உதவுகிறது, இது புதிய திறன்கள் நிலைத்திருக்க உதவும் என்பதில் நிரூபிக்கப்பட்டது.

இந்த திட்டங்கள் உணவின் பயணத்தை தோட்டத்திலிருந்து மேசைக்கும் வரையிலும் கொண்டுள்ளது. இதில் விவசாயம், லாஜிஸ்டிக்ஸ், உற்பத்தி, சில்லறை விற்பனை மற்றும் வாடிக்கையாளர் ஆகியவை உள்ளன - அனைத்தும் IoT சாதனங்களுக்கு பிரபலமான தொழிற்துறை பகுதிகள் ஆகும்.

## கல்வி முறைகள்

இந்தப் பாடத்திட்டத்தை உருவாக்கும் போது இரண்டு கல்வி மூல நெறிகள் தேர்ந்தெடுக்கப்பட்டுள்ளன: இது திட்டவரிசையாக இருத்தல் மற்றும் அடிக்கடி வினா பதில் பாடவினாக்களைச் சேர்த்தல். இந்த தொடர் இறுதியில், மாணவர்கள் ஒரு தாவரங்களை கண்காணிக்கும் மற்றும் நீர் ஊட்டும் அமைப்பு, ஒரு வாகன நிலைத்தடம், உணவை கண்காணிக்க மற்றும் பரிசோதிக்க ஒரு புத்திசாலி தொழிற்சாலை அமைப்பு மற்றும் குரல் கட்டுப்படுத்தும் சமையல் நேர அமைப்புகளை உருவாக்கி, IoT அடிப்படைகள் உட்பட குறியீடு எழுதுவது, மேகத் தோராயம், டெலிமெடரி பகுப்பாய்வு மற்றும் எல்லையில் AI இயக்குவது ஆகியவற்றையும் கற்கின்றனர்.

தினசரி உள்ளடக்கம் திட்டங்களுடன் பொருந்துவதால், மாணவர்களுக்கு சுவாரஸ்யமாகவும், கருத்துக்களை நிலைநிறுத்த உதவுகிற வண்ணமாகவும் செய்முறை அமைக்கப்படுகிறது.

மேலும், ஒரு வகுப்புக்கு முன் குறைந்த சிரம வினா மாணவ மரபை நோக்கி கவனத்தை செலுத்துகிறது, வகுப்புக்கு பின் இரண்டாவது வினா மேலதிக நினைவாற்றலை உறுதிப்படுத்துகிறது. இந்த பாடத்திட்டம் விருப்பமாகவும், சுவாரஸ்யமாகவும் வடிவமைக்கப்பட்டுள்ளது; முழுமையாகவும், ஒரு பகுதியை மட்டும் எடுத்துக் கொள்ளவும் இயலும். திட்டங்கள் சிறு அளவில் துவங்கி, 12 வாரங்கள் முடிவில் அதிக சிக்கலானதாக்கியும் விரிவடைகிறது.

ஒவ்வொரு திட்டமும் மாணவர்களுக்கும் ஆர்வலர்களுக்கும் கிடைக்கும் உண்மையான வன்பொருளை மையமாகக் கொண்டு உள்ளது. ஒவ்வொரு திட்டமும் குறிப்பிட்ட துறையில் கவனம் செலுத்தி, பொருத்தமான பின்னணி அறிவை வழங்குகிறது. ஒரு சிறந்த டெவலப்பர் ஆக, நீங்கள் எதிர்கொள்ளும் பிரச்சனைகள் எந்த துறைக்கு சொந்தமானவை என்பதைக் அறிதல் உதவும்; இந்த பின்னணி அறிவு மாணவர்களுக்கு IoT தீர்வுகளை மற்றும் கற்றல்களை உண்மையான உலக பிரச்சனைகளின் சூழலில் எண்ண உதவும். மாணவர்கள் தங்களது தீர்வுகளின் 'ஏன்' என்பதை அறிந்து, இறுதி பயனாளரை மதிப்பீடு செய்கின்றனர்.

## வன்பொருள்
நாங்கள் தனிப்பட்ட விருப்பம், நிரலாக்க மொழி அறிவு அல்லது விருப்பங்கள், கற்றல் இலக்குகள் மற்றும் கிடைக்கும் வசதிகளின் அடிப்படையில் திட்டங்களுக்கு பயன்படுத்த இரண்டு வகையான IoT ஹார்ட்வேர்களை தேர்வு செய்துள்ளோம். ஹார்ட்வேருக்கு அணுகல் இல்லாதவர்கள் அல்லது வாங்குவதற்கு முன்பு மேலும் கற்றுக்கொள்ள விரும்புவோர் பயன்பாட்டிற்காக 'மெய்நிகர் ஹார்ட்வேர்வersion' ஐவமும் வழங்கியுள்ளோம். முழு கிட்டுகளை எங்கள் நண்பர்கள் Seeed Studio இல் இருந்து வாங்குவதற்கான இணைப்புகள் உட்பட நீங்கள் மேலதிக தகவல்களை மற்றும் 'ஷாப்பிங் லிஸ்ட்' ஐ [hardware page](./hardware.md) இல் படிக்கலாம்.

## অফ்லைன் அணுகல்

[Docsify](https://docsify.js.org/#/) ஐ பயன்படுத்தி இந்த ஆவணத்தை ஆன்லைனுக்கு வெளியே இயக்கலாம். இந்த ரெப்போவை ஃபோர்க் செய்து, உங்கள் உள்ளூர் இயந்திரத்தில் [Docsify ஐ நிறுவுங்கள்](https://docsify.js.org/#/quickstart), பின்னர் இந்த ரெப்போவின் மூலக் கோப்பகத்தில் `docsify serve` என தட்டச்சு செய்யுங்கள். இணையதளம் உங்கள் உள்ளூர் கணினியில் பின் 3000ல் நடைபெறும்: `localhost:3000`.

## வினோதடி

### PDF

ஆன்லைன் வெளியே அணுகுவதற்காக இந்த உள்ளடக்கத்தின் PDF ஐ உருவாக்கலாம். இதற்கு [npm நிறுவப்பட்டுள்ளதை](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) உறுதி செய்து, இந்த ரெப்போவின் மூல கோப்பகத்தில் கீழ்காணும் கட்டளைகளை இயக்கவும்:

```sh
npm i
npm run convert
```
