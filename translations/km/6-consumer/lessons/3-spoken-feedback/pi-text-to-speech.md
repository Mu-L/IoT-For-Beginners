    ```python
    ssml += f'<voice xml:lang=\'{language}\' name=\'{voice}\'>'
    ssml += text
    ssml += '</voice>'
    ssml += '</speak>'
    ```

    SSML бһ“бҹҒбҹҮбһҖбҹҶбһҺбһҸбҹӢбһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„бһҹбҹҶбһЎбҹҒбһ„бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ бһҮбһ¶бһҳбһҪбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹҶбһӣбҹӮбһ„бҹ”

1. бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҹбҹҶбһҺбһҫ REST бһ“бһ·бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһјбһҢбһёбһҷбҹ„бһ”бҹҠбһёбһҺбһ¶бһҡбһёбҹ–

    ```python
    response = requests.post(url, headers=headers, data=ssml.encode('utf-8'))
    return io.BytesIO(response.content)
    ```

### бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡ - бһӣбҹҒбһ„бһҹбһҳбҹ’бһӣбҹҒбһ„

1. бһ…бһ»бҹҮбһҖбҹ’бһҡбҹ„бһҳбһҳбһ»бһҒбһ„бһ¶бһҡ `get_speech` бһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһӣбҹҒбһ„бһҹбһҳбҹ’бһӣбҹҒбһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһ–бһёбһҖбһ¶бһҡбһ бҹ… REST APIбҹ–

    ```python
    def play_speech(speech):
    ```

1. бһҹбһҳбҹ’бһӣбҹҒбһ„ `speech` бһҠбҹӮбһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһјбһҢбһёбһҷбҹ„бһ”бҹҠбһёбһҺбһ¶бһҡбһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһ–бһё REST APIбҹ” бһ”бҹ’бһҡбһҫбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбһңбһ¶бһҮбһ¶бһҜбһҖбһҹбһ¶бһҡ wave бһ бһҫбһҷбһ•бҹ’бһүбһҫбһңбһ¶бһ‘бҹ… PyAudio бһҠбһҫбһҳбҹ’бһ”бһёбһӣбҹҒбһ„бһҹбһҳбҹ’бһӣбҹҒбһ„бҹ–

    ```python
    def play_speech(speech):
        with wave.open(speech, 'rb') as wave_file:
            stream = audio.open(format=audio.get_format_from_width(wave_file.getsampwidth()),
                                channels=wave_file.getnchannels(),
                                rate=wave_file.getframerate(),
                                output_device_index=speaker_card_number,
                                output=True)

            data = wave_file.readframes(4096)

            while len(data) > 0:
                stream.write(data)
                data = wave_file.readframes(4096)

            stream.stop_stream()
            stream.close()
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫ PyAudio stream бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҷбһҖбһҹбһҳбҹ’бһӣбҹҒбһ„бҹ” бһ…бҹҶбһ“бһ»бһ…бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһӮбһә stream бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҹбҹ’бһҸбҹ’бһҡбһёбһҳбһ»бһҒбһ…бҹҒбһү (output stream) бһ бһҫбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһӣбҹҒбһ„ бһ бһҫбһҷбһ”бһүбҹ’бһ…бҹҒбһүбһ‘бҹ… streamбҹ”

    бһҳбһ·бһ“бһҠбһ¶бһҖбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ stream бһҠбһјбһ…бһҮбһ¶бһўбһҸбҹ’бһҡбһ¶бһӮбҹҶбһҡбһј (sample rate) бһҮбһ¶ hard code бһ‘бҹҒ бһҸбһ¶бһҳбһҳбҹ’бһӣбҹҒбҹҮбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһјбһҢбһёбһҷбҹ„бҹ”

1. бһҮбҹҶбһ“бһҪбһҹбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһҳбһ»бһҒбһ„бһ¶бһҡ `say` бһҮбһ¶бҹ–

    ```python
    speech = get_speech(text)
    play_speech(speech)
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бҹҶбһӣбҹӮбһ„бһўбһҖбҹ’бһҹбһҡбһ‘бҹ…бһҹбһјбһҡ бһҮбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһјбһҢбһёбһҷбҹ„бһ”бҹҠбһёбһҺбһ¶бһҡбһё бһ бһҫбһҷбһӣбҹҒбһ„бһҹбһҳбҹ’бһӣбҹҒбһ„бҹ”

1. бһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ бһҫбһҷбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһ•бһ„бһҠбҹӮбһҡбҹ” бһҖбҹҶбһҺбһҸбҹӢбһҳбҹүбҹ„бһ„бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһӣбһәбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹ„бһҷбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҗбһ¶бһҳбҹүбҹ„бһ„бһ”бһ¶бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ бһҫбһҷ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҳбһ¶бһ“бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ“бҹ…бһ–бҹҒбһӣбһҳбҹүбҹ„бһ„бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбҹ”

    бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹ `Invalid sample rate` бһҹбһјбһҳбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҡбһјбһ”бһҳбһ“бҹ’бһҸ `playback_format` бһҠбһјбһ…бһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҒбһ¶бһ„бһӣбһҫбҹ”

> рҹ’Ғ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһјбһҠбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ [code-spoken-response/pi](../../../../../6-consumer/lessons/3-spoken-feedback/code-spoken-response/pi)бҹ”

рҹҳҖ бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҳбҹүбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ!
