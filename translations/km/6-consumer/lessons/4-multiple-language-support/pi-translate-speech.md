### бһҖбһ¶бһҡбһ„бһ¶бһҡ - бһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“ translator бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҖбһ”бҹ’бһҡбҹӮбһўбһҸбҹ’бһҗбһ”бһ‘

    бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `<server language>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҸбҹҶбһ”бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ LUISбҹ”

    URL бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ API бһ“бҹҒбҹҮбһҳбһ·бһ“бһ•бҹ’бһҠбҹ„бһҸбһҸбҹҶбһ”бһ“бҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһ‘бҹҒ бһҸбһ¶бһҳбһ–бһ·бһҸбһҸбҹҶбһ”бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҠбһӣбҹӢбһҮбһ¶ headerбҹ” бһҖбһјбһ“бһҹбҹ„ API бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҒбһ»бһҹбһ–бһёбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҹбһ»бһ“бҹ’бһ‘бһҡбһҖбһҗбһ¶ бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһ”бһ¶бһ“ access token бһ–бһё token issuer API бһ‘бҹҒбҹ”

1. бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮ бһҖбҹҶбһҺбһҸбҹӢбһҹбҹҶбһҺбһ»бҹҶбһ”бҹӮбһ”бһ”бһ‘бһ“бһ·бһ„бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бҹ–

    ```python

        'from': from_language,
    ```

    `params` бһҖбҹҶбһҺбһҸбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбҹӮбһӣбһ•бҹ’бһҠбһӣбҹӢбһ‘бҹ…бһҖбһ¶бһҡбһ бҹ… API бһ–бһёбһ—бһ¶бһҹбһ¶ from бһ‘бҹ…бһ—бһ¶бһҹбһ¶ toбҹ” бһҖбһ¶бһҡбһ бҹ…бһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бһҖбһ”бҹ’бһҡбҹӮбһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹ…бһ—бһ¶бһҹбһ¶ `from` бһ‘бҹ…бһ—бһ¶бһҹбһ¶ `to`бҹ”

    `body` бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһҖбһ”бҹ’бһҡбҹӮбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһўбһ¶бһҡбҹүбҹҒ бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһўбһҸбҹ’бһҗбһ”бһ‘бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ бҹ…бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

1. бһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһ бҹ… REST API бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ–
