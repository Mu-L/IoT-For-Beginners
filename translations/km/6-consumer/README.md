# អ៊ិនធឺណិត​អ្នកប្រើប្រាស់ - បង្កើតជំនួយការអូស្យុងឆ្លាតវៃ

ម្ហូបត្រូវបានដាំបន់ ក្រោយមកបានដឹកទៅរោងចក្រ ដាក់ចំណាត់ថ្នាក់គុណភាព លក់នៅហាង ហើយឥឡូវនេះគឺពេលវេលាសម្រាប់ចម្អិន! មួយក្នុងចំណោមបំណែកសំខាន់នៃផ្ទះបាយគឺម៉ោងកំណត់ពេល។ នៅដើមវាត្រូវបានប្រើជាខ្សែម៉ោងខ្សាច់ - ម្ហូបរបស់អ្នកត្រូវបានចម្អិនពេលខ្សាច់ទាំងអស់រាបសារ​ចុះដល់បង្កងខាងក្រោម។ បន្ទាប់មកវាប្រើម៉ាស៊ីនដំណើរការ ម៉ាស៊ីនអគ្គិសនី។

ការប្តូរថ្មីៗនេះបានបញ្ចូលជាផ្នែកមួយនៃឧបករណ៍ឆ្លាតវៃរបស់យើង។ នៅផ្ទះបាយក្នុងគ្រប់គ្រឺងពិភពលោក អ្នកនឹងឮអ្នកចម្អិនច្រៀងថា "Hey Siri - set a 10 minute timer", ឬ "Alexa - cancel my bread timer"។ មិនបាច់ត្រូវផ្លាស់ទៅតាមផ្ទះបាយសម្រាប់ពិនិត្យម៉ោងកំណត់ទៀតទេ អ្នកអាចធ្វើបានពីទូរស័ព្ទរបស់អ្នក ឬហៅផ្សាយថ្លៃពេញបន្ទប់។

ក្នុងមេរៀនទាំង ៤ នេះ អ្នកនឹងរៀនពីរបៀបបង្កើតម៉ោងកំណត់ឆ្លាតវៃ ដោយប្រើ AI ដើម្បីស្គាល់សំឡេងរបស់អ្នក យល់អំពីអ្វីដែលអ្នកស្នើរសុំ និងឆ្លើយតបជាមួយព័ត៌មានអំពីម៉ោងកំណត់របស់អ្នក។ អ្នកនឹងបន្ថែមការគាំទ្រភាសាច្រើនផងដែរ។

> ⚠️ ការប្រើប្រាស់ទិន្នន័យសំឡេង និងរ៉ែគ្រឿងថតសំឡេងប្រើប្រាស់កម្រិតអង្គចងចាំច្រើន ដែលមានន័យថាមានការលំបាកក្នុងការបំពានកម្រិតលើម៉ាស៊ីនតូចៗ។ គម្រោងនេះបានកែច្នៃដើម្បីដោះស្រាយបញ្ហាទាំងនេះ ប៉ុន្តែគិតគូរថាលាប Wio Terminal មានភាពស្មុគស្មាញ ហើយប្រហែលជារយៈពេលច្រើនជាងមេរៀនផ្សេងទៀតក្នុងកម្មវិធីសិក្សានេះ។

> 💁 មេរៀនទាំងនេះនឹងប្រើប្រាស់ធនធានពពកខ្លះ។ បើអ្នកមិនបានបញ្ចប់មេរៀនទាំងអស់ក្នុងគម្រោងនេះ សូមប្រាកដថាអ្នកបាន [សំអាតគម្រោងរបស់អ្នក](../clean-up.md)។
