### бһ”бҹүбһ¶бһҡбһҳбҹүбһ„бҹӢ - бһ”бһҳбҹ’бһӣбҹӮбһ„бһҖбһјбһҠбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бҹ…бһҖбҹҶбһҺбһҸбҹӢбһңбһҸбҹ’бһҗбһ»

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһ“бҹӮбһҖ include бһҒбһ¶бһ„бһӣбһҫбһ“бҹғбһҜбһҖбһҹбһ¶бһҡ `main.cpp`бҹ–


1. бһҠбһјбһҡбһҲбҹ’бһҳбҹ„бҹҮбһҳбһ»бһҒбһ„бһ¶бһҡ `classifyImage` бһ‘бҹ…бһҮбһ¶ `detectStock` бһ‘бһ¶бҹҶбһ„бһҲбҹ’бһҳбҹ„бҹҮбһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `buttonPressed`бҹ”

1. бһ“бҹ…бһӣбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `detectStock` бһҗбҹ’бһӣбҹӮбһ„бһўбһҗбҹҒбһҡ threshold бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ…бҹҒбһүбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бһ”бҹ—бҹ–

    ```cpp
    const float threshold = 0.3f;
    ```

    бһҒбһ»бһҹбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бҹҶбһҡбһ„бҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһҸбҹ’бһҡбһЎбһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҶбҹ’бһ“бһ¶бҹҶбһ„, бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбҹҶбһҺбһҸбҹӢбһңбһҸбҹ’бһҗбһ»бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһ бҹӮбһӣбһҗбһ¶бһҗбһҷбһҖбҹ’бһҡбҹ„бһҳбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бҹ’бһҡбһ¶бһ“бһ…бҹ„бһӣбҹ”

1. бһ“бҹ…бһӣбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `detectStock` бһҗбҹ’бһӣбҹӮбһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбҹ–

    ```cpp
    void processPredictions(std::vector<JsonVariant> &predictions)
    {
        for(JsonVariant prediction : predictions)
        {
            String tag = prediction["tagName"].as<String>();
            float probability = prediction["probability"].as<float>();
    
            char buff[32];
            sprintf(buff, "%s:\t%.2f%%", tag.c_str(), probability * 100.0);
            Serial.println(buff);
        }
    }
    ```

    бһңбһ¶бһ‘бһ‘бһҪбһӣбһ”бһүбҹ’бһҮбһёбһ“бҹғбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ бһ бһҫбһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһңбһ¶бһ‘бҹ…бһҳбҹ’бһҮбһ»бһӣбһҶбҹ’бһӣбһ„бҹ”

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `detectStock` бһҮбҹҶбһ“бһҪбһҹбһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһ”бһҹбҹӢ `for` loop бһҠбҹӮбһӣбһҡбһ»бһүбһҸбһ¶бһҳбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҳбһҪбһҷбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```cpp
    std::vector<JsonVariant> passed_predictions;

    for(JsonVariant prediction : predictions) 
    {
        float probability = prediction["probability"].as<float>();
        if (probability > threshold)
        {
            passed_predictions.push_back(prediction);
        }
    }

    processPredictions(passed_predictions);
    ```

    бһңбһ¶бһҡбһ»бһүбһҸбһ¶бһҳбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҳбһҪбһҷ thresholdбҹ” бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһ бҹӮбһӣбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„ threshold бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ `list` бһ бһҫбһҷбһ•бҹ’бһүбһҫбһ‘бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ `processPredictions`бҹ”

1. бһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһӣбһҫбһңбһҸбҹ’бһҗбһ»бһӣбһҫвҖӢбһҸбһ»бһ бһҫбһҷбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ Cбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ“бҹ’бһҸбҹ’бһҡбһёбһҹбҹҠбҹҒбһҡбһёбһӣбҹ–
