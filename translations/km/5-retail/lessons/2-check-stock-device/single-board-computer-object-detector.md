### бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ - бһ…бһҳбҹ’бһӣбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–

    * бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Raspberry Pi бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбҹҶбһЎбһҫбһ„ PiCameraбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ„бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶вҖӢбһ“бҹ…бһ‘бһёбһҸбһ¶бҹҶбһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбһјбһ…бһҮбһ¶бһҠбһ¶бһҖбҹӢбһҒбҹ’бһҹбҹӮбһҳбһ¶бһ“бҹӢбһӣбһҫбһ”бҹ’бһҡбһўбһ”бҹӢ бһ¬бһҖбҹӮбһ„ бһ¬бһҖбҹҸбһҠбҹҶбһЎбһҫбһ„бһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶вҖӢбһҮбһ¶бһҳбһҪбһҷбһҸбһ¶бһ”бҹҗбһҡбһ–бһёбһҡвҖӢбһ•бҹ’бһӣбһјбһңбһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһўбһ”бҹӢбҹ”
    * бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбҹҶбһЎбһҫбһ„ CounterFit бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё CounterFit PyCameraбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҲбһҡ бһҹбһјбһҳбһҗбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҒбҹ’бһӣбҹҮбһҠбҹӮбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҖбһҘбһңбҹүбһ¶бһ“бҹӢбһҳбһ·бһ“бһ’бҹ’бһӣбһ¶бһ”бҹӢбһғбһҫбһүбһ‘бҹҒ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһңбҹӮбһ” бһҹбһјбһҳбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһ“бҹҒбҹҮбһ“бҹ…бһ‘бһёбһҸбһ¶бҹҶбһ„бһҠбҹӮбһӣбһўбһ¶бһ…бһғбһҫбһүбһҘбһңбҹүбһ¶бһ“бҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҡбһҖбһғбһҫбһүбҹ”

1. бһ’бҹ’бһңбһҫбһЎбһҫбһ„бһңбһ·бһүбһҮбҹҶбһ бһ¶бһ“бһ–бһё [бһҳбҹҒбһҡбҹҖбһ“ 2 бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ](../../../4-manufacturing/lessons/2-check-fruit-from-device/README.md#task---capture-an-image-using-an-iot-device) бһҠбһҫбһҳбҹ’бһ”бһёбһҗбһҸбһҡбһјбһ”бһ–бһёбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бҹ”

1. бһ’бҹ’бһңбһҫбһЎбһҫбһ„бһңбһ·бһүбһҮбҹҶбһ бһ¶бһ“бһ–бһё [бһҳбҹҒбһҡбҹҖбһ“ 2 бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ](../../../4-manufacturing/lessons/2-check-fruit-from-device/README.md#task---classify-images-from-your-iot-device) бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ…бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһҖбһјбһҠбһ“бҹҒбҹҮбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбһҘбһңбҹүбһ¶бһ“бҹӢбҹ”

## бһ”бҹ’бһҸбһјбһҡбһҖбһјбһҠбһ–бһёбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҖбһҡбһјбһ”бһ—бһ¶бһ–

бһҖбһјбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһҹбҹ’бһҡбһҠбҹҖбһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ“бһ№бһ„бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбһҘбһңбҹүбһ¶бһ“бҹӢбҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһӮбһәбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҠбҹӮбһӣбһ бҹ…бһ“бҹ…бһӣбһҫ Custom Vision SDK бһ бһҫбһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҖбһ¶бһҡбһ бҹ…бҹ”

### бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ - бһ”бҹ’бһҸбһјбһҡбһҖбһјбһҠбһ–бһёбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҖбһҡбһјбһ”бһ—бһ¶бһ–

1. бһӣбһ»бһ”бһ”бһёбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһјбһҠбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҮбһ¶бҹ–

    ```python
    results = predictor.classify_image(project_id, iteration_name, image)
    
    for prediction in results.predictions:
        print(f'{prediction.tag_name}:\t{prediction.probability * 100:.2f}%')
    ```

    бһӣбһ»бһ”бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ”бһёбһ“бҹҒбҹҮбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһҘбһңбҹүбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–бҹ–

    ```python
    results = predictor.detect_image(project_id, iteration_name, image)

    threshold = 0.3
    
    predictions = list(prediction for prediction in results.predictions if prediction.probability > threshold)
    
    for prediction in predictions:
        print(f'{prediction.tag_name}:\t{prediction.probability * 100:.2f}%')
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ бҹ…бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ `detect_image` бһ“бҹ…бһӣбһҫ predictor бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҖбһҘбһңбҹүбһ¶бһ“бҹӢбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһңбһ¶бһ”бҹ’бһҡбһҳбһјбһӣбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһӣбһҫбһҹбһҖбһҳбҹ’бһҡбһ·бһҸбһ„бһ¶бһҷбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ бһ бһҫбһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ–бһҪбһҖбһңбһ¶бһ…бҹҒбһүбһ“бҹ…бһӣбһҫ consoleбҹ”

    бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–вҖӢбһҠбҹӮбһӣбһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһӣбһ¶бһҖбһҳбһҪбһҷбҹ— бһўбҹ’бһ“бһҖбһҡбһҖбһҘбһңбҹүбһ¶бһ“бҹӢбһўбһ¶бһ…бһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бҹ’бһҡбһҫбһ“бһ”бһ¶бһ“ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ¶бһ”бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбһҳбҹ’бһҡбһ„бһ…бҹҒбһүбҹ”
