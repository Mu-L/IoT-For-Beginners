# ប្រើឧបករណ៍រកមុខវត្ថុរបស់អ្នកនៅលើជ្រុង

## សេចក្ដីណែនាំ

ក្នុងគម្រោងចុងក្រោយ អ្នកបានដាក់អ្នកចាត់ថ្នាក់រូបភាពរបស់អ្នកទៅលើជ្រុង។ ជាដូចគ្នានេះ គោលបំណងជាមួយឧបករណ៍រកមុខវត្ថុរបស់អ្នក គឺដាក់វាជាម៉ូឌែលតូច និងដំណើរការ វានៅលើជ្រុង ដើម្បីចូលប្រើជ្រួលជ្រោយពីឧបករណ៍ IoT របស់អ្នក។
