# бһҖбһ¶бһҡбһўбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GPS - Raspberry Pi

бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹҶбһҷбҹ„бһӮ GPS бһ‘бҹ…бһҖбһ¶бһ“бҹӢ Raspberry Pi бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ–бһёбһңбһ¶бҹ”

## бһ бһ¶бҹҠбһҠбһңбҹӮбһҡбһҳбҹҒ

Raspberry Pi бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹҶбһҷбҹ„бһӮ GPSбҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһӮбһә [бһ§бһ”бһҖбһҡбһҺбҹҚ Grove GPS Air530](https://www.seeedstudio.com/Grove-GPS-Air530-p-4584.html)бҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ GPS бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһҷбҹүбһ¶бһ„бһӣбһҝбһ“ бһ“бһ·бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһё 2 бһ•бҹ’бһ“бҹӮбһҖ - бһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһ…бһҳбҹҒбһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһўбһ„бҹӢбһҸбҹӮбһ“бһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹ„бһҷбһҒбҹ’бһҹбҹӮбһҸбһјбһ…бһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһҡбһӣбһҖбһҖбһ¶бҹҶбһҡбһҹбҹ’бһҳбһёбһ–бһёбһ•бҹ’бһҖбһ¶бһҷбһҷбҹ„бһ„бҹ”

бһ“бҹҒбҹҮбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ UART бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһ”бһүбҹ’бһ…бһјбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GPS бһҸбһ¶бһҳбһҡбһҷбҹҲ UARTбҹ”

1. бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣ Raspberry Pi бһҳбһ·бһ“бһҳбһ¶бһ“бһҗбһ¶бһҳбһ–бһӣ бһҹбһјбһҳбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ…бһ»бһ„бһҒбҹ’бһңбҹҮбһ“бҹғбһҒбҹ’бһҹбҹӮ Grove бһ‘бҹ…бһ”бҹ’бһҡбһўбһ”бҹӢ UART бһҠбҹӮбһӣбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҮбһ¶ **UART** бһ“бҹ…бһӣбһҫ Grove Base hat бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… Piбҹ” бһ”бҹ’бһҡбһўбһ”бҹӢбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ‘бһёбһҸбһ¶бҹҶбһ„бһ“бҹ…бһҮбһҪбһҡбһҳбһҖбһҺбһ¶бһҸбҹӢбһ•бҹ’бһ“бҹӮбһҖбһҖбһҺбҹ’бһҠбһ¶бһӣ бһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһ”бҹӮбһҡбһҹбҹ’бһӣбһ»бһҸбһҖбһ¶бһ”бһјбһҹбҹҒбһўбҹҒбһҹ бһҶбҹ’бһңбҹҒбһ„бһ–бһёбһӮбҹ’бһҳбһ¶бһ“бһ…бҹҶбһ–бҹ„бҹҮбһҹбҹҶбһҺбһ»бҹҶбһ”бҹҠбһҖ USB бһ“бһ·бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһўбҹҠбһёбһ’бһҫбһҺбһ·бһҸбҹ”

1. бһҠбһ¶бһҖбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ GPS бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ„бҹӢбһҸбҹӮбһ“бһ¶бһҒбһ¶бһ”бҹӢбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһІбҹ’бһҷбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҳбҹҒбһғбһ”бһ¶бһ“ - бһӣбҹ’бһўбһ”бһҫбһ“бҹ…бһҖбҹ’бһ”бҹӮбһҡбһ”бһ„бҹ’бһўбһҪбһ…бһ”бһҫбһҖ бһ¬бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ” бһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡ бһҹбҹҶбһЎбҹҒбһ„бһҹбһүбҹ’бһүбһ¶бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһ”бһ¶бһҹбҹӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһўбҹ’бһңбһёбһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮбһўбһ„бҹӢбһҸбҹӮбһ“бһ¶бҹ”

## бһ–бһ·бһ”бҹ’бһҡбһҸбһ·бһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚ GPS

Raspberry Pi бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ GPS бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”

### бһҮбҹҶбһ“бһҪбһҷ - бһ–бһ·бһ”бҹ’бһҡбһҸбһ·бһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚ GPS

бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

1. бһ”бһҫбһҖбһ—бҹ’бһӣбһҫбһ„ Pi бһ бһҫбһҷбһҡбһ„бҹӢбһ…бһ¶бҹҶбһІбҹ’бһҷбһңбһ¶бһ”бһҫбһҖбһҡбһҪбһ…бҹ”

1. бһ§бһ”бһҖбһҡбһҺбҹҚ GPS бһҳбһ¶бһ“ LED 2бһҠбҹӮбһӣ - LED бһ–бһҺбҹҢбһҒбҹҖбһңбһҠбҹӮбһӣбһ—бҹ’бһӣбһәбһ”бһүбҹ’бһ…бһ¶бҹҶбһ„бһ–бҹҒбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“ бһ“бһ·бһ„ LED бһ–бһҺбҹҢбһ”бҹғбһҸбһ„бһҠбҹӮбһӣбһ—бҹ’бһӣбһәбһ”бһүбҹ’бһ…бһ¶бҹҶбһ„бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһңбһ·бһ“бһ¶бһ‘бһёбһ–бҹҒбһӣбһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ•бҹ’бһҖбһ¶бһҷбҹ” бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ LED бһҒбҹҖбһңбһ—бҹ’бһӣбһәбһ–бҹҒбһӣбһ”бһҫбһҖ Piбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮ LED бһ”бҹғбһҸбһ„бһ“бһ№бһ„бһ—бҹ’бһӣбһә - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһ”бһ¶бһ“ бһҹбһјбһҳбһҸбҹҶбһҡбһјбһңбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһўбһ„бҹӢбһҸбҹӮбһ“бһ¶бҹ”

1. бһ”бһҫбһҖ VS Code бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ¬бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҮбҹҶбһ“бһҪбһҷ Remote SSH extensionбҹ”

    > вҡ пёҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбҹ„бһ„бһ‘бҹ… [бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„ бһ“бһ·бһ„бһ”бһҫбһҖ VS Code бһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 1 бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ](../../../1-getting-started/lessons/1-introduction-to-iot/pi.md)бҹ”

1. бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбҹҶбһҺбҹӮбһҗбҹ’бһҳбһёбһ“бҹғ Raspberry Pi бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ Bluetooth бһҳбһ¶бһ“бһҮбһҳбҹ’бһӣбҹ„бҹҮбһҡбһңбһ¶бһ„бһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Bluetooth бһ“бһ·бһ„бһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбһҠбҹӮбһӣ Grove UART бһ”бҹ’бһҡбһҫбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮ бһҹбһјбһҳбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

    1. бһ–бһё Terminal бһҡбһ”бһҹбҹӢ VS Code бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡ `/boot/config.txt` бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `nano` бһҠбҹӮбһӣбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһҖбҹ’бһ“бһ»бһ„ Terminal бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    1. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ¶бһҮбҹҶбһ бһ¶бһ“бһҒбһ¶бһ„бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһ“бһ·бһ„бһ…бҹҒбһүбһ–бһё nano

    1. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Pi бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһЎбһҫбһ„бһңбһ·бһү бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһЎбһҫбһ„бһңбһ·бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code бһ–бҹҒбһӣ Pi бһ”бһ¶бһ“бһ”бһҫбһҖбһЎбһҫбһ„бһңбһ·бһүбҹ”

1. бһ–бһё Terminal бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ•бҹ’бһ‘бҹҮбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ `pi` бһҲбҹ’бһҳбҹ„бҹҮбһҗбһ¶ `gps-sensor`бҹ” бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ“бҹҒбҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҗбһ¶ `app.py`бҹ”

1. бһ”бһҫбһҖбһҗбһҸбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code

1. бһҳбһјбһҢбһ»бһӣ GPS бһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ UART бһҸбһ¶бһҳбһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбһҳбһҪбһҷбҹ” бһҸбһҳбҹ’бһӣбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Pip `pyserial` бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбһ–бһёбһҖбһјбһҠ Python бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `app.py` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

    ```python
    import time
    import serial
    
    serial = serial.Serial('/dev/ttyAMA0', 9600, timeout=1)
    serial.reset_input_buffer()
    serial.flush()
    
    def print_gps_data(line):
        print(line.rstrip())
    
    while True:
        line = serial.readline().decode('utf-8')
    
        while len(line) > 0:
            print_gps_data(line)
            line = serial.readline().decode('utf-8')
    
        time.sleep(1)
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ“бһ¶бҹҶбһ…бһјбһӣбһҳбһјбһҢбһ»бһӣ `serial` бһ–бһёбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Pip `pyserial`бҹ” бһңбһ¶бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһё `/dev/ttyAMA0` - бһҠбҹӮбһӣбһҮбһ¶бһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбһҠбҹӮбһӣ Grove Pi Base Hat бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹ’бһҡбһҖ UART бһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһңбһ¶бһ”бҹ’бһҡбһӣбһјбһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбһ“бҹҒбҹҮбҹ”

    бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡ `print_gps_data` бһҠбҹӮбһӣбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҖбһ¶бһ“бҹӢ consoleбҹ”

    бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҖбһјбһҠбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҮбһ¶бһўбһ“бһ“бҹ’бһҸбһ—бһ¶бһ– бһўбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһўбһ¶бһ…бһ–бһёбһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ¶бһӣбҹӢбһңбһҠбҹ’бһҠбһ бһ¶бһҸбҹӢбҹ” бһңбһ¶бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ `print_gps_data` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бһёбһҳбһҪбһҷбҹ—бҹ”

    бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһўбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҡбһҪбһ… бһңбһ¶бһҠбҹҶбһҺбҹҒбһёбһҡбһҖбҹҶбһҺбһҸбҹӢ бҹЎ бһңбһ·бһ“бһ¶бһ‘бһё бһ бһҫбһҷбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸбҹ”

      UnicodeDecodeError: 'utf-8' codec can't decode byte 0xf1 in position 0: invalid continuation byte

    while True:
        try:
            line = serial.readline().decode('utf-8')
              
            while len(line) > 0:
                print_gps_data()
                line = serial.readline().decode('utf-8')
      
        # бһҳбһ¶бһ“бһҡбҹҶбһ–бҹҒбһ…бһ…бҹғбһҶбһ“бҹ’бһ‘бҹҚбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бҹғбһ‘бһёбһҳбһҪбһҷбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҸбһҪбһўбһҖбҹ’бһҹбһҡбҹ”
        # бһўбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ–бҹҒбһүбһҳбһҪбһҷбһ‘бҹҖбһҸбһ бһҫбһҷбһ”бһ“бҹ’бһҸбҹ”

        except UnicodeDecodeError:
            line = serial.readline().decode('utf-8')
