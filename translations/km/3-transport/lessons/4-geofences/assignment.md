# ផ្ញើការជូនដំណឹងដោយប្រើ Twilio

## សេចក្តីណែនាំ

នៅក្នុងកូដរបស់អ្នករហូតមកនេះ អ្នកបានកត់ត្រាទទឹងទៅកាន់ geofence តែប៉ុណ្ណោះ។ ក្នុងការប្រឡងនេះ អ្នកត្រូវបន្ថែមការជូនដំណឹង មួយក្នុងចំណោមសារអត្ថបទ ឬអ៊ីមែល នៅពេលកូអរដោនេ GPS នៅក្នុង geofence។

Azure Functions មានជម្រើសជាច្រើនសម្រាប់ការចងក្រង រួមទាំងសេវាកម្មភាគីទីបីដូចជា Twilio មួយវេទិកាសារគមនាគមន៍។

* ចុះឈ្មោះសម្រាប់គណនីមួយដោយឥតគិតថ្លៃនៅ [Twilio.com](https://www.twilio.com)
* អានឯកសារពាក់ព័ន្ធនឹងការចងក្រង Azure Functions ទៅ Twilio SMS នៅលើទំព័រ [Microsoft docs Twilio binding for Azure Functions](https://docs.microsoft.com/azure/azure-functions/functions-bindings-twilio?WT.mc_id=academic-17441-jabenn&tabs=python)។
* អានឯកសារពាក់ព័ន្ធនឹងការចងក្រង Azure Functions ទៅ Twilio SendGrid ដើម្បីផ្ញើអ៊ីមែល នៅលើទំព័រ [Microsoft docs Azure Functions SendGrid bindings](https://docs.microsoft.com/azure/azure-functions/functions-bindings-sendgrid?WT.mc_id=academic-17441-jabenn&tabs=python)។
* បន្ថែមការចងក្រងទៅកម្មវិធី Functions របស់អ្នក ដើម្បីទទួលការជូនដំណឹងនៅលើកូអរដោនេ GPS វានៅក្នុងឬក្រៅ geofence - មិនទាំងពីរទេ។
