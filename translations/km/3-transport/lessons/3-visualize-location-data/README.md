бһҸбһ¶бһҡбһ¶бһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһҷбҹүбһ¶бһ„бһӣбһҝбһ“бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹҶбһҺбһҫбһҳбһҠбһёбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ…бҹҶбһҺбһ»бһ…бһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһ№бһҖбһ”бһ¶бһ“бһ”бһҫбһҖбҹ”

бһҸбһ¶бһҡбһ¶бһ„бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҡбҹҖбһ„бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҒбҹ” бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҠбҹӮбһӣбһҸбһ¶бһҳбһҠбһ¶бһ“бһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһўбһ¶бһ…бһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһ¬бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһӣбҹҗбһҸбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҹбһүбҹ’бһүбһ¶бҹ— бһҠбһјбһ…бһҮбһ¶ бһҡбһјбһ”бһҳбҹҒбһғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҗбҹ’бһ„бҹғбһҳбһ¶бһ“бһ–бһ–бһҖ бһ‘бһ№бһҖбһ—бҹ’бһӣбҹҖбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҗбҹ’бһ„бҹғбһ—бҹ’бһӣбҹҖбһ„ бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ” бһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҷбҹүбһ¶бһ„бһ…бҹҶбһҡбһјбһ„ бһ…бҹ’бһҡбһҫбһ“бһҹбҹ’бһӣбһјбһҸбһ”бһјбһҸ бһ“бһ·бһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһӮбһҪбһҡбһұбҹ’бһҷбһҹбһҫбһ…бҹ”

вң… бһӮбһ·бһҸбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһүбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһҸбһҫбһҡбһ”бҹҖбһ”бһҺбһ¶бһҠбҹӮбһӣбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ бһ“бһ·бһ„бһ”бһ¶бһ“бһўбһ¶бһ…бһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹ?

бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһўбһ¶бһ…бһұбҹ’бһҷбһҳбһ“бһ»бһҹбҹ’бһҹбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһңбһ¶бһҹбҹӢбһҳбһҪбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһӣбһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбһӮбһәбһ–бһ·бһ”бһ¶бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҒбһҺбҹҲбһ–бҹҒбһӣбһ–бһ“бҹ’бһӣбһәбһҖбҹ’бһҡбһ бһҳбһ—бҹ’бһӣбһәбһҡбҹҮбһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҳбһ“бһ»бһҹбҹ’бһҹбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бҹ” бһ“бҹ…бһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһӮбһәбһҮбһ¶бһ—бҹ’бһӣбһҫбһ„бһҡбҹҮбһ—бҹ’бһӣбһә!

бһ“бҹ…бһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GPS бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһўбһ¶бһ…бһҮбһ¶бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹ…бһӣбһҫбһ•бҹӮбһ“бһ‘бһёбҹ” бһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһҗбһҷбһ“бҹ’бһҸбһҠбһ№бһҖбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһўбһ¶бһ…бһҮбһҪбһҷбһҖбһҳбҹ’бһҳбһҖбһҡбһ“бҹ…бһ•бҹ’бһ‘бҹҮбһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ бһҠбһҫбһҳбҹ’бһ”бһёбһғбһҫбһүбһ–бҹҒбһӣбһҡбһҗбһҷбһ“бҹ’бһҸбһ“бһ№бһ„бһҳбһҖбһҠбһӣбҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ•бҹӮбһ“бһ‘бһёбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҡбһјбһ”бһ—бһ¶бһ–бһҡбһҗбһҷбһ“бҹ’бһҸбһ“бҹ…бһ‘бһёбһҸбһ¶бҹҶбһ„бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһўбҹҶбһ–бһёбһҳбһ¶бһҸбһ·бһҖбһ¶бһҖбҹ’бһ“бһ»бһ„бһҡбһҗбһҷбһ“бҹ’бһҸ бһ“бҹ„бҹҮбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹ…бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ бһўбһ¶бһ…бһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһҸбһ¶бһҳбһ•бҹӮбһ“бһҖбһ¶бһҡ - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһғбһҫбһүбһҡбһҗбһҷбһ“бҹ’бһҸбһ‘бһ№бһҖбһҖбһҖбһ“бҹ…бһҮбһ·бһҸбһҒбһ¶бһ„ бһӮбҹҒбһҠбһ№бһ„бһҗбһ¶бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ„бһҸбһ¶бҹҶбһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ‘бһјбһҡбһ‘бһ№бһҖбһҖбһҖбҹ”

вң… бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһҳбһҪбһҷбһҖбһјбһҠбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһ•бҹӮбһ“бһ‘бһёбһҒбҹ’бһӣбҹҮбҹ— [mapping code samples](https://docs.microsoft.com/samples/browse?WT.mc_id=academic-17441-jabenn&products=azure-maps)

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһ‘бһёбһҮбһ¶бһӮбҹ’бһҡбһҝбһ„бһ…бһҳбҹ’бһӣбһ„бһҹбҹ’бһ„бһҪбһҸ бһҹбҹ’бһӣбһ¶бһҖ, бһҡбһјбһ”бһ—бһ¶бһ–бһ•бҹӮбһ“бһҠбһё, бһҡбһјбһ”бһ—бһ¶бһ–бһҖбһ¶бҹҶбһҡбһҹбҹ’бһҳбһёбһ•бҹӮбһ“бһҠбһёбһҠбҹӮбһӣбһ•бҹ’бһҹбҹҶбһӣбһҫбһ•бҹ’бһӣбһјбһң, бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹӮбһ“бһ‘бһёбһҹбҹҶбһҺбһӣбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“, бһ•бҹӮбһ“бһ‘бһёбһҳбһ¶бһ“бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһӣбһ¶бһ”бһҡбһӣбҹ„бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһ–бһҹбҹӢ, бһ•бҹӮбһ“бһ‘бһёбһҳбһҫбһӣбһҷбһ”бҹӢ бһ“бһ·бһ„бһ•бҹӮбһ“бһ‘бһёбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһ”бҹ’бһӣбҹӮбһҖбһҒбҹ’бһ–бһҹбҹӢбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ–бҹҒбһӣбһ“бҹҒбҹҮбһӣбһҫбһ•бҹӮбһ“бһ‘бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһјбһӣбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ [Azure Event Grid](https://azure.microsoft.com/services/event-grid/?WT.mc_id=academic-17441-jabenn)бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһҳбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһҡбһјбһ”бһҡбһ¶бһ„бһ•бҹӮбһ“бһ‘бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбҹ„бһҷбһ”бһҫбһҖбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһ•бҹӮбһ“бһ‘бһёбһҺбһ¶бһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ…бһ»бһ… бһ”бһүбҹ’бһ…бҹҒбһүбһҖбҹ’бһ”бһ¶бһӣ бһ“бһ·бһ„бһ‘бһ¶бһүбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһјбһ”бһҡбһ¶бһ„бһ•бҹӮбһ“бһ‘бһё бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ”бһҺбҹ’бһҠбҹ„бһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҠбһјбһ…бһҮбһ¶ бһ”бҹҠбһјбһ”бһјбһӣ бһҒбҹ’бһҹбҹӮ бһ”бһҡбһ·бһңбҹҒбһҺ бһ•бҹӮбһ“бһ‘бһёбһҖбҹҶбһҠбҹ… бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ•бҹӮбһ“бһ‘бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ SDK бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһӣбһҫ API Azure Maps бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ [REST API](https://docs.microsoft.com/javascript/api/azure-maps-rest/?WT.mc_id=academic-17441-jabenn&view=azure-maps-typescript-latest), [Web SDK](https://docs.microsoft.com/azure/azure-maps/how-to-use-map-control?WT.mc_id=academic-17441-jabenn), бһ¬ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһӣбҹҗбһҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ [Android SDK](https://docs.microsoft.com/azure/azure-maps/how-to-use-android-map-control-library?WT.mc_id=academic-17441-jabenn&pivots=programming-language-java-android)бҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫ web SDK бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһјбһҡбһ•бҹӮбһ“бһ‘бһё бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбһ‘бһёбһҸбһ¶бҹҶбһ„ GPS бһҡбһ”бһҹбҹӢбһҹбҹҒбһ“бһҹбҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ Azure Maps

бһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһәбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҺбһ“бһё Azure Mapsбҹ”

### бһҖбһ¶бһҡбһ„бһ¶бһҡ - бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ Azure Maps

1. бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ–бһё Terminal бһ¬ Command Prompt бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ Azure Maps бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“ `gps-sensor` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

    ```sh
    az maps account create --name gps-sensor \
                           --resource-group gps-sensor \
                           --accept-tos \
                           --sku S1
    ```

    бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ Azure Maps бһҲбҹ’бһҳбҹ„бҹҮ `gps-sensor`бҹ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸ `S1` бһҠбҹӮбһӣбһҮбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢ бһ”бһјбһҖбһҡбһҪбһҳбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ бҹ…бһҮбһ¶бһҹбҹҒбһңбһ¶ бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бҹ”

    > рҹ’Ғ бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбҹ’бһӣбҹғбһҲбҹ’бһ“бһҪбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Azure Maps бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бҹ… [бһ‘бҹҶбһ–бҹҗбһҡбһҸбһҳбҹ’бһӣбҹғ Azure Maps](https://azure.microsoft.com/pricing/details/azure-maps/?WT.mc_id=academic-17441-jabenn)бҹ”

1. бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӣбҹҒбһҒбһҖбһјбһҠ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“бһ•бҹӮбһ“бһ‘бһёбһ“бҹҒбҹҮбҹ” бһ”бҹ’бһҡбһҫбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһјбһҠбһ“бҹҒбҹҮбҹ–

    ```sh
    az maps account keys list --name gps-sensor \
                              --resource-group gps-sensor \
                              --output table
    ```

    бһ…бҹҶбһӣбһ„бһҸбһҳбҹ’бһӣбҹғ `PrimaryKey` бҹ”

## бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһ‘бһёбһӣбһҫбһӮбҹҒбһҳбҹҒбһ”бҹ’бһӣбһ»бһҖ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҸбһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбһәбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһ‘бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӣбһҫбһӮбҹҒбһҳбҹҒбһ”бҹ’бһӣбһ»бһҖбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ `html` бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҸбһјбһ…вҖӢвҖӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҮбһ¶бһ”бҹӢбһ…бһ·бһҸбҹ’бһҸбһҗбһ¶ бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ¬бһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ¶бһҖбһҠбһҮбһ¶бһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһ…бһӣбҹҗбһҸбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһ“бҹҒбҹҮ!

### бһҖбһ¶бһҡбһ„бһ¶бһҡ - бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһ‘бһёбһӣбһҫбһӮбҹҒбһҳбҹҒбһ”бҹ’бһӣбһ»бһҖ

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҲбҹ’бһҳбҹ„бҹҮ index.html бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҺбһ¶бһҳбһҪбһҷбһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳ HTML markup бһҠбһҫбһҳбҹ’бһ”бһёбһҗбҹӮбһ‘бһ¶бҹҶбһ•бҹӮбһ“бһ‘бһёбҹ–

    ```html
    <html>
    <head>
        <style>
            #myMap {
                width:100%;
                height:100%;
            }
        </style>
    </head>
    
    <body onload="init()">
        <div id="myMap"></div>
    </body>
    </html>
    ```

    бһ•бҹӮбһ“бһ‘бһёбһ“бһ№бһ„бһ•бҹ’бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `div` `myMap`бҹ” бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһңбһ¶бһҜбһҖбһ‘бҹҒбһҹбһҖбһ¶бһҡбһ–бһ¶бһҡбһ–бҹ’бһҡбһӣбҹӮбһ„бһ‘бһ‘бһ№бһ„бһ“бһ·бһ„бһҖбһҳбҹ’бһ–бһҹбҹӢбһӮбҹҒбһҳбҹҒбһ”бҹ’бһӣбһ»бһҖбҹ”

    > рҹҺ“ `div` бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһӮбҹҒбһҳбҹҒбһ”бҹ’бһӣбһ»бһҖбһҠбҹӮбһӣбһўбһ¶бһ…бһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ”бһ¶бһ“бҹ”

1. бһҖбҹ’бһҡбҹ„бһҳбһҹбҹ’бһӣбһ¶бһҖбһ”бһҫбһҖ `<head>`, бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹ’бһӣбһ¶бһ”бһҹбҹ’бһ‘бҹғбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһ‘бһё бһ“бһ·бһ„бһҹбҹ’бһҖбҹ’бһҡбһёбһ”бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ–бһё Web SDK бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҡбһ”бһҹбҹӢбһңбһ¶бҹ–

    ```html
    <link rel="stylesheet" href="https://atlas.microsoft.com/sdk/javascript/mapcontrol/2/atlas.min.css" type="text/css" />
    <script src="https://atlas.microsoft.com/sdk/javascript/mapcontrol/2/atlas.min.js"></script>
    ```

    бһҹбҹ’бһӣбһ¶бһ”бһҹбҹ’бһ‘бҹғбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ•бҹӮбһ“бһ‘бһёбһҳбһҫбһӣбһ‘бҹ… бһ бһҫбһҷбһҜбһҖбһҹбһ¶бһҡбһҹбҹ’бһҖбҹ’бһҡбһёбһ”бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ•бҹӮбһ“бһ‘бһёбҹ” бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһ“бҹҒбҹҮбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ header files бһҮбһ¶C++ бһ¬бһ“бһ¶бҹҶбһ…бһјбһӣбһҳбҹүбһјбһҢбһ»бһӣ Python бҹ”

1. бһҖбҹ’бһҡбҹ„бһҳбһҹбҹ’бһҖбҹ’бһҡбһёбһ”бһ“бҹ„бҹҮ бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһӣбһ»бһҖбһҹбҹ’бһҖбҹ’бһҡбһёбһ”бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ•бҹӮбһ“бһ‘бһёбҹ”

    ```javascript
    <script type='text/javascript'>
        function init() {
            var map = new atlas.Map('myMap', {
                center: [-122.26473, 47.73444],
                zoom: 12,
                authOptions: {
                    authType: "subscriptionKey",
                    subscriptionKey: "<subscription_key>",

                }
            });
        }
    </script>
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<subscription_key>` бһҮбһ¶бһҳбһҪбһҷбһӣбҹҒбһҒ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһё Azure Maps бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

    бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһҫбһҖбһӮбҹҒбһҳбҹҒбһ”бҹ’бһӣбһ»бһҖ `index.html` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһ‘бһёбһ бһҫбһҷбһ•бҹ’бһҸбҹ„бһҸбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢ Seattleбҹ”

    ![A map showing Seattle, a city in Washington State, USA](../../../../../translated_images/km/map-image.8fb2c53eb23ef39c.webp)

    вң… бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ zoom бһ“бһ·бһ„ center бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҡбһјбһ”бһ—бһ¶бһ–бһ•бҹӮбһ“бһ‘бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бҹҶбһҺбһ»бһ…бһҠбҹӮбһ“бһ‘бҹ’бһңбһ¶бһҡбһҠбҹӮбһӣбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ“бһ№бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ latitude бһ“бһ·бһ„ longitude бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһҳбһ’бҹ’бһҷбһҳ бһ•бҹӮбһ“бһ‘бһёбһЎбһҫбһ„бһңбһ·бһүбҹ”

> рҹ’Ғ бһңбһ·бһ’бһёбһӣбҹ’бһўбһҮбһ¶бһ„бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһӮбһәбһҠбҹҶбһЎбһҫбһ„ [http-server](https://www.npmjs.com/package/http-server)бҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбҹҶбһЎбһҫбһ„ [node.js](https://nodejs.org/) бһ“бһ·бһ„ [npm](https://www.npmjs.com/) бһҳбһ»бһ“бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбҹҶбһЎбһҫбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһҺбҹӮбһ“бһ¶бҹҶбһҜбһҖбһҹбһ¶бһҡ `index.html` бһ бһҫбһҷбһңбһ¶бһҷ `http-server`бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһ“бһ№бһ„бһ”бһҫбһҖбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ”бҹ’бһӣбһ»бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ [http://127.0.0.1:8080/](http://127.0.0.1:8080/)бҹ”

## бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ GeoJSON

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ” бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһ‘бһё бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҠбһҖбһӮбҹ’бһҡбһ¶бһ”бҹӢ GPS бһ–бһёбһӮбһҺбһ“бһёбһҹбҹ’бһҠбһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһ“бҹ…бһӣбһҫбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҳбһҪбһҷбһ“бҹғбһҹбһүбҹ’бһүбһ¶бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹ…бһӣбһҫбһ•бҹӮбһ“бһ‘бһёбһҳбһҪбһҷбҹ” бһҳбһ»бһ“бһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһ”бҹӮбһ”бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҳбһҖбһҳбһҫбһӣбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ [GeoJSON](https://wikipedia.org/wiki/GeoJSON) бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҠбҹ„бһҷ Azure Maps бһҳбһ»бһ“бҹ”

[GeoJSON](https://geojson.org/) бһӮбһәбһҮбһ¶бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ JSON бһ”бһҫбһҖбһ…бҹҶбһ  бһҳбһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһңбһ¶бһҠбҹ„бһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹҶбһҡбһјбһ”бҹ’бһҡбһҫ [geojson.io](https://geojson.io) бһҠбҹӮбһӣбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҖбҹҶбһ бһ»бһҹбһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ GeoJSON бһҸбҹӮбһҳбһҪбһҷбҹ”

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GeoJSON бһӮбҹҶбһҡбһјбһҳбһҫбһӣбһ‘бҹ…бһҠбһјбһ…бһ“бҹҒбҹҮбҹ–

```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "geometry": {
        "type": "Point",
        "coordinates": [
          -2.10237979888916,
          57.164918677004714
        ]
      }
    }
  ]
}
```

бһўбҹ’бһңбһёбһҠбҹӮбһӣбһӮбһҪбһҡбһІбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҮбһ¶бһ–бһ·бһҹбҹҒбһҹ бһӮбһәбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `Feature` бһҖбҹ’бһ“бһ»бһ„ `FeatureCollection`бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһҸбҹ’бһҗбһ»бһ“бҹҒбҹҮ бһўбһ¶бһ…бһҡбһҖбһғбһҫбһү `geometry` бһҠбҹӮбһӣбһҳбһ¶бһ“ `coordinates` бһҹбҹҶбһҠбҹ…бһӣбһҫбһ‘бһ‘бһ№бһ„ бһ“бһ·бһ„бһҡбһҷбҹҲбһ”бһҺбҹ’бһҸбҹ„бһҷбҹ”

вң… бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ geoJSON бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҹбһјбһҳбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢ `latitude` бһ“бһ·бһ„ `longitude` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһҸбҹ’бһҗбһ» бһ”бһҫбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ…бҹҶбһҺбһ»бһ…бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖвҖӢбһ“бһ№бһ„бһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһӮбһҪбһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҒ! GeoJSON бһҹбһ„бҹ’бһғбһ№бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢ `lon,lat` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ»бһ…бһҳбһҪбһҷбҹ— бһҳбһ·бһ“бһҳбҹӮбһ“ `lat,lon`бҹ”
`Geometry` бһўбһ¶бһ…бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бһҮбһ¶бһҗбһ„бҹӢбһҸбҹӮбһҳбһҪбһҷ бһ¬бһ”бҹүбһјбһӣбһёбһ бҹ’бһӮбһ»бһ“бҹ” бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһңбһ¶бһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡвҖӢбһҹбҹ’бһҡбһ»бһҖбһ–бһёбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ бһӮбһәбһӣбһёбһ„бһ бҹ’бһӮбһёбһ‘бһ»бһҠ бһ“бһ·бһ„бһӣбһ¶бһҸбһёбһ‘бһ»бһҠбҹ”

вң… Azure Maps бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбҹҖбһ”бһ…бҹҶ GeoJSON бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ [бһ”бһ“бҹ’бһҗбҹӮбһҳ](https://docs.microsoft.com/azure/azure-maps/extend-geojson?WT.mc_id=academic-17441-jabenn) бһҡбһҪбһҳбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһјбһҹбһҲбһ»бһҸ бһ“бһ·бһ„бһ—бһ¶бһ–бһҷбһ“бҹ’бһҸбһҮбһ¶бһ—бһ¶бһӮбһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

## бһӮбһјбһҹбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GPS бһӣбһҫбһ•бҹӮбһ“бһ‘бһёбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ GeoJSON

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһ бҹҠбһ»бһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһӣбһ№бһҖ бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҮбһ¶бһҖбҹҶбһҺбһҸбҹӢбһ бҹҠбһ»бһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһ‘бһ»бһҖ blob бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ‘бһ¶бһүбһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһ“бһ·бһ„бһ”бҹҶбһ”бҹӮбһҖбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷ Azure Maps бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бҹ”

### бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… - бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһ‘бһ»бһҖбһ–бһёбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ

бһ”бһҫбһўбҹ’бһ“бһҖбһ бҹ…бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҡбһ”бһҹбҹӢбһ•бҹ’бһ‘бһ»бһҖбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ—бҹ’бһүбһ¶бһҖбҹӢбһ•бҹ’бһўбһҫбһӣбһғбһҫбһүбһҖбҹҶбһ бһ»бһҹбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бһ„бһҹбһјбһӣбһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ” бһ–бҹ’бһҡбҹ„бҹҮбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбһ¶бһҖбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ· [CORS](https://developer.mozilla.org/docs/Web/HTTP/CORS) бһӣбһҫбһ•бҹ’бһ‘бһ»бһҖбһ“бҹҒбҹҮ вҖӢбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһҺбҹ’бһҠбһ¶бһүбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһўбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһңбһ¶бһ”бһ¶бһ“бҹ”

> рҹҺ“ CORS бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ "бһҖбһ¶бһҡбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ”бҹ’бһҡбһ—бһ–бһҶбҹ’бһӣбһ„" бһ бһҫбһҷбһ’бһҳбҹ’бһҳбһҸбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ” бһңбһ¶бһ”бһүбҹ’бһҲбһ”бҹӢбһҳбһ»бһҒбһҒбҹ’бһҹбҹӮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҳбһҖбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

1. бһ”бһҫбһҖбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖ CORSбҹ–

    ```sh
    az storage cors add --methods GET \
                        --origins "*" \
                        --services b \
                        --account-name <storage_name> \
                        --account-key <key1>
    ```

    бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `<storage_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҺбһ“бһёбһ•бҹ’бһ‘бһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `<key1>` бһҮбһ¶бһҳбһҪбһҷбһҖбһјбһ“бһҹбҹ„бһӮбһҺбһ“бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһёбһ•бҹ’бһ‘бһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

    бһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҺбһ¶бһҳбһҪбһҷ (бһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷ `*` бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ…бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ) бһўбһ¶бһ…бһҠбһ¶бһҖбҹӢбһҹбҹҶбһҺбһҫ *GET* бһӮбһәбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһӮбһҺбһ“бһёбһ•бҹ’бһ‘бһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ `--services b` бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбҹҶбһҺбһҸбҹӢбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ blobs бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

### бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… - бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GPS бһ–бһёбһ•бҹ’бһ‘бһ»бһҖ

    бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `<storage_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҺбһ“бһёбһ•бҹ’бһ‘бһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `<subscription_key>` бһҮбһ¶бһҳбһҪбһҷбһҖбһјбһ“бһҹбҹ„ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһё Azure Maps бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

    бһҳбһ¶бһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ” бһҮбһ¶бһҠбҹҶбһ”бһјбһ„ бһҖбһјбһҠбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GPS бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ–бһёбһҖбһүбҹ’бһ…бһ”бҹӢ blob бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ URL endpoint бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҺбһ“бһёбһ•бҹ’бһ‘бһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” URL бһ“бҹҒбҹҮбһ‘бһ¶бһүбһҷбһҖбһ–бһё `gps-data` бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ’бһ“бһ’бһ¶бһ“бһӮбһәбһҖбһүбҹ’бһ…бһ”бҹӢ (`restype=container`) бһ бһҫбһҷбһ”бһүбҹ’бһҮбһёбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһўбҹҶбһ–бһё blobs бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбҹ” бһ”бһүбҹ’бһҮбһёбһ“бҹҒбҹҮбһҳбһ·бһ“бһҸбҹ’бһҡбһӣбһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ blob бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ“бҹ„бҹҮбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҸбҹ’бһҡбһӣбһ”бҹӢ URL бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ blob бһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ blob бҹ”

    > рҹ’Ғ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбһ¶бһҖбҹӢ URL бһ“бҹҒбҹҮбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһё blobs бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһүбҹ’бһ…бһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ’бһ¶бһҸбһ»бһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ `Url` бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ blob бҹ”

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ•бҹ’бһ‘бһ»бһҖбһҖбҹ’бһ”бһ¶бһӣ blob бһҡбһ¶бһӣбҹӢбһҒбҹ’бһ‘бһ„бҹӢбһҠбҹ„бһҷбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ `loadJSON` бһҠбҹӮбһӣбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ“бҹҒбҹҮбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ•бҹӮбһ“бһ‘бһё бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһ‘бҹ…бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ `ready`бҹ” бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһ–бҹҒбһӣбһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”

    бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ ready бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Azure Maps - бһӮбһәбһҮбһ¶бһҖбһүбҹ’бһ…бһ”бҹӢбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ GeoJSON бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбҹ” бһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ”бҹҠбһјбһ”бҹҠбҹӮбһӣ - бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһҡбһ»бһҳбһҡбһ„бҹ’бһңбһ„бҹӢбһӣбһҫбһ•бҹӮбһ“бһ‘бһёбһ”бҹ’бһҡбһ бҹӮбһӣбһҳбһҪбһҷбһӣбһҫбһ…бҹҶбһҺбһ»бһ…бһҖбҹ’бһ“бһ»бһ„ GeoJSONбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡ `loadJSON` бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ»бһҖбһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҳбһ»бһҒбһ„бһ¶бһҡ `init`бҹ–

    ```javascript
    var map, features;

    function loadJSON(file) {
        var xhr = new XMLHttpRequest();
        features = [];
        xhr.onreadystatechange = function () {
            if (xhr.readyState === XMLHttpRequest.DONE) {
                if (xhr.status === 200) {
                    gps = JSON.parse(xhr.responseText)
                    features.push(
                        new atlas.data.Feature(new atlas.data.Point([parseFloat(gps.gps.lon), parseFloat(gps.gps.lat)]))
                    )
                }
            }
        };
        xhr.open("GET", file, true);
        xhr.send();
    }    
    ```

    бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҠбҹ„бһҷбһҡбһ”бҹҖбһ” fetch бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһӣбҹӮбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ JSON бһ бһҫбһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһңбһ¶бһІбҹ’бһҷбһўбһ¶бһ“бһҮбһ¶ бһЎбһ»бһ„бһ бҹ’бһӮбһёбһ‘бһ»бһҠ бһ“бһ·бһ„бһӣбһ¶бһҸбһёбһ‘бһ»бһҠ бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ geoJSONбҹ”
