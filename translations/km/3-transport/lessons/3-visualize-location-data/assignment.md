# ផ្សព្វផ្សាយកម្មវិធីរបស់អ្នក

## សេចក្ដីណែនាំ

មានវិធីជាច្រើនដែលអ្នកអាចផ្សព្វផ្សាយកម្មវិធីរបស់អ្នក ដើម្បីអ្នកអាចចែករំលែកវាជាមួយពិភពលោក រួមមានការប្រើ GitHub pages ឬការប្រើអ្នកផ្តល់សេវាកម្មជាច្រើន។ វិធីដ៏អស្ចារ្យមួយក្នុងការធ្វើបែបនេះគឺការប្រើ Azure Static Web Apps។ ក្នុងភារកិច្ចនេះ សាងសង់កម្មវិធីវេបរបស់អ្នកហើយផ្សព្វផ្សាយវាទៅកាន់ពពកដោយតាមដាន [សេចក្ដីណែនាំទាំងនេះ](https://github.com/Azure/static-web-apps-cli) ឬមើល [វីដេអូទាំងនេះ](https://www.youtube.com/watch?v=ADVGIXciYn8&list=PLlrxD0HtieHgMPeBaDQFx9yNuFxx6S1VG&index=3)។
អត្ថប្រយោជន៍មួយនៃការប្រើ Azure Static Web Apps គឺអ្នកអាចលាក់កូនសោ API ដែលមាននៅក្នុងកំពូលដូនពពកបាន ដូច្នេះយកឱកាសនេះ ដើម្បីធ្វើឱ្យ subscriptionKey របស់អ្នកក្លាយជาตัวអថេរមួយ ហើយរក្សាទុកវានៅក្នុងពពក។
