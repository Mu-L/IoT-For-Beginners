# ស៊ើបអង្កេតកំណត់តម្លៃមុខងារ

## សេចក្តីណែនាំ

ការតភ្ជាប់មុខងារអនុញ្ញាតឱ្យកូដរបស់អ្នករក្សាទុកប្លុកទៅឃ្លាំងប្លុកដោយបង្វិលវាចេញពីមុខងារ `main`។ គណនីឃ្លាំង Azure, ការប្រមូល និងព័ត៌មានផ្សេងទៀតត្រូវបានកំណត់នៅក្នុងឯកសារ `function.json`។

នៅពេលធ្វើការជាមួយ Azure ឬបច្ចេកវិទ្យា Microsoft ផ្សេងទៀត ប្រភពព័ត៌មានល្អបំផុតគឺ [ឯកសាររបស់ Microsoft នៅលើ docs.com](https://docs.microsoft.com/?WT.mc_id=academic-17441-jabenn)។ នៅក្នុងការងារនេះ អ្នកនឹងត្រូវអានឯកសារតភ្ជាប់មុខងារ Azure Functions ដើម្បីដឹងពីរបៀបកំណត់តម្លៃមុខងារចេញ។

ទំព័រមួយចំនួនដែលត្រូវបើកដើម្បីចាប់ផ្តើមមានដូចជា៖

* [មូលដ្ឋាននៃការហៅ និងការតភ្ជាប់ខ្សែដុំ Azure Functions](https://docs.microsoft.com/azure/azure-functions/functions-triggers-bindings?WT.mc_id=academic-17441-jabenn&tabs=python)
* [ការតភ្ជាប់ឃ្លាំងប្លុក Azure Blob storage សម្រាប់ Azure Functions នៅលើទិដ្ឋភាពទូទៅ](https://docs.microsoft.com/azure/azure-functions/functions-bindings-storage-blob?WT.mc_id=academic-17441-jabenn)
* [ការតភ្ជាប់បញ្ចូលផលប្លុក Azure Blob storage សម្រាប់ Azure Functions](https://docs.microsoft.com/azure/azure-functions/functions-bindings-storage-blob-output?WT.mc_id=academic-17441-jabenn&tabs=python)
