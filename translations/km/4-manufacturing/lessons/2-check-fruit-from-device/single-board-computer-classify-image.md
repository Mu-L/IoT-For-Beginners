# бһ…бҹӮбһҖбһ…бһ¶бһҸбҹӢбһҡбһјбһ”бһ—бһ¶бһ– - бһҹбһҳбҹ’бһ—бһ¶бһҡбҹҲ IoT бһҷбҹүбһ¶бһ„бһ–бһ·бһҸ бһ“бһ·бһ„ Raspberry Pi

бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ•бҹ’бһүбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ¶бһ”бҹӢбһҷбһҖбһҠбҹ„бһҷбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Custom Vision бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹӮбһҖбһ…бһ¶бһҸбҹӢбһңбһ¶бҹ”

## бһ•бҹ’бһүбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ… Custom Vision

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Custom Vision бһҳбһ¶бһ“ Python SDK бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹӮбһҖбһ…бһ¶бһҸбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бҹ”

### бһ”бҹҶбһҺбһ„бһҖбһ¶бһҡ - бһ•бҹ’бһүбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ… Custom Vision

1. бһ”бһҫбһҖбһҗбһҸ `fruit-quality-detector` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Codeбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһӣбһҫбһҖбҹҶбһҺбһҸбҹӢ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ”бһҡбһ·бһҷбһ¶бһӮбһ“бҹ’бһӣбһ„бһ“бҹ„бҹҮбһҖбҹҶбһ–бһ»бһ„бһҡбһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ terminalбҹ”

1. Python SDK бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһүбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ… Custom Vision бһҳбһ¶бһ“бһ“бҹ…бһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Pip packageбҹ” бһҠбҹҶбһЎбһҫбһ„бһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҸбһ‘бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```sh
    pip3 install azure-cognitiveservices-vision-customvision
    ```

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһҖбһ¶бһҹ import бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһӣбһҫбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `app.py`бҹ–

    ```python
    from msrest.authentication import ApiKeyCredentials
    from azure.cognitiveservices.vision.customvision.prediction import CustomVisionPredictionClient
    ```

    бһ“бҹҒбҹҮбһ“бһ¶бҹҶбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһҖбһ–бһёбһҳбһјбһҢбһ»бһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Custom Vision бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҮбһ¶бһҳбһҪбһҷ key бһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚ бһ бһҫбһҷбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһҸбһӣбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢ client бһҠбҹӮбһӣбһўбһ¶бһ…бһ бҹ… Custom Visionбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹ…бһ…бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбҹ–

    ```python
    prediction_url = '<prediction_url>'
    prediction_key = '<prediction key>'
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<prediction_url>` бһҮбһ¶бһҳбһҪбһҷ URL бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһ„бһ–бһёбһ”бҹ’бһҡбһўбһ”бҹӢбһҹбһ“бҹ’бһ‘бһ“бһ¶ *Prediction URL* бһҳбһ»бһ“бһ“бҹҒбҹҮ бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбҹ” бһҮбҹҶбһ“бһҪбһҹ `<prediction key>` бһҮбһ¶бһҳбһҪбһҷ key бһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһ„бһ–бһёбһ”бҹ’бһҡбһўбһ”бҹӢбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ“бҹ„бҹҮбһҠбҹӮбһҡбҹ”

1. URL бһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһўбһ”бҹӢбһҹбһ“бҹ’бһ‘бһ“бһ¶ *Prediction URL* бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһҡбһҪбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ“бҹ…бһ–бҹҒбһӣбһ бҹ…бһҶбһ¶бһ“бҹӮбһӣ REST бһ–бһёбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” Python SDK бһ”бҹ’бһҡбһҫбһ•бҹ’бһ“бҹӮбһҖбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бһ“бҹғ URL бһҒбҹ’бһӣбһҪбһ“бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ”бҹӮбһҖ URL бһ“бҹҒбҹҮбһ…бҹҒбһүбһҮбһ¶бһ–бһёбһҡбһҹбҹҶбһҺбһ»бҹҶвҖӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ–

    ```python
    parts = prediction_url.split('/')
    endpoint = 'https://' + parts[2]
    project_id = parts[6]
    iteration_name = parts[9]
    ```

    бһ“бҹҒбҹҮбһ”бҹҶбһ”бҹӮбһҖ URL бһҠбһҖ endpoint бһҠбҹӮбһӣбһҮбһ¶ `https://<location>.api.cognitive.microsoft.com` бһҳбһҪбһҷ project ID бһ“бһ·бһ„бһҲбҹ’бһҳбҹ„бҹҮбһ“бҹғ iteration бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҹбһ¶бһҷбҹ”

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚ predictor бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҮбһ¶бһҳбһҪбһҷбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```python
    prediction_credentials = ApiKeyCredentials(in_headers={"Prediction-key": prediction_key})
    predictor = CustomVisionPredictionClient(endpoint, prediction_credentials)
    ```

    `prediction_credentials` бһ”бҹҶбһ–бһ¶бһҖбҹӢ key бһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбҹ” бһ–бһҪбһҖбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ client бһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһүбҹ’бһ…бһјбһ“бһ‘бҹ… endpoint бҹ”

1. бһ•бҹ’бһүбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ… custom vision бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```python
    image.seek(0)
    results = predictor.classify_image(project_id, iteration_name, image)
    ```

    бһ“бҹҒбҹҮбһҹбһҳбҹ’бһўбһ¶бһҸбһҡбһјбһ”бһ—бһ¶бһ–бһІбҹ’бһҷбһ‘бҹ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ бһҫбһҷбһ”бһүбҹ’бһҮбһјбһ“бһңбһ¶бһ‘бҹ… client бһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбҹ”

1. бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһҳбһҪбһҷбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```python
    for prediction in results.predictions:
        print(f'{prediction.tag_name}:\t{prediction.probability * 100:.2f}%')
    ```

    бһ“бҹҒбҹҮбһ“бһ№бһ„бһҡбҹҶбһҖбһ·бһӣбһҸбһ¶бһҳбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖ бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһҪбһҖбһңбһ¶бһ“бҹ…бһӣбһҫ terminalбҹ” бһ”бҹ’бһҡбһјбһ”бһ¶бһ”бһ¶бһӣбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ…бһ»бҹҮбһ…бһ»бҹҮбһ–бһё 0-1, бһҠбҹӮбһӣ 0 бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҳбһ¶бһ“бһұбһҖбһ¶бһҹ 0% бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһӮбһјбһҡбһ•бҹ’бһӮбһ„бһ“бһ№бһ„бһҹбҹ’бһӣбһ¶бһҖ бһ“бһ·бһ„ 1 бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ 100% бҹ”

    > рҹ’Ғ бһўбҹ’бһ“бһҖбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ—бһ¶бһӮбһҡбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһӣбһ¶бһҖбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбҹ” бһҹбҹ’бһӣбһ¶бһҖбһҳбһҪбһҷбһ“бһёбһҳбһҪбһҷбҹ—бһ“бһ№бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһјбһ”бһ¶бһ”бһ¶бһӣбһҗбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһ•бҹ’бһӮбһјбһҡбһ•бҹ’бһӮбһ„бһ“бһ№бһ„бһҹбҹ’бһӣбһ¶бһҖбһ“бҹ„бҹҮбҹ”

1. бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбҹ„бһҷбһІбҹ’бһҷбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹ…бһӣбһҫбһ•бҹ’бһӣбҹӮбһҲбһҫбһҳбһҪбһҷ бһ¬бһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһҡбһјбһ”бһ—бһ¶бһ–бһҹбһҳбһҡбһҳбҹ’бһҷ бһ¬бһ•бҹ’бһӣбҹӮбһҲбһҫбһҠбҹӮбһӣбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҸбһ¶бһҳ webcam бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бҹ’бһҡбһҫ hardware IoT бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ consoleбҹ–

    ```output
    (.venv) вһң  fruit-quality-detector python app.py
    ripe:   56.84%
    unripe: 43.16%
    ```

    бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһҗбһҸ бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ‘бһ¶бҹҶбһ„ **Predictions** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Custom Vision бҹ”
