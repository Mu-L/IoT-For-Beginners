# ឆ្លើយតបទៅនឹងលទ្ធផលចំណាត់ថ្នាក់ទាំងនេះ

## សេចក្តីណែនាំ

ឧបករណ៍របស់អ្នកបានចាត់ថ្នាក់រូបភាព ហើយមានតម្លៃសម្រាប់ការព្យាករណ៍។ ឧបករណ៍របស់អ្នកអាចប្រើព័ត៌មាននេះ ដើម្បីធ្វើអ្វីមួយ - វាអាចផ្ញើទៅកាន់ IoT Hub សម្រាប់ដំណើរការដោយប្រព័ន្ធផ្សេងទៀត ឬវាអាចគ្រប់គ្រងឧបករណ៍ដូចជា LED ដើម្បីភ្លឺពេលផ្លែឈើមិនទាន់ពេញ។

បន្ថែមកូដទៅឧបករណ៍របស់អ្នក ដើម្បីឆ្លើយតបដោយវិធីដែលអ្នកជ្រើសរើស - ទាំងផ្ញើទិន្នន័យទៅកាន់ IoT Hub, គ្រប់គ្រងឧបករណ៍ ឬបញ្ចូលទាំងពីរនិងផ្ញើទិន្នន័យទៅ IoT Hub ជាមួយកូដ serverless ដែលកំណត់ថាផ្លែឈើពេញរួចឬមិនហើយ ហើយផ្ញើសេចក្តីបញ្ជាថោក្នុងការគ្រប់គ្រងឧបករណ៍។
