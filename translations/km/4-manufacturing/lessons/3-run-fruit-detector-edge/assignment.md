## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶ

бһҳбһ·бһ“бһҳбҹӮбһ“бһҸбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бҹҶбһӣбһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһ…бҹ’бһҡбһҖбһ…бһ»бһ„бһ”бһ¶бһ“бһ‘бҹҒ бһўбҹ’бһңбһёбҹ—бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҳбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһ”бһ¶бһ“ бһҹбһјбһҳбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ…бһјбһӣбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ IoT Edgeбҹ” бһҖбһјбһҠбһӮбҹ’бһҳбһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶ Azure Functions бһҠбһјбһ…бһҮбһ¶ бһҸбҹ’бһҡбҹҖбһҳбһ”бһ„бҹ’бһ бҹ„бҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ—бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбһ“бһ·бһ„бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ“бҹ…бһӣбһҫ IoT Edgeбҹ”

бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹҒбһҡбҹҖбһ“бһҳбһҪбһҷбһ–бһёбһҳбһ»бһ“бһ“бҹҒбҹҮ бһ бһҫбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Azure Functions бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡ IoT Edgeбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ’бҹ’бһңбһҫбһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҖбҹ’бһ“бһ»бһ„ [бһҳбҹҒбһҡбҹҖбһ“бһ”бһ„бҹ’бһҡбҹҖбһ“бҹ– бһ”бҹ’бһҡбһҫ Azure Functions бһҮбһ¶ IoT Edge modules бһ“бҹ…бһӣбһҫбһҜбһҖбһҹбһ¶бһҡбһҳбһёбһҖбҹ’бһҡбһјбһҹбһјбһ бҹ’бһң](https://docs.microsoft.com/azure/iot-edge/tutorial-deploy-function?WT.mc_id=academic-17441-jabenn&view=iotedge-2020-11)бҹ”
