# бһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶ - Wio Terminal

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣбһӣбһҫ Wio Terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһҹбһҳбҹ’бһ—бһ¶бһҡбҹҲбһҡбһ№бһ„

бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶ **бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣ** бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ [photodiode](https://wikipedia.org/wiki/Photodiode) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһӣбҹӮбһ„бһ–бһ“бҹ’бһӣбһәбһ‘бҹ…бһҮбһ¶бһҹбһүбҹ’бһүбһ¶вҖӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбҹ” бһңбһ¶бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһўбһ¶бһЎбһјбһҖбһҠбҹӮбһӣбһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһӮбһҸбҹӢбһ–бһё 0 бһҠбһӣбҹӢ 1,023 бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ”бһҡбһ·бһҳбһ¶бһҺвҖӢбһ–бһ“бҹ’бһӣбһәвҖӢбһҠбҹӮбһӣвҖӢбһҳбһ¶бһ“бһ”бҹ’бһҡбһҮбһ¶бһ”бҹ’бһҡбһ·бһҷбһҳбһ·бһ“бһ”бһ¶бһ“бһ•бҹӮбһ“бһ‘бһёбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҜбһҖбһҸбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҺбһ¶бһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶ [lux](https://wikipedia.org/wiki/Lux)бҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„ Wio Terminal бһ бһҫбһҷбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҸбһ¶бһҳбһҡбһҷбҹҲбһ…бҹ’бһҡбһҖбһ”бҹ’бһӣбһ¶бһҹбҹ’бһ‘бһ·бһ…бһҗбҹ’бһӣбһ¶бһ–бһёбһҒбһ¶бһ„бһӣбһҫбҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣ

бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ”

### бһҖбһ¶бһҡбһ„бһ¶бһҡ

бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

1. бһ”бһҫбһҖбһӮбһҳбҹ’бһҡбҹ„бһ„ nightlight бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһ“бһ“бҹғбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹҒбҹҮ

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ `setup` бҹ–

    ```cpp
    pinMode(WIO_LIGHT, INPUT);
    ```

    бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹҒбҹҮбһҖбҹҶбһҺбһҸбҹӢбһ–бһ·бһ“бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бҹ”

    бһ–бһ·бһ“ `WIO_LIGHT` бһӮбһәбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ“бҹғбһ–бһ·бһ“ GPIO бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣбһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбҹ” бһ–бһ·бһ“бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһ‘бһ·бһҹ `INPUT` бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһңбһ¶бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҳбһҪбһҷ бһ бһҫбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһ–бһёбһ–бһ·бһ“бһ“бҹҒбҹҮбҹ”

1. бһӣбһ»бһ”бһҳбһ¶бһҸбһ·бһҖбһ¶вҖӢбһҡбһ”бһҹбҹӢбһўбһ“бһ»бһӮбһҳбһ“бҹҚ `loop` бһІбҹ’бһҷбһҹбҹ’бһўбһ¶бһҸбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бҹ…бһўбһ“бһ»бһӮбһҳбһ“бҹҚ `loop` бһҠбҹӮбһӣбһҘбһЎбһјбһңбһӮбҹ’бһҳбһ¶бһ“бһўбҹ’бһңбһё:

    ```cpp
    int light = analogRead(WIO_LIGHT);
    Serial.print("Light value: ");
    Serial.println(light);
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһўбһ¶бһЎбһјбһҖбһ–бһёбһ–бһ·бһ“ `WIO_LIGHT`бҹ” бһңбһ¶бһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ–бһё 0 бһҠбһӣбҹӢ 1,023 бһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ” бһҸбһҳбҹ’бһӣбҹғбһ“бҹҒбҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһ–бһ„бҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбһ¶бһ“бһңбһ¶вҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Serial Monitor бһ“бҹ…бһ–бҹҒбһӣбһҖбһјбһҠбһ“бҹҒбҹҮбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” `Serial.print` бһҹбһҡбһҹбҹҒбһҡбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһҖбҹ’бһ“бһ»вҖӢбһ„вҖӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҗбҹ’бһҳбһёбһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҡбһ¶бһӣбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бһ№бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷ `Light value:` бһ бһҫбһҷбһ”бһүбҹ’бһ…бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһ“бҹ’бһӣбһәбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣбһҸбһјбһ…бҹЎбһңбһ·бһ“бһ¶бһ‘бһё (1,000ms) бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғ `loop` бһ–бҹ’бһҡбҹ„бҹҮбһҖбһҳбҹ’бһҡбһ·бһҸбһ–бһ“бҹ’бһӣбһәбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ‘бҹҒбҹ” бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣвҖӢбһ“бҹҒбҹҮбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷвҖӢбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

    ```cpp
    delay(1000);
    ```

1. бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ Wio Terminal бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ бһҫбһҷбһ•бҹ’бһ‘бһ»бһҖбһҖбһјбһҠбһҗбҹ’бһҳбһёбһ‘бҹ… бһҠбһјбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҳбһ»бһ“бҹ”

1. бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… Serial Monitorбҹ” бһҸбһҳбҹ’бһӣбҹғбһ–бһ“бҹ’бһӣбһәбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹ…бһҸбһ¶бһҳбһҖбһ»бһ„бһҹбһјбһӣбҹ” бһӣбһ¶бһҖбҹӢбһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҖбһ¶бһҡвҖӢбһ–бһ“бҹ’бһӣбһәбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҳбһ–бһӣбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ Wio Terminal бһ бһҫбһҷбһҸбһҳбҹ’бһӣбҹғбһ“бһ№бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбҹ”
