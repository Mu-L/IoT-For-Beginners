# ស្រាវជ្រាវអំពីឧបករណ៍សែសឺរ និងឧបករណ៍បញ្ចេញសំពាធ

## សេចក្ដីណែនាំ

មេរៀននេះបានគ្របដណ្តប់អំពីឧបករណ៍សែសឺរ និងឧបករណ៍បញ្ចេញសំពាធ។ ស្រាវជ្រាវ និងពណ៌នាអំពីឧបករណ៍សែសឺរមួយ និងឧបករណ៍បញ្ចេញសំពាធមួយ ដែលអាចប្រើជាមួយឧបករណ៍អភិវឌ្ឍន៍ IoT រួមមាន៖

* តើវាធ្វើអ្វី
* អេឡិចត្រូនិច/ឧបករណ៍ដែកអាចប្រើប្រាស់នៅខាងក្នុង
* តើវាជាអាណាឡូក ឬឌីជីថល
* តើអង្គភាព និងជួរបញ្ចូល ឬវាស់វែងមានអ្វីខ្លះ
