1. бһ—бҹ’бһҮбһ¶бһ”бҹӢ Wio Terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” Wio Terminal бһҳбһ¶бһ“бһ…бҹ’бһҡбһҖ USB-C бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ…бҹ’бһҡбһҖ USB бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” Wio Terminal бһҳбһ¶бһ“бһҒбҹ’бһҹбҹӮ USB-C бһ‘бҹ… USB-A бһҮбһ¶бһҳбһҪбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ“бҹ…бһҖбһҡбһҺбһёбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҸбҹӮбһ…бҹ’бһҡбһҖ USB-C бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҒбҹ’бһҹбҹӮ USB-C бһ¬бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһҳбҹ’бһӣбҹӮбһ„ USB-A бһ‘бҹ… USB-Cбҹ”

1. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„ [Wio Terminal Wiki WiFi Overview documentation](https://wiki.seeedstudio.com/Wio-Terminal-Network-Overview/) бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ”бһ…бҹҶ Wio Terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ бҹ’бһңбһҫбһҳбһңбҹӮбһҡбҹ”

### Investigate the PlatformIO project

бһҮбҹҶбһ бһ¶бһ“бһҡбһ»бһҖбһҡбһҖбһҖбҹ’бһ“бһ»бһ„ VS Code бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһҗбһҸ бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё PlatformIO wizardбҹ”

#### Folders

* `.pio` - бһҗбһҸбһ“бҹҒбҹҮбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбҹ„бһҷ PlatformIO бһҠбһјбһ…бһҮбһ¶вҖӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ бһ¬бһҖбһјбһҠбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбҹ’бһҖбһ¶бһ”бҹ” бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһӣбһ»бһ”бһ…бҹ„бһӣ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһјбһҠбһ”бҹ’бһҡбһ—бһ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹ…бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбһјбһ…бһҮбһ¶ GitHub бҹ”

* `.vscode` - бһҗбһҸбһ“бҹҒбҹҮбһ•бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбҹ„бһҷ PlatformIO бһ“бһ·бһ„ VS Codeбҹ” бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһӣбһ»бһ”бһ…бҹ„бһӣ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһјбһҠбһ”бҹ’бһҡбһ—бһ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹ…бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбһјбһ…бһҮбһ¶ GitHub бҹ”

* `include` - бһҗбһҸбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҜбһҖбһҹбһ¶бһҡбһҳбһ»бһҒбһ”бҹӮбһ”бһҖбҹ’бһҡбҹ…бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ–бҹҒбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бһјбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһјбһҠбҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҗбһҸбһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ‘бҹҒбһҹбһ¶бһ‘бһ“бҹҒбҹҮбһ‘бҹҒбҹ”

* `lib` - бһҗбһҸбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҖбҹ’бһҡбҹ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ бҹ…бһ–бһёбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҗбһҸбһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ‘бҹҒбһҹбһ¶бһ‘бһ“бҹҒбҹҮбһ‘бҹҒбҹ”

* `src` - бһҗбһҸбһ“бҹҒбҹҮбһ•бҹ’бһ‘бһ»бһҖбһҖбһјбһҠбһҡбһҪбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһ”бһјбһ„ бһңбһ¶бһ“бһ№бһ„бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһӮбһә `main.cpp`бҹ”

* `test` - бһҗбһҸбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹүбһјбһҢбһ»бһӣбһҜбһҖбһҸбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

#### Files

* `main.cpp` - бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ `src` бһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮ бһңбһ¶бһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһјбһҠбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```cpp
    #include <Arduino.h>
    ```

    ```cpp
    void setup()
    {
        Serial.begin(9600);

        while (!Serial)
            ; // бһҡбһ„бҹӢбһ…бһ¶бҹҶбһҹбҹҠбҹҒбһҡбһёбһҷбҹүбһ¶бһӣбҹӢбһІбҹ’бһҷбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ
    ```
