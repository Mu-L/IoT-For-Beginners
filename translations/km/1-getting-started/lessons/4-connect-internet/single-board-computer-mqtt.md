# бһҸбҹ’бһҡбһҪбһҸбһӮбҹ’бһҡбһ„бһ—бҹ’бһӣбһҫбһ„бһҡбһ¶бһҸбҹ’бһҡбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһўбҹҠбһ·бһ“бһ’бһәбһҡбһҺаіҶбһҸ - бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҳбһ“бһ»бһҹбҹ’бһҹбһ–бһ·бһҸбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ“бһ·бһ„ Raspberry Pi

бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷ *test.mosquitto.org* бһ”бҹ’бһҡбһҫ MQTT бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһўбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһ„бҹӢбһҹбһ¶бһҡбһ–бһ“бҹ’бһӣбһә бһ“бһ·бһ„ бһ‘бһ‘бһҪбһӣбһ”бһүбҹ’бһҮбһ¶бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбһҪбһҸбһӮбҹ’бһҡбһ„ LEDбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢ Raspberry Pi бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҳбҹүбһјбһ‘бҹ’бһҷбһҪбһҸбһ‘бҹ… MQTT broker бҹ”

## бһҸбһҳбҹ’бһӣбһҫбһ„бһҖбһүбҹ’бһ…бһ”бҹӢ MQTT client

бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷ MQTT broker бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбһҳбҹ’бһӣбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ MQTT бһҸбһ¶бһҳбһҡбһҷбҹҲ pip package бһ“бҹ…бһӣбһҫ Pi бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ¬ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһңбҹҒбһҡбһёбһҹбҹ’бһңбһ¶бһӣбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбҹүбһјбһ‘бҹ’бһҷбһҪбһҸбҹ”

### бһҖбһ¶бһҡбһ„бһ¶бһҡ

бһҸбһҳбҹ’бһӣбһҫбһ„бһҖбһүбҹ’бһ…бһ”бҹӢ pip

1. бһ”бһҫбһҖбһӮбһҳбҹ’бһҡбҹ„бһ„ nightlight бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code бҹ”

1. бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбҹүбһјбһ‘бҹ’бһҷбһҪбһҸ бһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһ¶бһӣ бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ‘Terminal бһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһңбҹҒбһҡбһёбһҹбҹ’бһңбһ¶бһӣбҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫ Raspberry Pi бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бҹ’бһҡбһҫбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһңбҹҒбһҡбһёбһҹбҹ’бһңбһ¶бһӣбҹӢбһ‘бҹҒбҹ”

1. бһҡбһҸбҹӢбһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбһҫбһ„бһҖбһүбҹ’бһ…бһ”бҹӢ MQTT pip:

    ```sh
    pip3 install paho-mqtt
    ```

## бһҖбһјбһҠбһ§бһ”бһҖбһҡбһҺбҹҚ

бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһјбһҠбҹ”

### бһҖбһ¶бһҡбһ„бһ¶бһҡ

бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бҹ…бһҖбҹҶбһ–бһјбһӣбһҜбһҖбһҹбһ¶бһҡ `app.py`бҹ–

    ```python
    import paho.mqtt.client as mqtt
    ```

    бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ `paho.mqtt.client` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҸбһ¶бһҳ MQTTбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһ„бҹӢбһҹбһ¶бһҡбһ–бһ“бҹ’бһӣбһә бһ“бһ·бһ„ LEDбҹ–

    ```python
    id = '<ID>'

    client_name = id + 'nightlight_client'
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<ID>` бһҮбһ¶бһҳбһҪбһҷ ID бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбҹӮбһҳбһҪбһҷ бһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҮбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһ“бҹғ client бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҠбҹӮбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бҹ„бҹҮ бһ“бһ·бһ„ бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбҹ” broker *test.mosquitto.org* бһӮбһәбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһҳбһ“бһ»бһҹбҹ’бһҹбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ“бһ·бһҹбҹ’бһҹбһ·бһҸбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһ’бҹ’бһңбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮбҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ client MQTT бһ“бһ·бһ„бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбҹӮбһҳбһҪбһҷбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮбһ”бһүбҹ’бһ бһ¶бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ ID бһ“бҹҒбҹҮбһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһјбһҠбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ•бһ„бһҠбҹӮбһҡбҹ”

    > рҹ’Ғ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбһјбһ…бһҮбһ¶ [GUIDGen](https://www.guidgen.com) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ ID бһҠбҹӮбһӣбһ‘бҹҒбһҡбҹ”

    `client_name` бһӮбһәбһҮбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһ‘бҹ’бһҡбһ№бһ„бһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ client MQTT бһ“бҹҒбҹҮбһ“бҹ…бһӣбһҫ brokerбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҖбҹ’бһҡбҹ„бһҳбһҖбһјбһҠбһҗбҹ’бһҳбһёбһ“бҹҒбҹҮ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ”бҹ’бһҹбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MQTT client бһ бһҫбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT brokerбҹ–

    ```python
    mqtt_client = mqtt.Client(client_name)
    mqtt_client.connect('test.mosquitto.org')
    
    mqtt_client.loop_start()

    print("MQTT connected!")
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ”бҹ’бһҹбһ¶бһҡбһҳбһҪбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT broker бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ бһ бһҫбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбҹ’бһҡбһҖ бһҖбҹ’бһҡбҹ„бһҷ бһҠбҹӮбһӣбһҹбҹ’бһҠбһ¶бһ”бҹӢбһҹбһ¶бһҡвҖӢбһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбҹ”

1. бһҡбһҸбҹӢбһҖбһјбһҠбһҠбһјбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбһҸбҹӢбһҖбһјбһҠбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһ“бһ“бҹғбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбҹүбһјбһ‘бҹ’бһҷбһҪбһҸ бһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһ¶бһӣ бһ‘бҹ…вҖӢбһҡбһ¶бһӣбҹӢ CounterFit app бһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бһҫбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһ„бҹӢбһҹбһ¶бһҡбһ–бһ“бҹ’бһӣбһә бһ“бһ·бһ„ LED бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһӣбһҫ pin бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” 

    ```output
    (.venv) вһң  nightlight python app.py 
    MQTT connected!
    Light level: 0
    Light level: 0
    ```

> рҹ’Ғ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһҖбһјбһҠбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ [code-mqtt/virtual-device](../../../../../1-getting-started/lessons/4-connect-internet/code-mqtt/virtual-device) бһ¬ бһҗбһҸ [code-mqtt/pi](../../../../../1-getting-started/lessons/4-connect-internet/code-mqtt/pi)бҹ”
