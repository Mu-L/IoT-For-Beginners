# бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бһ“бҹ’бһӣбһәбһҷбһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ - Wio Terminal

бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹ’бһҡбһјбһңбһҖбһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ *test.mosquitto.org* бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ MQTT бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҹбғ”бғңбғЎбһҪбһҡбһ–бһ“бҹ’бһӣбһә бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһ”бһүбҹ’бһүбһ¶бһҸбҹ’бһҡбһҝбһ„бһ бҹ…бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ LEDбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢ Wio Terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҖбһ¶бһ“бҹӢ MQTT brokerбҹ”

## бһҠбҹҶбһЎбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ WiFi бһ“бһ·бһ„ MQTT бһҡбһ”бһҹбҹӢ Arduino

бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһ–бһҸбҹҢбһҳбһ¶бһ“бһ‘бҹ… MQTT broker бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбҹҶбһЎбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Arduino бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҲбһёбһ” WiFi бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Wio Terminal бһ бһҫбһҷбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷ MQTTбҹ” бһ“бҹ…бһ–бҹҒбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ Arduino бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһјбһҠбһ”бһҫбһҖбһ…бҹҶбһ  бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҳбһ»бһҒбһ„бһ¶бһҡбҹ” Seeed бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Wio Terminal бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһңбһ¶бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲ WiFiбҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MQTT broker бһ бһҫбһҷбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбһҮбһ¶бһҖбһјбһҠбһҠбһҫбһҳбһҠбҹӮбһӣбһўбһ¶бһ…бһ“бһ¶бҹҶбһ…бһјбһӣбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ‘бҹ… PlatformIO бһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Arduino бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ” Arduino framework бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһң hardware бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҠбҹӮбһӣбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбһ¶бһҮбһ¶бһ„бһ“бҹҒбҹҮбҹ” бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҒбҹ’бһӣбҹҮбҹ— бһҠбһјбһ…бһҮбһ¶ бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Seeed WiFi бһӮбһәбһҮбһ¶бһҹбһҳбҹ’бһ”бһ‘бһ¶бһ“бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ·бһҹбҹҒбһҹбҹ”

бһ”бһҺбҹ’бһҺбһӣбҹҗбһҷбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһ‘бһјбһ‘бҹ… бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһңбһ·бһү бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ¬бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҠбҹҶбһЎбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбҹ”

вң… бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ бһ“бһ·бһ„бһҡбһ”бҹҖбһ”бһҡбһҖбһ“бһ·бһ„бһҠбҹҶбһЎбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһҜбһҖбһҹбһ¶бһҡбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ PlatformIO](https://docs.platformio.org/en/latest/librarymanager/index.html)бҹ”

## бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT

бһҖбҹ’бһҡбҹ„бһҷбһ–бһё Wio Terminal бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… WiFi бһ”бһ¶бһ“ бһңбһ¶бһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT broker бһ”бһ¶бһ“бҹ”

### бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… - бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT

бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT brokerбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бһӣбҹӢбһ…бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `config.h` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ MQTT brokerбҹ–

    ```cpp
    // бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ MQTT
    const string ID = "<ID>";
    
    const string BROKER = "test.mosquitto.org";
    const string CLIENT_NAME = ID + "nightlight_client";
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<ID>` бһҮбһ¶бһҳбһҪбһҷбһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҮбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё client бһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮ бһ“бһ·бһ„бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҠбҹӮбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһ•бҹ’бһҹбһ¶бһҷ бһ“бһ·бһ„бһҮбһ¶бһңбҹ” Broker *test.mosquitto.org* бһҮбһ¶бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ”бҹ’бһҡбһҫ бһҡбһҪбһҳбһҳбһ¶бһ“бһ“бһ·бһҹбҹ’бһҹбһ·бһҸбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҡбҹҖбһ“бһҠбҹ„бһҷбһҳбһ¶бһ“бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бҹҒбҹҮбҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ client MQTT бһ“бһ·бһ„бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҸбҹӮбһҳбһҪбһҷбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ‘бҹҒбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ ID бһ“бҹҒбҹҮ бһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһјбһҠбһҳбҹүбһ¶бһҹбҹҠбһёбһңбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһҡбҹ„бһҷбһ“бҹғбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бҹҒбҹҮбҹ”

    > рҹ’Ғ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбҹӮбһ”бһҹбһ¶бһҷбһҠбһјбһ…бһҮбһ¶ [GUIDGen](https://www.guidgen.com) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ ID бһҸбҹӮбһҳбһҪбһҷбҹ”

    `BROKER` бһӮбһәбһҮбһ¶ URL бһҡбһ”бһҹбҹӢ MQTT brokerбҹ”

    `CLIENT_NAME` бһӮбһәбһҮбһ¶ бһҲбҹ’бһҳбҹ„бҹҮбһҜбһҖбһҜбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ client MQTT бһ“бҹҒбҹҮбһ“бҹ…бһӣбһҫ brokerбҹ”

1. бһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡ `main.cpp` бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҖбҹ’бһҡбҹ„бһҳбһҳбһ»бһҒбһ„бһ¶бһҡ `connectWiFi` бһ“бһ·бһ„бһҒбһ¶бһ„бһӣбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `setup`бҹ–

    ```cpp
    WiFiClient wioClient;
    PubSubClient client(wioClient);
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбһ·бһҗбһ·бһҮбһ“ WiFi бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ WiFi бһҡбһ”бһҹбҹӢ Wio Terminal бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһңбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбһ·бһҗбһ·бһҮбһ“ MQTTбҹ”

1. бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҖбһјбһҠбһ“бҹҒбҹҮ бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```cpp
    void reconnectMQTTClient()
    {
        while (!client.connected())
        {
            Serial.print("Attempting MQTT connection...");
    
            if (client.connect(CLIENT_NAME.c_str()))
            {
                Serial.println("connected");
            }
            else
            {
                Serial.print("Retying in 5 seconds - failed, rc=");
                Serial.println(client.state());
                
                delay(5000);
            }
        }
    ```
