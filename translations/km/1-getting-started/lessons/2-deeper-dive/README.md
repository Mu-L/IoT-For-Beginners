## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһҮбҹ’бһҡбҹ…бһҮбһ¶бһ„бһҳбһ»бһҒбһ‘бҹ…бһӣбһҫбһӮбҹҶбһ“бһ·бһҸбһҒбҹ’бһӣбҹҮбҹ—бһҠбҹӮбһӣбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–

* [бһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҶбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё IoT](#бһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҶбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё-iot)
* [бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҮбҹ’бһҡбҹ…бһҮбһ¶бһ„бһ‘бҹ…бһӣбһҫ microcontrollers](#бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҮбҹ’бһҡбҹ…бһҮбһ¶бһ„бһ‘бҹ…бһӣбһҫ-microcontrollers)
* [бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҮбҹ’бһҡбҹ…бһҮбһ¶бһ„бһ‘бҹ…бһӣбһҫ single-board computers](#бһ…бҹҶбһҺбһҸбҹ’бһҡбһ¶бһ”бҹӢбһңбҹӮбһ„бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҗбһҸбһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһҸбһ¶бһҡ)

## бһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҶбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё IoT

бһ•бҹ’бһ“бҹӮбһҖбҹў бһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё IoT бһӮбһә *бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ* бһ“бһ·бһ„ *бһҡбһ”бһҹбҹӢ*бҹ” бһҷбһҫбһ„бһҳбһҖбһҳбһҫбһӣбһ•бҹ’бһ“бҹӮбһҖбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбһҷбҹүбһ¶бһ„бһӣбһҳбҹ’бһўбһ·бһҸбһҮбһ¶бһ„бһ“бҹҒбҹҮбҹ”

### бһҡбһ”бһҹбҹӢ

![A Raspberry Pi 4](../../../../../translated_images/km/raspberry-pi-4.fd4590d308c3d456.webp)

**бһҡбһ”бһҹбҹӢ** бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһ–бһ·бһ—бһ–бһҡбһ¶бһ„бһҖбһ¶бһҷбҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһ¶бһ‘бһјбһ‘бҹ…бһҮбһ¶бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҸбһјбһ…бҹ— бһҸбһҳбҹ’бһӣбҹғбһ…бһ»бҹҮбһҗбҹ„бһҖ бһҳбҹ’бһ бһјбһ”бһҘбһ“бҹ’бһ’бһ“бҹҲбһҸбһ·бһ… бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһ”бһҝбһ“бһ‘бһ¶бһ” вҖ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ microcontrollers бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“ RAM бһӮбһ·бһҸбһҮбһ¶бһӮбһёбһЎбһјбһ”бҹғ (бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһҮбһ¶бһӮбһёбһЎбһјбһ”бҹғбһ“бҹ… PC) бһҠбҹ„бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһ№бһҳбһҡбһҷбҹҲбһҳбҹүбҹҒбһ бҹ’бһӮбһ¶бһ бҹҠбҹӮбһҡбһёбһҸ (бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһҮбһ¶бһӮбһёбһЎбһјбһ бҹҠбҹӮбһҡбһёбһҸбһ“бҹ… PC) бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбһҸбһ·бһ…бһ–бҹҒбһҖ бһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҮбһ¶бһҹбһ”бҹ’бһҸбһ¶бһ бҹҸ бһҒбҹӮ бһ¬бһҶбҹ’бһ“бһ¶бҹҶ бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҗбҹ’бһҳбҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһ–бһ·бһ—бһ–бһҡбһ¶бһ„бһҖбһ¶бһҷ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбҹҒбһ“бһҹбһәбһҡбһ…бҹҶбһӣбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ бһ¬бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣ бһ¬ actuator бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҮбһ¶бһҡбһјбһ”бһҡбһ¶бһ„бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ’бһҳбҹ’бһҳбһҸбһ¶бһҳбһҪбһҷбһӮбһә smart thermostat вҖ” бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹҒбһ“бһҹбһәбһҡбһҖбҹҶбһҠбҹ…, бһңбһ·бһ’бһёбһҖбҹҶбһҺбһҸбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҠбҹӮбһӣбһ…бһ„бҹӢбһ”бһ¶бһ“бһҠбһјбһ…бһҮбһ¶ dial бһ¬ touchscreen, бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹҶбһҠбҹ… бһ¬бһҸбҹ’бһҡбһҮбһ¶бһҖбҹӢбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһҫбһҖбһ–бҹҒбһӣбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҒбһ»бһҹбһ–бһёбһҮбһҪбһҡбһҠбҹӮбһӣбһ…бһ„бҹӢбһ”бһ¶бһ“бҹ” бһҹбҹҒбһ“бһҹбһәбһҡбһҖбҹҶбһҠбҹ…бһңбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҗбһ¶бһҳбҹҒбһҢбҹҖбһҳбһҸбҹ’бһҡбһҮбһ¶бһҖбҹӢбһ–бҹҒбһҖ бһ бһҫбһҷ actuator бһ”бһҫбһҖбһҖбҹҶбһҠбҹ…бҹ”

![A diagram showing temperature and a dial as inputs to an IoT device, and control of a heater as an output](../../../../../translated_images/km/basic-thermostat.a923217fd1f37e5a.webp)

бһҳбһ¶бһ“бһҡбһ”бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһӮбҹ’бһҡбһҝбһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ”бһ¶бһ“ бһ…бһ¶бһ”бҹӢбһ–бһё hardware бһ•бҹ’бһҠбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӣбһҫ sensing бһҳбһҪбһҷбһҳбһ»бһҒ бһҠбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһјбһ‘бҹ… бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ smartphone бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бһ„бһҠбҹӮбһҡ! Smartphone бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбҹҒбһ“бһҹбһәбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһӮбһ·бһҸбһҗбһ¶ бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһҮбһ»бҹҶбһңбһ·бһү бһ бһҫбһҷ actuator бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһ–бһ·бһ—бһ–бһ–бһёбһҖбҹ’бһҡбҹ… вҖ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ”бҹ’бһҡбһҫбһҹбҹҒбһ“бһҹбһәбһҡбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖ GPS бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„ Speaker бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ•бҹ’бһӣбһјбһңбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбҹ„бһӣбһҠбҹ…бҹ”

вң… бһҹбһјбһҳбһӮбһ·бһҸбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ“бҹ…бһҮбһ»бҹҶбһңбһ·бһү бһҠбҹӮбһӣбһўбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҗбҹ’бһӣбҹӮбһ„бһҹбҹҒбһ“бһҹбһәбһҡбһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһңбһ¶бһ‘бҹ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһҪбһҷбһӮбһә thermostat бһ“бҹ…бһӣбһҫбһ…бһ„бҹ’бһҖбҹ’бһҡбһ¶бһ“бҹ” бһҸбһҫбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбһ‘бҹҒ?

### бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ

бһ•бҹ’бһ“бҹӮбһҖ **бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ** бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё IoT бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ“бһ·бһ„бһҮбһҪбһҷбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһӣбһҫбһҹбҹҶбһҺбһҫбһҹбһ»бҹҶбһҠбһӣбҹӢ actuator бһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ IoTбҹ”

бһӣбҹҶбһ“бҹ…бһ’бһҳбҹ’бһҳбһҸбһ¶бһҳбһҪбһҷ бһӮбһәбһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһҳбһҪбһҷ бһҠбҹӮбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… бһ бһҫбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһ“бҹ„бҹҮбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ¶бһҸбһ»бһҠбһјбһ…бһҮбһ¶ бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһҹбһ¶бһҡ бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ·бһүбҹ” бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһ“бһ№бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ¬бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹҒбһ“бһҹбһәбһҡ бһ¬бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹҒбһ“бһҹбһәбһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбҹҒбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбҹ”

бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ thermostat бһҠбһҫбһҳбҹ’бһ”бһё Smarter thermostat бһ“бһ№бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбһ¶бһҳ WiFi бһ•бҹ’бһ‘бҹҮбһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloudбҹ” бһңбһ¶бһҹбҹҶбһҠбҹ…бһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһ бһҫбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ…бһңбҹҒбһ”бһҹбһ¶бһҷбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ•бҹ’бһ‘бҹҮбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ“бһ·бһ„бһҖбһ“бҹ’бһӣбһ„бһҳбһҖбһҠбҹ„бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бҹ” бһҹбҹҒбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„ cloud бһ“бһ№бһ„бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ•бҹ’бһ‘бҹҮбһ…бһ„бҹӢбһ”бһ¶бһ“ бһ бһҫбһҷбһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҸбһ¶бһҳбһҡбһҷбҹҲбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖ бһ¬бһ”бһ·бһ‘бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹҶбһҠбҹ…бҹ”

бһ”бһҫбһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶ I бһҖбҹ’бһ“бһ»бһ„ IoT бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ Internet бһҖбҹҸбһҠбҹ„бһҷ бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһЎбһҫбһҷбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҒбҹ’бһӣбҹҮбҹ— бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ 'edge' бһӮбһәбһ§бһ”бһҖбһҡбһҺбҹҚ gateway бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһ”бһҺбҹ’бһҸбһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ бҹ…бһ‘бҹ…бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбҹ” бһңбһ¶бһўбһ¶бһ…бһӣбһҝбһ“бһҮбһ¶бһ„бһ–бҹҒбһӣбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“ бһ¬бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҷбһәбһҸ бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһўбһ¶бһҖбһҒбһҺ:бһ–бҹҒбһӣбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҳбһ·бһ“бһўбһ¶бһ…бһ…бһјбһӣбһ”бһ¶бһ“ бһҠбһјбһ…бһҮбһ¶ бһӣбһҫбһҖбһ”бһүбҹ’бһ…бһ¶бһҖбҹӢбһ‘бһјбһҖ бһ¬бһҸбҹҶбһ”бһ“бҹӢбһӮбҹ’бһҡбҹ„бҹҮбһҳбһ бҹүбһ¶бһўбһ¶бһҹбһ“бҹ’бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһңбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһҳбһ“бһ»бһҹбҹ’бһҹбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ“бҹ„бҹҮбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһІбҹ’бһҷбһҜбһҖбһҮбһ“бҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһҒбҹ’бһӣбҹҮбҹ—бһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһјбһҠбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ cloud бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҳбһјбһӣбһ“бһ·бһ„бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһҳбһ·бһ“бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһҪбһҷбһӮбһәбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бҹҮбһҶбҹ’бһӣбһ¶бһҸбһҠбһјбһ…бһҮбһ¶ Apple HomePod, Amazon Alexa бһ¬ Google Home бһҠбҹӮбһӣбһ“бһ№бһ„бһҹбҹ’бһҸбһ¶бһ”бҹӢбһҹбҹҶбһЎбҹҒбһ„бһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҖбҹ’бһ“бһ»бһ„ cloud бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бһ№бһ„ "бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶" бһ“бҹ…бһ–бҹҒбһӣбһ–бһ¶бһҖбҹ’бһҷбһ¬бһ”бҹ’бһҡбһҷбҹ„бһӮбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ“бһ·бһҷбһ¶бһҷ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ•бҹ’бһүбһҫбһҹбҹҶбһЎбҹҒбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһ№бһ„бһ”бһүбҹ’бһҲбһ”бҹӢбһ•бҹ’бһүбһҫбһҹбҹҶбһЎбҹҒбһ„бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһңбһ¶бһңбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҗбһ¶бһҳбһ¶бһ“бһҖбһ¶бһҡбһҲбһ”бҹӢбһҹбҹҶбһЎбҹҒбһ„бһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ·бһҷбһ¶бһҷбһҳбһ»бһ“бһ–бҹҒбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһ–бһ¶бһҖбҹ’бһҷбһ•бҹ’бһҠбһӣбҹӢбһҹбһүбҹ’бһүбһ¶ бһ“бһ·бһ„бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ·бһҷбһ¶бһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҲбһ”бҹӢбһҹбҹ’бһҠбһ¶бһ”бҹӢвҖӢбһ“бһ№бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ‘бҹ…бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹ„бҹҮбһ“бһ·бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһҜбһҖбһҮбһ“бҹ”

вң… бһҹбһјбһҳбһӮбһ·бһҸбһўбҹҶбһ–бһёбһҹбҹ’бһҗбһ¶бһ“аёҒаёІаёЈаё“а№Ңбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһ—бһ¶бһ–бһҜбһҖбһҮбһ“бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбҹ’бһўбһҮбһ¶бһ„бһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢ edge бһҮбһ¶бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ cloudбҹ” бһҮбһ¶бһӮбһ“бҹ’бһӣбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһ·бһҸвҖ” бһӮбһ·бһҸбһўбҹҶбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶ бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһҗбһҸбһҡбһјбһ”бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

### бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ IoT

бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҺбһ¶бһҳбһҪбһҷ бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҮбһ¶бһҖбһҸбҹ’бһҸбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ” бһҳбһ¶бһ“бһҖбҹҶбһ”бҹ’бһӣбҹӮбһ„бһ…бһ¶бһҹбҹӢбһҳбһҪбһҷбһҗбһ¶ 'S бһҖбҹ’бһ“бһ»бһ„ IoT бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ Security' вҖ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҳбһ·бһ“бһҳбһ¶бһ“ S бһҖбҹ’бһ“бһ»бһ„ IoT бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶бһңбһ¶бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһЎбһҫбһҷбҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһ бһҫбһҷбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбһјбһ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһ“бҹ„бҹҮ вҖ” бһ”бһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ cloud бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһ¶бһ“ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ»бһӣбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ¬бһңбһёбһҡбһ»бһҹбһўбһ¶бһ…бһҶбҹ’бһӣбһ„бҹ” бһңбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ–бҹ’бһҡбҹ„бҹҮбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ [Stuxnet worm](https://wikipedia.org/wiki/Stuxnet) бһ”бһ¶бһ“бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһңбҹүбһ¶бһӣбҹӢбһңбҹүбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ centrifuges бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһңбһ¶бҹ” бһўбҹ’бһ“бһҖбһ…бҹ„бһҡбһҖбһҳбҹ’бһҳбһҖбҹҸбһ”бһ¶бһ“бһҷбһҖбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһё [бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһўбһ“бҹӢбҹ—бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҡбһҷбҹҲ baby monitors](https://www.npr.org/sections/thetwo-way/2018/06/05/617196788/s-c-mom-says-baby-monitor-was-hacked-experts-say-many-devices-are-vulnerable) бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбһ¶бһҳбһҠбһ¶бһ“бһ•бҹ’бһ‘бҹҮбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

> рҹ’Ғ бһўбҹ’бһ“бһҖбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ“бһ·бһ„ edge device бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһ”бһҺбҹ’бһҠбһ¶бһүбһҠбҹӮбһӣбһ”бһ·бһ‘бһҹбҹ’бһҡбһ”бһ–бһёбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһІбҹ’бһҷбһҜбһҖбһҮбһ“бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ” бһ“бҹҒбҹҮбһӮбһәбһ бҹ…бһҗбһ¶ [air-gapping](https://wikipedia.org/wiki/Air_gap_(networking))бҹ”

## бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҮбҹ’бһҡбҹ…бһҮбһ¶бһ„бһ‘бҹ…бһӣбһҫ microcontrollers

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“ бһҷбһҫбһ„бһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶ microcontrollersбҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҳбһҖбһҳбһҫбһӣбһҮбҹ’бһҡбҹ…бһҮбһ¶бһ„бһ–бһёбһ–бһҪбһҖбһңбһ¶бҹ”

### CPU

CPU бһӮбһәбһҮбһ¶бһҶбҹ’бһ“бһҫбһҳбһҳбһҪбһҷбһ“бҹғ microcontrollerбҹ” бһңбһ¶бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһўбһ¶бһ…бһ•бҹ’бһүбһҫбһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ”бһ¶бһ“бҹ” CPU бһўбһ¶бһ…бһҳбһ¶бһ“бһҳбһҪбһҷбһ…бҹ’бһҡбһҫбһ“ core вҖ” бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ CPU бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

CPU бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҖбҹ’бһӣбһ»бһҖбһҖбһ¶бһЎбһ„бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһҹбҹҠбһёбһ…бҹ’бһҡбһҫбһ“бһӣбһ¶бһ“бһӣбһ¶бһ“бһ“бһ¶бһҖбҹӢбһ‘бһ”бһёбһӣбһ¶бһ“бһ”бһҪбһ“бһӣбһ¶бһ“бһ–бһёбһҡбһҒбҹ’бһ“бһ¶бһҸбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһңбһ·бһ“бһ¶бһ‘бһёбҹ” бһҡбһ¶бһӣбҹӢбһҖбҹ’бһӣбһ»бһҷ бһ¬ бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣ CPU бһўбһ¶бһ…бһ”бҹҶбһ–бҹҒбһүбҹ” бһҡбһ¶бһӣбҹӢбһҖбҹ’бһӣбһ»бһҷ CPU бһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһҡбҹ…бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһ¬бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бҹ” бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһҖбҹ’бһҡбһҸбһёбһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбһ»бһ“бһ–бҹҒбһӣбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҸбһ¶бһҳбҹ”

бһҖбҹ’бһӣбһ»бһҖбһ–бһёбһӣбһҝбһ“ бһ”бҹ’бһҡбһ¶бһ”бҹӢбһўбҹҶбһ–бһёбһ…бҹҶбһ“бһҪбһ“бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһңбһ·бһ“бһ¶бһ‘бһё бһ бһҫбһҷбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһӮбһә CPU бһӣбһҝбһ“бһҮбһ¶бһ„бҹ” бһӣбҹ’бһ”бһҝбһ“ CPU бһҳбһ¶бһ“бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһҮбһ¶ [Hertz (Hz)](https://wikipedia.org/wiki/Hertz) бһҠбҹӮбһӣ 1 Hz бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһңбһӮбҹ’бһӮбһҳбһҪбһҷбһ¬бһҖбҹ’бһӣбһ»бһҖбһҸбҹ’бһҡбһјбһҹбҹҠбһёбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһңбһ·бһ“бһ¶бһ‘бһёбҹ”

> рҹҺ“ бһӣбҹ’бһ”бһҝбһ“ CPU бһҮбһ¶бһ‘бһјбһ‘бҹ…бһҳбһҫбһӣбһҮбһ¶ MHz бһ¬ GHzбҹ” 1MHz бһӮбһә бҹЎбһӣбһ¶бһ“бһ бһәбһҸ (million Hz), 1GHz бһӮбһә бҹЎбһ–бһ¶бһ“бҹӢбһӣбһ¶бһ“бһ бһәбһҸ (billion Hz)бҹ”

> рҹ’Ғ CPU бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбһ¶бһҳбһҡбһҷбҹҲ [бһңбһӮбҹ’бһӮ fetch-decode-execute](https://wikipedia.org/wiki/Instruction_cycle)бҹ” бһҡбһ¶бһӣбҹӢбһҖбҹ’бһӣбһ»бһҖбһҖбһ“бҹ’бһӣбҹӮбһ„ CPU бһ“бһ№бһ„бһ‘бһ¶бһүбһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҳбҹүбҹӮбһҳбһҳбҹүбһҡбһё decode бһңбһ¶ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ¶ бһҠбҹӮбһӣбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбһјбһ…бһҮбһ¶ ALU бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһјбһҖбһӣбҹҒбһҒбһ–бһёбһҡбһ…бҹҶбһ“бһҪбһ“бҹ” бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҒбҹ’бһӣбҹҮбҹ—бһ…бҹҶбһҺбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһӣбһ»бһҖ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһӮбҹ’бһӮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ“бһ№бһ„бһҡбһҸбҹӢбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбҹ’бһӣбһ»бһҖбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбҹ”

Microcontrollers бһҳбһ¶бһ“бһӣбҹ’бһ”бһҝбһ“бһҖбҹ’бһӣбһ»бһҖбһ‘бһ¶бһ”бһҮбһ¶бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһӣбһҫбһҸбһ» бһ¬ laptop бһ¬бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҠбҹғбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Wio Terminal бһҳбһ¶бһ“ CPU бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ… 120MHz бһ¬ 120,000,000 бһңбһӮбҹ’бһӮбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһңбһ·бһ“бһ¶бһ‘бһёбҹ”

вң… бһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһ бҹӮбһӣбһҖбһ“бҹ’бһӣбҹӮбһ„ PC бһ¬ Mac бһҳбһ’бҹ’бһҷбһҳбһҳбһ¶бһ“ CPU бһҠбҹӮбһӣбһҳбһ¶бһ“ cores бһ…бҹ’бһҡбһҫбһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһ–бһ бһ» GHz бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһ“бҹҗбһҷбһҗбһ¶бһҖбҹ’бһӣбһ»бһҖбһҸбҹ’бһҡбһјбһҹбҹҠбһёбһӣбһ¶бһ“бһӣбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһңбһ·бһ“бһ¶бһ‘бһёбҹ” бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһӣбҹ’бһ”бһҝбһ“бһҖбҹ’бһӣбһ»бһҖбһҡбһ”бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бҹҶбһ бҹҶбһӣбҹ’бһ”бһҝбһ“бһ“бһ№бһ„бһҸбһ¶бһҳ Wio Terminalбҹ”

бһҡбһ¶бһӣбҹӢбһңбһӮбҹ’бһӮбһҖбҹ’бһӣбһ»бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһҠбҹ…бҹ” бһӣбҹ’бһ”бһҝбһ“бһҖбҹ’бһӣбһ»бһҖбһ”бҹҶбһ•бһ»бһҸ бһ…бҹҶбһҺбһ¶бһҷбһҗбһ¶бһҳбһ–бһӣбһ…бҹ’бһҡбһҫбһ“бһ“бһ·бһ„бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҖбҹҶбһҠбҹ…бһ…бҹ’бһҡбһҫбһ“бҹ” PC бһҳбһ¶бһ“ heat sinks бһ“бһ·бһ„бһңбһёбһҹбһҹбҹ’бһңбһёбһ„бһҖбһ¶бһҸбҹӢбһҸбҹ’бһҡбһҮбһ¶бһҖбҹӢ бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһ–бһҪбһҖбһңбһ¶бһ“бһ№бһ„бһҖбҹҶбһҠбҹ…бһҒбҹ’бһӣбһҪбһ“ бһ бһҫбһҷбһ”бһ·бһ‘бһҒбҹ’бһ‘бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҸбһјбһ…бҹ” Microcontrollers бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһӮбҹ’бһҳбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһҮбһ¶бһҖбҹӢбһҮбһ¶бһ„ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ бҹӮбһӣбһӣбһҝбһ“бһҸбһ·бһ…бһҮбһ¶бһ„бҹ” PC бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҗбһ¶бһҳбһ–бһӣбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё бһ¬бһҗбҹ’бһҳбһ’бҹҶбҹ—бһҡбһҷбҹҲбһ–бҹҒбһӣбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҳбҹүбҹ„бһ„ бһҒбһҺбҹҲ microcontrollers бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҮбһ¶бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҗбҹ’бһ„бҹғ бһҒбҹӮ бһ…бһјбһҸбһҠбһӣбҹӢбһҶбҹ’бһ“бһ¶бҹҶбһҠбҹ„бһҷбһҗбҹ’бһҳбһҸбһјбһ…бҹ—бҹ” Microcontrollers бһҖбҹҸбһўбһ¶бһ…бһҳбһ¶бһ“ cores бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһӣбҹ’бһ”бһҝбһ“бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ бһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ‘бҹ… core бһҗбһ¶бһҳбһ–бһӣбһ‘бһ¶бһ”бһ–бҹҒбһӣ CPU бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбһ·бһ… бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбҹ”

> рҹ’Ғ PC бһ“бһ·бһ„ Mac бһҒбҹ’бһӣбҹҮбҹ—бһҖбҹҶбһ–бһ»бһ„бһҷбһҖбһҡбһ”бҹҖбһ”бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҡбһңбһ¶бһ„ cores бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„ cores бһҹбһ“бҹ’бһҹбҹҶбһҗбһ¶бһҳбһ–бһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҮбҹҖбһҹбһңбһ¶бһ„бһҗбҹ’бһҳбһ”бҹ’бһҡбһҫбһӣбһҝбһ“бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ chip M1 бһҖбҹ’бһ“бһ»бһ„ laptop Apple бһҗбҹ’бһҳбһёбҹ—бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһңбһ¶ бһҡбһңбһ¶бһ„ 4 performance cores бһ“бһ·бһ„ 4 efficiency cores бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһ–бҹҒбһӣбһҮбһёбһңбһ—бһ¶бһ–бһҗбҹ’бһҳбһ¬бһӣбҹ’бһ”бһҝбһ“бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”

вң… бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ“бҹ’бһҸбһ·бһ…бҹ– бһўбһ¶бһ“бһўбҹҶбһ–бһё CPU бһ“бҹ…бһӣбһҫ [бһўбһҸбҹ’бһҗбһ”бһ‘ CPU Wikipedia](https://wikipedia.org/wiki/Central_processing_unit)

#### бһҖбһ¶бһҡбһ„бһ¶бһҡ

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң Wio Terminalбҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫ Wio Terminal бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ CPUбҹ” бһҹбһјбһҳбһҡбһҖбһ•бҹ’бһ“бҹӮбһҖ *Hardware Overview* бһ“бҹ…бһӣбһҫ [бһ‘бҹҶбһ–бҹҗбһҡбһ•бһӣбһ·бһҸбһ•бһӣ Wio Terminal](https://www.seeedstudio.com/Wio-Terminal-p-4509.html) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„ бһ бһҫбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҡбһҖ CPU бһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бһ„бҹ’бһўбһҪбһ…бһ”бҹ’бһӣаҙҫаҙёаөҚаҙұаөҚаҙұбһ·бһ…бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ”

### бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ

Microcontrollers бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҳбһ¶бһ“бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһ–бһёбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘ вҖ” бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһ…бҹғбһҠбһ“бҹ’бһҷ (RAM)бҹ”

бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбһәбһҳбһ·бһ“бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҖбҹ’бһҡбҹ„бһҷбһ–бҹҒбһӣбһ“бҹ…бһ–бҹҒбһӣбһӮбҹ’бһҳбһ¶бһ“бһҗбһ¶бһҳбһ–бһӣбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бҹҒбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҖбһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

RAM бһҮбһ¶бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҡбһҪбһҳбһҳбһ¶бһ“бһўбһҗбҹҒбһҡ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ—бһҡбһ·бһҷбһ¶бҹ” RAM бһӮбһәбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһ–бҹҒбһӣбһӮбҹ’бһҳбһ¶бһ“бһҗбһ¶бһҳбһ–бһӣ бһ”бһүбҹ’бһ…бҹҒбһүбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҳбһҪбһҷбһЎбһҫбһ„бһңбһ·бһүбһҶбһ¶бһ”бҹӢбҹ—бҹ”
> рҹҺ“ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҠбһ»бһҖбһ•бҹ’бһ‘бһ»бһҖбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ“бҹ…бһҸбҹӮбһҳбһ¶бһ“бһ“бҹ…бһ–бҹҒбһӣбһӮбҹ’бһҳбһ¶бһ“бһҗбһ¶бһҳбһ–бһӣбҹ”

> рҹҺ“ RAM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһЎбһҫбһ„бһңбһ·бһүбһ–бҹҒбһӣбһӮбҹ’бһҳбһ¶бһ“бһҗбһ¶бһҳбһ–бһӣ

бһҠбһјбһ…бһҮбһ¶бһҳбһҪбһҷ CPU бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһӣбһҫбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһӮбһәбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҸбһјбһ…бһҮбһ¶бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ PC бһ¬ Macбҹ” бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҳбһҪбһҷбһ’бһҳбҹ’бһҳбһҸбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“ RAM 8 Gigabytes (GB) бһ¬ 8,000,000,000 бһ”бҹғ бһҒбһҺбҹҲбһҠбҹӮбһӣбһ”бҹғбһҳбһҪбһҷбһӮбһәбһҮбһ¶вҖӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһўбһҖбҹ’бһҹбһҡбһҸбҹӮбһҳбһҪбһҷбһ¬бһӣбҹҒбһҒбһ–бһё 0-255бҹ” бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҳбһ¶бһ“ RAM бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮ Kilobytes (KB) бһ”бҹӮбһ”бһҠбҹӮбһӣбһҒбһёбһЎбһјбһ”бҹғбһҮбһ¶бһ”бҹ’бһҡбһҳбһ¶бһҺ 1,000 бһ”бҹғбҹ” Wio terminal бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҒбһ¶бһ„бһӣбһҫбһҳбһ¶бһ“ RAM 192KB бһ¬ 192,000 бһ”бҹғ - бһҸбһјбһ…бһҮбһ¶бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ–бһёбһҡбһҷбҹҲбһҮбһ¶бһ„ 40,000 бһҠбһ„!

бһӮбҹҶбһ“бһјбһҹбһҸбһ¶бһ„бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹғбһ‘бҹҶбһ бҹҶбһҡбһңбһ¶бһ„ 192KB бһ“бһ·бһ„ 8GB - бһ…бҹҶбһҺбһ»бһ…бһҸбһјбһ…бһ“бҹ…бһҖбһҺбҹ’бһҸбһ¶бһӣбһ”бһ„бҹ’бһ бһ¶бһү 192KBбҹ”

бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹҸбһҸбһјбһ…бһҮбһ¶бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ•бһ„бһҠбҹӮбһҡбҹ” бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ’бһҳбҹ’бһҳбһҸбһ¶ бһўбһ¶бһ…бһҳбһ¶бһ“бһҢбһёбһҹбһҠбҹ’бһҡбһ¶бһҷ 500GB бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҒбһҺбҹҲбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһўбһ¶бһ…бһҳбһ¶бһ“бһҸбҹӮбһҮбһ¶бһҖбһёбһЎбһјбһ”бҹғ бһ¬бһҳбҹҒбһҖбһ¶бһ”бҹғ (MB) бһҸбһ·бһ…бһҸбһҪбһ… (1MB бһӮбһә 1,000KB бһ¬ 1,000,000 бһ”бҹғ)бҹ” Wio terminal бһҳбһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё 4MB бҹ”

вң… бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһўбҹ’бһңбһёбһҳбһҪбһҷбҹ– бһҸбһҫ RAM бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҡбһ”бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? бһңбһ¶бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҮбһ¶бһҳбһҪбһҷбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһә?

### Input/Output

бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ/бһ”бһүбҹ’бһ…бҹҒбһү (I/O) бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҹбһ¶бҹҶбһ„бһҹбҹҗбһҡбһ“бһ·бһ„бһ•бҹ’бһүбһҫбһҹбһүбҹ’бһүбһ¶бһ”бһүбҹ’бһҮбһ¶бһҮбһјбһ“бһұбҹ’бһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһҮбһ¶бһҖбһҳбҹ’бһҳбҹ” бһ–бһҪбһҖбһңбһ¶бһ‘бһјбһ‘бҹ…бһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһҳбһҪбһҷбһ“бҹғбһ–бһ¶бҹҶбһ„бһ”бһүбҹ’бһ…бһјбһӣ/бһ”бһүбҹ’бһ…бҹҒбһүбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ‘бһјбһ‘бҹ… (GPIO)бҹ” бһ–бһ¶бҹҶбһ„бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҠбһјбһ…бһҮбһ¶бһ”бһүбҹ’бһ…бһјбһӣ (бһҡбҹҶбһӣбһ№бһҖбһҗбһ¶бһ–бһҪбһҖбһңбһ¶бһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣбһҹбһүбҹ’бһүбһ¶) бһ¬бһ”бһүбҹ’бһ…бҹҒбһү (бһ•бҹ’бһүбһҫбһҹбһүбҹ’бһүбһ¶)бҹ”

рҹ§ в¬…пёҸ бһ–бһ¶бҹҶбһ„бһ”бһүбҹ’бһ…бһјбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҹбһ¶бҹҶбһ„бһҹбҹҗбһҡ

рҹ§ вһЎпёҸ бһ–бһ¶бҹҶбһ„бһ”бһүбҹ’бһ…бҹҒбһүбһ•бҹ’бһүбһҫбһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһҮбһ¶

вң… бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһ“бҹҒбҹҮбһ“бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

#### бһҸбһҪбһ“бһ¶бһ‘бһё

бһҹбҹҠбһҫбһ”бһўбһ„бҹ’бһҖбҹҒбһҸ Wio Terminalбҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫ Wio Terminal бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бһ¶бҹҶбһ„ GPIOбҹ” бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бҹ’бһ“бҹӮбһҖ *Pinout diagram* бһ“бҹ…бһӣбһҫ [бһ‘бҹҶбһ–бҹҗбһҡбһ•бһӣбһ·бһҸбһ•бһӣ Wio Terminal](https://www.seeedstudio.com/Wio-Terminal-p-4509.html) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһҗбһ¶бһҸбһҫбһ–бһ¶бҹҶбһ„бһҺбһ¶бһўбҹ’бһңбһёбһҮбһ¶вҖӢбһ–бһ¶бҹҶбһ„бһҺбһ¶бҹ” Wio Terminal бһҳбһ¶бһ“бһҹбҹ’бһ‘бһёбһҖбһӮбһәбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбһ¶бһҖбҹӢбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбһҮбһ¶бһҳбһҪбһҷбһӣбҹҒбһҒбһ–бһ¶бҹҶбһ„ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҹбһјбһҳбһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһҘбһЎбһјбһңбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҳбһ¶бһ“бҹ”

бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһә бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҸбһјбһ…бһҮбһ¶бһ‘бһјбһ‘бҹ… бһҠбһјбһ…бһҮбһ¶ [Freescale Kinetis KL03 MCU бһҠбҹӮбһӣбһҸбһјбһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„ depresion бһҡбһ”бһҹбҹӢбһ”бһ¶бһӣбҹӢбһ бҹ’бһӮбҹ„бһӣ](https://www.edn.com/tiny-arm-cortex-m0-based-mcu-shrinks-package/)бҹ” CPU бһҸбҹӮбһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ PC бһўбһ¶бһ…бһҳбһ¶бһ“бһңбһ·бһҳбһ¶бһҸбҹ’бһҡбһҡбһ”бһҹбҹӢбһңбһ¶ 40mm x 40mm бһ бһҫбһҷбһ“бҹҒбҹҮбһҳбһ·бһ“бһҡбһ¶бһ”бҹӢбһ”бҹҶбһ–бһ„бҹӢбһҖбҹҶбһҠбҹ…бһ“бһ·бһ„бһҹбҹҶбһҺбһ№бһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶ CPU бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһӣбһҫбһҹбһ–бһёбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһңбһ·бһ“бһ¶бһ‘бһёбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһҳбҹ’бһҠбҹ…бһ…бҹ’бһҡбһҫбһ“бһ‘бҹҒбһ‘бҹҒ бһҠбҹӮбһӣбһ’бҹҶбһҮбһ¶бһ„бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһҳбһҪбһҷбһҳбҹүбҹҒбһ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ·бһ“бһ·бһҷбҹ„бһӮ Wio terminal бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһә бһҖбҹҒбһҹ бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ“бһ¶бһ“бһ¶бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ» бһӮбһәбһҳбһ·бһ“бһ’бҹҶбһҮбһ¶бһ„ CPU Intel i9 бһҜбһҖбҹ„ бһ“бһ·бһ„бһҸбһ·бһ…бһҮбһ¶бһ„ CPU бһҳбһ¶бһ“бһҹбҹҶбһҺбһ№бһҸбһҖбһҳбҹ’бһҠбҹ…бһ“бһ·бһ„бһҖбһёбһЎбһ¶бһӮбҹ’бһҡбһҝбһ„бһҸбҹ’бһҡбһҮбһ¶бһҖбҹӢбһ‘бҹҒ!

### Frames бһҮбҹҶбһ“бһҪбһҷ бһ“бһ·бһ„вҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’вҖӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ

бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһ”бҹ’бһҡбҹҒбһҹбҹҗбһҡбһҡбһҸбҹӢ бһ“бһ·бһ„бһ‘бҹҶбһ бҹҶбһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһ‘бһ¶бһ” бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ (OS) бһҸбһ¶бһҳбһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһ“бҹ…бһӣбһҫбһҸбһ»бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Windows, Linux бһ¬ macOS) бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҸбҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһЎбһҫбһҷбҹ” бһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһә бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹҶбһ–бҹҒбһүбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҳбһҪбһҷбһ¬бһ…бҹ’бһҡбһҫбһ“ бһҳбһ·бһ“бһҠбһјбһ…бһҮбһ¶вҖӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһјбһ‘бҹ…бһҠбһјбһ…бһҮбһ¶ PC бһ¬ Mac бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡвҖӢбһўбһ¶бҹҶбһ„бһ’бһҫбһ бҹ’бһңбҹҒбһҹбһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһўбҹ’бһ“бһҖбһҸбһ“бҹ’бһҡбҹ’бһҸбһё бһ¬бһңбһёбһҠбҹҒбһўбһј бһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҡбһҹбҹҒбһҡбһҜбһҖбһҹбһ¶бһҡбһ¬бһҖбһјбһҠ бһӣбҹҒбһ„бһ бҹ’бһӮбҹҒбһҳ бһ¬бһҡбһ»бһҖбһҡбһҖбһўбҹҠбһ·бһ“бһ’бһәбһҺбһ·бһҸбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһҘбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһұбҹ’бһҷбһўбһ¶бһ…бһҖбһҹбһ¶бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһўбһ¶бһ…бһҡбһҸбҹӢбһ”бһ¶бһ“ бһ”бҹ’бһҡбһҫ API бһҠбҹӮбһӣбһўбһ¶бһ…бһ“бһ·бһҷбһ¶бһҷбһ‘бҹ…бһ“бһ№бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһҮбһ¶ бһ”бһёбҹ’бһҮбһҪбһҸбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣ бһӮбһәбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ бҹҒбһҸбһ»бһ•бһӣбһҠбһҫбһҳбһ•бһӣбһ·бһҸбһ•бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶бһ“бһ№бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡ frames бһҮбҹҶбһ“бһҪбһҷбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҸбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ 'бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ' бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ„бҹӢбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ frames бһ“бҹ„бҹҮбҹ”

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ - бһҠбҹӮбһӣбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ бҹ…бһҗбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бҹ’бһҡбһ¶бһҹ (RTOS) бһҠбһјбһ…бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹ… бһ“бһ·бһ„бһҳбһҖбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһҮбһ¶ бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ‘бҹҶбһ„бһ“бҹӢбһҒбҹ’бһ‘бһ¶бһҸбһ”бҹҶбһ•бһ»бһҸ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҠбһјбһ…бһҮбһ¶бҹ–

* бһҳбһҫбһӣбһҗбҹӮбһҖбһ¶бһҡбһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бҹ’бһҡбһҫбһ“бһҮбһҪбһҡ (multi-threading) бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҡбһҸбҹӢblock бһҖбһјбһҠбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһҠбһјбһ…бһҮбһ¶бһҸбһ¶бһҳбһҖбһјбһҡ(cores) бһ…бҹ’бһҡбһҫбһ“ бһ¬бһ”бҹ’бһҠбһјбһҡбһ…бһ»бҹҮбһҮбһ»бҹҶбһӣбһҫбһҖбһјбһҡбҹЎ
* бһ”бһҺбҹ’бһҠбһ¶бһү бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбһ¶бһ…бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
* бһ‘бҹҶбһ”бһ“бҹӢбһ…бҹҶбһҺбһ»бһ…бһўбһ¶бҹҶбһ„бһ’бһҫбһ бҹ’бһңбҹҒбһҹбһҡбһјбһ”бһ—бһ¶бһ– (GUI) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ„бҹӢбһўбһ¶бҹҶбһ„бһ’бһҫбһ бҹ’бһңбҹҒбһҹбһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (UI) бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбҹ”

вң… бһўбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё RTOS бһ–бһёбһҡ: [Azure RTOS](https://azure.microsoft.com/services/rtos/?WT.mc_id=academic-17441-jabenn), [FreeRTOS](https://www.freertos.org), [Zephyr](https://www.zephyrproject.org)

[Arduino](https://www.arduino.cc) бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҮбһ¶ frames бһҮбҹҶбһ“бһҪбһҷбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһӣбҹ’бһ”бһёбһӣбҹ’бһ”бһ¶бһүбһҮбһ¶бһ„бһӮбҹҒ бһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ·бһҹбҹ’бһҹ бһҳбһ“бҹ„бһӮбһҳбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбҹ” Arduino бһӮбһәбһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһҹбҹҖбһңбһ—бҹ…бһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһ…бһ”бһҫбһҖбһ…бҹҶбһ бһҡбһҠбҹӮбһӣбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ¶бҹҶбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ·бһ„бһҡбҹүбһјбһ”бһјбһҸбһ·бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бһ·бһүбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„ Arduino бһҠбҹӮбһӣбһҹбһҳбһ“бһ№бһ„ Arduino бһҳбһҖбһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бһӣбһ·бһҸбһ•бһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ¬бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһ•бҹ’бһҹбҹҒбһ„бҹ— бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ”бҹ’бһҡбһҫ frames Arduino бҹ”

бһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„ Arduino бһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶ C бһ¬ C++бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ C/C++ бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһ’бҹ’бһңбһҫбһҮбһ¶бһҖбһјбһҠбһҸбһјбһ…бҹ— бһ“бһ·бһ„бһҡбһҸбҹӢбһӣбһҝбһ“ бһҠбҹӮбһӣбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбһјбһ…бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбҹ” бһҹбҹ’бһ“бһјбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Arduino бһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһҗбһ¶ sketch бһ бһҫбһҷбһҮбһ¶бһҖбһјбһҠ C/C++ бһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡ 2 бһҲбҹ’бһҳбҹ„бҹҮ `setup` бһ“бһ·бһ„ `loop`бҹ” бһ–бҹҒбһӣбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһ•бһ»бһҹбһЎбһҫбһ„бһҖбһјбһҠбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ frames Arduino бһ“бһ№бһ„бһҡбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһҳбҹ’бһҸбһ„бһҳбһҪбһҷ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһңбһ¶бһ“бһ№бһ„бһҡбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ `loop` бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮбһ”бһ“бҹ’бһҸбһҡбһ бһјбһҸбһҠбһӣбҹӢбһҗбһ¶бһҳбһ–бһӣбһҒбһ¶бһҸбҹ”

бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ•бҹ’бһҸбһҫбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… WiFi бһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ–бһ–бһҖ бһ¬бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҖбҹҶбһҺбһҸбҹӢбһ–бһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бҹҒбһүбҹ” бһҖбһјбһҠбһ‘бһ¶бһүбһҳбһ»бһҒбһ„бһ¶бһҡ `loop` бһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһјбһҠбһҖбҹҶбһҺбһҸбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһўбһ¶бһ“бһ–бһёбһҹбһ¶бҹҶбһ„бһҹбҹҗбһҡбһ“бһ·бһ„бһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ–бһ–бһҖбҹ” бһҮбһ¶бһ‘бһјбһ‘бҹ… бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҷбһәбһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ— (loop) бһ§. бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ¶бҹҶбһ„бһҹбҹҗбһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһӮбҹ’бһҡбҹ„бһ„бһҡбһҖбҹӢ 10 бһңбһ·бһ“бһ¶бһ‘бһё бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһҷбһәбһҸ 10 бһңбһ·бһ“бһ¶бһ‘бһёбһ“бҹ…бһ…бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһўбҹ„бһҷбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһәбһўбһ¶бһ…бһЎбһҫбһ„бһҹбҹ’бһҠбһ¶бҹҶ бһ“бһ·бһ„бһҹбһ“бҹ’бһҹбҹҶбһҗбһ¶бһҳбһ–бһӣ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҡбһҸбҹӢбһЎбһҫбһ„бһӣбһҫбһ‘бҹҖбһҸбһ“бҹ…бһ–бҹҒбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ“бҹ…бһ–бҹҒбһӣ 10 бһңбһ·бһ“бһ¶бһ‘бһёбһ‘бҹҖбһҸбҹ”

вң… бһңбһ·бһҳбһ»бҹҮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһӮбһәбһ бҹ…бһҗбһ¶ *event loop* бһ¬ *message loop*бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһҫбһҹбҹ’бһ‘бһ¶бһҸбҹӢбһҖбҹ’бһҡбҹ„бһҳбһ–бһёбһҡбһ бҹ„бһ„бһ“бҹҒбҹҮ бһ бһҫбһҷбһӮбһәбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбһ»бһҸбһјбһ…бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбһјбһ…бһҮбһ¶ Windows, macOS бһ¬ Linuxбҹ” `loop` бһҡбһ„бҹӢбһ…бһ¶бҹҶбһҹбһ¶бһҡбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚ UI бһҠбһјбһ…бһҮбһ¶бһ•бҹ’бһҠбһ¶бһҖбҹӢбһ”бҹҠбһјбһҸбһ»бһ„ бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбһјбһ…бһҮбһ¶ бһҸбһҳбҹ’бһҡбһ„бҹӢбһ–бһ¶бһҖбҹ’бһҷ бһ“бһ·бһ„бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһўбһҸбҹ’бһҗбһ”бһ‘бһўбҹҶбһ–бһё event loop](https://wikipedia.org/wiki/Event_loop)бҹ”

Arduino бһ•бҹ’бһҸбһӣбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһЎбһә бһ“бһ·бһ„бһ–бһ¶бҹҶбһ„ I/O бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһ‘бһ¶бһҸбҹӢбһҖбҹ’бһҡбҹ„бһҳ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҸбҹӢбһӣбһҫбһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ§. бһҳбһ»бһҒбһ„бһ¶бһҡ [`delay`](https://www.arduino.cc/reference/en/language/functions/time/delay/) бһ“бһ№бһ„бһҲбһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһҷбҹҲбһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢ бһҳбһ»бһҒбһ„бһ¶бһҡ [`digitalRead`](https://www.arduino.cc/reference/en/language/functions/digital-io/digitalread/) бһ“бһ№бһ„бһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ `HIGH` бһ¬ `LOW` бһ–бһёбһ–бһ¶бҹҶбһ„бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҖбһјбһҠбһҡбһҸбҹӢ бһ‘бҹ„бҹҮбһҮбһ¶бһ”бҹҠбһјбһҸбһ»бһ„бһ“бҹ„бҹҮбһҹбҹ’бһҗбһ·бһҸбһӣбһҫбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһҺбһ¶бһҳбһҪбһҷбҹ” бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ’бһ¶бһ“бһ¶бһҗбһ¶ бһҖбһјбһҠ Arduino бһҠбҹӮбһӣбһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһӣбһҫбһӮбҹ’бһҡбһ¶бһ“бҹӢбһ”бҹ’бһҠбһјбһҡ бһҖбҹҸбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ compile бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„ Arduino бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ”бһ¶бһ“ бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ–бһ¶бҹҶбһ„бһӮбһәбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбҹҶбһ–бһҫбһҠбҹӮбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

бһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ‘бһёбһ”бһёвҖӢбһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Arduino бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„ Arduino бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҹбһ„бҹӢбһҹSensor бһ“бһ·бһ„ actuator бһ¬бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ IoT бһӣбһҫбһ–бһ–бһҖбҹ”

##### бһҸбһҪбһ“бһ¶бһ‘бһё

бһҹбҹҠбһҫбһ”бһўбһ„бҹ’бһҖбҹҒбһҸ Wio Terminalбҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫ Wio Terminal бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҹбһјбһҳбһўбһ¶бһ“бһҳбһҫбһӣбһҖбһјбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһ“бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ»бһ“бҹ” бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһ“бһ·бһ„ `loop`бҹ” бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹҒбһүбһҸбһ¶бһҳбһҡбһҷбҹҲ serial output бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ loop бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ” бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҡбһҹбҹҒбһҡбһҠбһӣбҹӢбһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһё бһ“бһ·бһ„бһҳбһҫбһӣбһҗбһ¶бһҖбһјбһҠбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҸбҹӮбһҳбҹ’бһҠбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҠбһ„бһ”бһҫбһҖбһЎбһҫбһ„бҹ” бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһҫбһҖбһЎбһҫбһ„бһңбһ·бһүбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹҠбһјбһҸбһ»бһ„бһҗбһ¶бһҳбһ–бһӣбһ“бҹ…бһҖбҹ’бһ”бҹӮбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ бҹ…бһҳбҹ’бһҸбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҠбһ„бһ”бһҫбһҖбһЎбһҫбһ„бҹ”

## бһ…бҹҶбһҺбһҸбҹ’бһҡбһ¶бһ”бҹӢбһңбҹӮбһ„бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҗбһҸбһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһҸбһ¶бһҡ

бһ“бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ»бһ“ бһҷбһҫбһ„бһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҗбһҸбһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһҸбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бҹ”

### Raspberry Pi

![бһҡбһјбһ”бһҹбһүбҹ’бһүбһ¶ Raspberry Pi](../../../../../translated_images/km/raspberry-pi-logo.4efaa16605cee054.webp)

[бһҠбҹғбһӮбһјбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ Raspberry Pi](https://www.raspberrypi.org) бһӮбһәбһҮбһ¶бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҳбһ·бһ“бһҡбһҖбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ…бҹҶбһҺбҹҒбһүбһ–бһёбһ…бһҖбҹ’бһҡбһ—бһ–бһ бҹ’бһҹбһҒбһ¶бһ„бһӣбһ·бһ…бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһҶбҹ’бһ“бһ¶бҹҶ 2009 бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһўбҹҶбһ–бһёбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ“бҹ…бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ¶бһӣбһ¶бһҡбҹҖбһ“бҹ” бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮ бһ–бһҪбһҖбһӮбҹҒбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҗбһҸбһҸбҹӮбһҳбһҪбһҷ бһҖбҹ’бһҡбҹ„бһҳбһҲбҹ’бһҳбҹ„бҹҮ Raspberry Piбҹ” Raspberry Pi бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘ 3 бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҹбҹҶбһҒбһ¶бһ“бҹӢ - бһҡбһјбһ”бһҡбһ¶бһ„бһ‘бҹҶбһ бҹҶбһ–бҹҒбһүбһӣбҹҒбһү бһҳбһҪбһҷ Pi Zero бһҸбһјбһ…бһҮбһ¶бһ„ бһ“бһ·бһ„бһҳбҹүбһјбһҢбһ»бһӣбһҖбҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ (compute module) бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ•бҹ’бһӣбһјбһңбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҗбҹ’бһҳбһёбһ”бҹҶбһ•бһ»бһҸбһ“бҹғ Raspberry Pi бһ‘бҹҶбһ бҹҶбһ–бҹҒбһүбһӣбҹҒбһүбһӮбһә Raspberry Pi 4Bбҹ” бһңбһ¶бһҳбһ¶бһ“ CPU бһ–бһёбһҡбһјбһ”бһҖбһҡбһҺбһё (quad-core) бһҡбһҸбҹӢбһӣбҹ’бһ”бһҝбһ“ 1.5GHz бһҳбһ¶бһ“ RAM бһҮбһ¶ 2, 4 бһ¬ 8GB бһҳбһ¶бһ“бһ‘бһёбһҸбһ¶бҹҶбһ„бһўбҹҠбһёбһ’бһәбһҺбһ·бһҸ (gigabit ethernet) WiFi бһ…бҹ’бһҡбһҖ HDMI 2 бһ…бҹ’бһҡбһҖ бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ 4k бһ…бҹ’бһҡбһҖбһ”бһүбҹ’бһ…бҹҒбһүбһҹбҹҶбһЎбҹҒбһ„ бһ“бһ·бһ„бһңбһёбһҠбҹҒбһўбһј бһҹбһҳбһ¶бһҹбһ—бһ¶бһӮ USB (2 USB 2.0 бһ“бһ·бһ„ 2 USB 3.0) бһ–бһ¶бҹҶбһ„ GPIO 40 бһҒбҹ’бһҹбҹӮбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶ SD card slot бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһҸбһ¶бһҡбһ¶бһ„бһңбһ¶бһҹбҹӢбһ‘бҹҶбһ бҹҶ 88mm x 58mm x 19.5mm бһ“бһ·бһ„бһҳбһ¶бһ“бһҗбһ¶бһҳбһ–бһӣбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ USB-C 3Aбҹ” бһҸбһҳбҹ’бһӣбҹғбһ…бһ¶бһ”бҹӢбһ–бһё 35 бһҠбһ»бһӣбҹ’бһӣбһ¶бһҡ бһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһ„ PC бһ¬ Macбҹ”

> рҹ’Ғ бһҖбҹҸбһҳбһ¶бһ“бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҸбҹӮбһҳбһҪбһҷ Pi400 бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢ Pi4 бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҸбһ¶бһҡбһ…бһ»бһ…бҹ”

![Raspberry Pi Zero](../../../../../translated_images/km/raspberry-pi-zero.f7a4133e1e7d54bb.webp)

Pi Zero бһҸбһјбһ…бһҮбһ¶бһ„ бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҗбһ¶бһҳбһ–бһӣбһ‘бһ¶бһ”бҹ” бһңбһ¶бһҳбһ¶бһ“ CPU бһҳбҹүбһјбһҺбһјбһҖбһјбһҡ 1GHz RAM 512MB WiFi (бһҳбҹүбһјбһҠбҹӮбһӣ Zero W) бһ…бҹ’бһҡбһҖ HDMI бһҸбҹӮбһҳбһҪбһҷ бһ…бҹ’бһҡбһҖ micro-USB бһ–бһ¶бҹҶбһ„ GPIO 40 бһҒбҹ’бһҹбҹӮ бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶ бһ“бһ·бһ„ SD card slotбҹ” бһңбһ¶бһҳбһ¶бһ“бһңбһ·бһҳбһ¶бһҸбҹ’бһҡбһ“бҹ… 65mm x 30mm x 5mm бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбһҸбһ·бһ…бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ” бһҸбһҳбҹ’бһӣбҹғ Zero бһӮбһә 5 бһҠбһ»бһӣбҹ’бһӣбһ¶бһҡ бһ бһҫбһҷбһҳбҹүбһјбһҠбҹӮбһӣ W бһҳбһ¶бһ“ WiFi бһҸбһҳбҹ’бһӣбҹғ 10 бһҠбһ»бһӣбҹ’бһӣбһ¶бһҡбҹ”

> рҹҺ“ CPU бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ•бһӣбһ·бһҸбһ•бһӣ ARM бһҳбһ·бһ“бһҳбҹӮбһ“бһҠбһјбһ…бһҮбһ¶ Intel/AMD бһ”бҹ’бһҡбһ—бҹҒбһ‘ x86 бһ¬ x64 бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһӣбһҫ PC бһ“бһ·бһ„ Mac бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҒбҹ” бһңбһ¶бһҠбһјбһ…бһ“бһ№бһ„ CPU бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһғбһҫбһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣ бһ“бһ·бһ„бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһ…бһӣбҹҗбһҸбһ…бҹ’бһҡбһҫбһ“ бһӮбҹ’бһҡбһҝбһ„ Microsoft Surface X бһ“бһ·бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ“бҹғ Apple Silicon бһӣбһҫ Mac бһ“бһ¶бһ–бҹҒбһӣбһҗбҹ’бһҳбһёбһ“бҹҒбҹҮбҹ”

Variant бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғ Raspberry Pi бһҡбһҸбҹӢ Debian Linux бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘ Raspberry Pi OSбҹ” бһңбһ¶бһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹӮбһ”бһңбҹӮбһ„бһҖбһҳбҹ’бһҡбһ·бһҸ lite бһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“ desktop бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„ 'headless' бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһЎбһ»бһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһ¬бһ”бҹӮбһ”бһңбҹӮбһ„бһ–бҹҒбһүбһӮбһҺбҹҲ desktop бһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ browser бһңбҹҒбһ” бһҹбһјбһ бҹ’бһңбһңбҹҒбһўбҹҠбһёбһ бҹ’бһңбһ·бһҹ бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ бһ“бһ·бһ„бһ бҹ’бһӮбҹҒбһҳбҹ” бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһӮбһә Debian Linux бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Debian бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ CPU ARM бһҖбҹ’бһ“бһ»бһ„ Piбҹ”

#### бһҸбһҪбһ“бһ¶бһ‘бһё

бһҹбҹҠбһҫбһ”бһўбһ„бҹ’бһҖбҹҒбһҸ Raspberry Piбҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫ Raspberry Pi бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҹбһјбһҳбһўбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ‘бһ»бһ“бһ“бһ·бһ„бһҹбһҳбһ¶бһҹбһ—бһ¶бһӮбһ“бҹ…бһӣбһҫбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бҹ”

* бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһё CPU бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ“бҹ…бһӣбһҫ [бһ‘бҹҶбһ–бҹҗбһҡбһҜбһҖбһҹбһ¶бһҡбһ•бһӣбһ·бһҸбһ•бһӣ Raspberry Pi](https://www.raspberrypi.org/documentation/hardware/raspberrypi/). бһўбһ¶бһ“бһ–бһё CPU бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ“бҹ…бһҡбһ”бһҖ Pi бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбҹ”
* бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бһ¶бҹҶбһ„ GPIOбҹ” бһўбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһ–бһ¶бҹҶбһ„бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бҹ… [бһҜбһҖбһҹбһ¶бһҡбһ–бһё GPIO Raspberry Pi](https://www.raspberrypi.org/documentation/hardware/raspberrypi/gpio/README.md)бҹ” бһ”бҹ’бһҡбһҫ [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GPIO Pin](https://www.raspberrypi.org/documentation/usage/gpio/README.md) бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ–бһ¶бҹҶбһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһӣбһҫ Pi бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

### бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҗбһҸбһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһҸбһ¶бһҡ

бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҗбһҸбһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһҸбһ¶бһҡбһҮбһ¶бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһүбҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҳбһ¶бһ“бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҮбҹҶбһ“бһҪбһҷ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбһјбһҠбһҠбҹҸбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһҪбһҖбһңбһ¶ бһҳбһ·бһ“бһҠбһјбһ…бһҮбһ¶бһҳбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһҠбҹӮбһӣбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҸбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһҖбҹ’бһ“бһ»бһ„ frames бһҠбһјбһ…бһҮбһ¶ Arduinoбҹ” бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ–бһ¶бҹҶбһ„ GPIO бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫ бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҹбһ¶бҹҶбһ„бһҹбҹҗбһҡбһ“бһ·бһ„ actuatorбҹ”

вң… бһҸбһҫбһўбҹ’бһ“бһҖбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ? бһҸбһҫбһ—бһ¶бһҹбһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһӣбһҫ Linux бһ‘бҹҒ?
бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһӮбҹҒбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“бһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё IoT бһӣбһҫ Raspberry Pi бһӮбһә Pythonбҹ” бһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһ’бҹҶбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ бһ¶бһҠбһңбҹҒбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Pi бһ бһҫбһҷбһҮбһ·бһҸбһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбһјбһҠбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бһҮбһ¶бһӘбһ”бһҖбһҡбһҺбҹҚбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Pythonбҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҒбҹ’бһӣбҹҮбҹ—бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ 'hats' - бһ бҹ…бһҷбҹүбһ¶бһ„бһ“бҹҒбҹҮбһ–бҹ’бһҡбҹ„бҹҮбһ–бһҪбһҖбһңбһ¶бһўбһ„бҹ’бһӮбһ»бһҷбһӣбһҫ Pi бһҠбһјбһ…бһҮбһ¶бһҖбҹ’бһҡбһЎбһ¶бһ бһҫбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҹбҹҠбһҖбһҒбҹ’бһҹбҹӮбһ’бҹҶбһ‘бҹ…бһҖбһ¶бһ“бҹӢ 40 бһ–бһјбһӣ GPIOбҹ” бһ–бһҪбһҖ hats бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһ“бҹ’бһҗбҹӮбһҳ бһҠбһјбһ…бһҮбһ¶ бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһ—бҹ’бһ“бһҪбһ„ бһЎбһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷ бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһӮбһјбһҡбһ•бҹ’бһӮбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ—бҹ’бһ“бһҪбһ„бһҮбһ¶бһҳбһҪбһҷбһҒбҹ’бһҹбҹӮбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбҹ”

### бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҸбҹӮбһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ IoT бһҮбһ¶бһҮбҹҶбһ“бһ¶бһүбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ

бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҸбҹӮбһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ IoT бһҮбһ¶бһҮбҹҶбһ“бһ¶бһүбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ бһҳбһ·бһ“бһҳбҹӮбһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһЎбһҫбһ„бһҳбһҪбһҷбһ‘бҹҒбҹ” бһ–бһҪбһҖбһңбһ¶бһўбһ¶бһ…бһ•бҹ’бһҸбһӣбҹӢбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһүбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ бһ¶бһҠбһңбҹҒбһҡбһ“бһ·бһ„бһҡбһҸбҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҡбһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ¶бһ“ [бһҳбҹүбһјбһҢбһ»бһӣ Raspberry Pi 4 compute module](https://www.raspberrypi.org/blog/raspberry-pi-compute-module-4/) бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбһјбһ… Raspberry Pi 4 бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹ’бһ“бһ»бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбһјбһ… бһ“бһ·бһ„бһҗбҹ’бһӣбҹғбһҗбҹ„бһҖбһҮбһ¶бһ„ бһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһ…бҹ’бһҡбһҖбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһЎбһҫбһ„бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”

## рҹҡҖ бһ”бһүбҹ’бһ бһ¶

## бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ

* бһўбһ¶бһ“ [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Arduino](https://www.arduino.cc/en/Guide/Introduction) бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҠбһ№бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһңбҹҒбһ‘бһ·бһҖбһ¶ Arduinoбҹ”
* бһўбһ¶бһ“ [бһҖбһ¶бһҡвҖӢбһҠбҹӮбһӣвҖӢбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶвҖӢбһ–бһё Raspberry Pi 4](https://www.raspberrypi.org/products/raspberry-pi-4-model-b/) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё Raspberry Pisбҹ”
* бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһҫбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ“бҹғбһӮбҹҶбһ“бһ·бһҸ бһ“бһ·бһ„бһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘ [What the FAQ are CPUs, MPUs, MCUs, and GPUs бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһҹбҹ’бһҹбһ“бһ¶бһңбһ·бһҮбҹ’бһҮбһ¶бһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё Engineering Journal](https://www.eejournal.com/article/what-the-faq-are-cpus-mpus-mcus-and-gpus/)бҹ”

вң… бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҸбһ¶бһҳбһҸбҹҶбһҺбҹҮбһ“бҹ… [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ бһ¶бһҠбһңбҹӮбһҡбһҡбһ”бһҹбҹӢ](../../../hardware.md) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһ бһ¶бһҠбһңбҹӮбһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ¬бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһҫбһҡбһ…бһўбһҹбҹӢбҹ”
