# ការចាប់ផ្តើមជាមួយ IoT

ក្នុងផ្នែកនេះនៃមេរៀន អ្នកនឹងត្រូវបានណែនាំអំពីអន្ដរជាតិនៃអ្វីៗ ហើយរៀនពីគំនិតមូលដ្ឋានរួមទាំងការសង់គម្រោង IoT 'Hello World' ដំបូងរបស់អ្នកដែលភ្ជាប់ទៅកាន់ពពក។ គម្រោងនេះគឺជាឡាំភ្លើងយាមដែលភ្លឺពេលកម្រិតရှိវាស់ដោយឧបករណ៍ស្ទង់ភ្លឺធ្លាក់ចុះ។
