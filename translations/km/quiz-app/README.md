бһ“бҹҒбҹҮбһҮбһ¶бһҳбһҠбҹҖбһӣбһҮбҹҶбһ бһ¶бһ“вҖӢбһ…бһ¶бһ”бҹӢбһ–бһёбһҠбһҫбһҳбһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ–

1. бһ•Fork бһҸбһ¶бһҡбһ¶бһ„ GitHub
бһ’бһ¶бһ“бһ¶бһҗбһ¶вҖӢбһҖбһјбһҠбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһҺбҹ’бһҸбһ¶бһү static бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„ GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” Fork бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ”

2. бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Azure Static Web App
- бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„ [бһӮбһҺбһ“бһё Azure](http://azure.microsoft.com)
- бһ…бһјбһӣвҖӢбһ‘бҹ… [бһ•бҹ’бһ‘бһ¶бҹҶбһ„ Azure portal](https://portal.azure.com) 
- бһ…бһ»бһ…бһӣбһҫ вҖңCreate a resourceвҖқ бһ бһҫбһҷбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ вҖңStatic Web AppвҖқбҹ”
- бһ…бһ»бһ… вҖңCreateвҖқбҹ”

3. бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Static Web App
- бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ– бһҖбһ¶бһҡбһҮбһ¶бһңбҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһ¶бһҡбһҮбһ¶бһң Azure бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
- бһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“бһҗбҹ’бһҳбһё бһ¬бһ”бҹ’бһҡбһҫбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ”
- бһҲбҹ’бһҳбҹ„бҹҮбҹ– бһ•бҹ’бһҸбһӣбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё static web бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
- бһҸбҹҶбһ”бһ“бҹӢбҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһ“бҹ…бһҮбһ·бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”

- #### бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбҹ–
- бһ”бҹ’бһҡбһ—бһ–бҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ вҖңGitHubвҖқбҹ”
- бһӮбһҺбһ“бһё GitHubбҹ– бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷ Azure бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһӮбһҺбһ“бһё GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
- бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбһ„бҹ’бһӮбһ—бһ¶бһ– GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
- бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһғбҹ’бһӣбһ¶бҹҶбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё static web бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
- бһҹбһ¶бһҒбһ¶бҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбһ¶бһҒбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ–бһёбһңбһ¶бҹ”

- #### бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһ¶бһ„бһҹбһ„бҹӢбҹ–
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷ (бһ§. React, Angular, Vue бһӣбҹ”)бҹ”
- бһ‘бһёбһҸбһ¶бҹҶбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ– бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһ§. / бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһңбһ¶бһҮбһ¶бһҡбһҝбһ“бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„)бҹ”
- бһ‘бһёбһҸбһ¶бҹҶбһ„ APIбҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“ API бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһңбһ¶ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ)бҹ”
- бһ‘бһёбһҸбһ¶бҹҶбһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ– бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ¶бһ„бһҹбһ„бҹӢ (бһ§. build бһ¬ dist)бҹ”

4. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ
бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ…бһ»бһ… вҖңCreateвҖқбҹ” Azure бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“бһ‘бһ¶бһҳбһ‘бһ¶бһҡ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ•бһӣбһ·бһҸ GitHub Actions workflow бһӣбһҫбһғбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

5. GitHub Actions Workflow
Azure бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ GitHub Actions workflow бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (.github/workflows/azure-static-web-apps-<name>.yml) бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” Workflow бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢ бһ“бһ·бһ„бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбҹ”

6. бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ
бһ…бһјбһӣбһ‘бҹ…бһҗбҹҒбһ” вҖңActionsвҖқ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„ GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһғбһҫбһү workflow бһҖбҹҶбһ–бһ»бһ„бһҡбһҸбҹӢбҹ” Workflow бһ“бҹҒбҹҮбһ“бһ№бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢ бһ“бһ·бһ„бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё static web бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ… Azureбҹ”
бһ–бҹҒбһӣ workflow бһ”бһүбҹ’бһ…бһ”бҹӢ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ•бҹ’бһҹбһ¶бһҷбһ”бһ“бҹ’бһҸбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҫ URL Azure бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбҹ”

  push:
    branches:

    types: [opened, synchronize, reopened, closed]
    branches:

    name: Build and Deploy Job
    steps:

      - name: Build And Deploy

        with:

          repo_token: ${{ secrets.GITHUB_TOKEN }}
          action: "upload"
          app_location: "quiz-app" #App source code path
          api_location: ""API source code path optional
          output_location: "dist" #Built app content directory - optional
