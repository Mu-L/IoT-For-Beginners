# គាំទ្រ

## របៀបដាក់បញ្ហានិងទទួលបានជំនួយ

គម្រោងនេះប្រើ GitHub Issues ដើម្បីតាមដានកម្រិតកំហុសនិងការស្នើសុំមុខងារ។ សូមស្វែងរកបញ្ហាមុនសិនមុនដាក់បញ្ហាថ្មីដើម្បីជៀសវាងការចម្លង។ សម្រាប់បញ្ហាថ្មីៗ សូមដាក់កំហុសឬស្នើសុំមុខងាររបស់អ្នកជា Issue ថ្មី។

សម្រាប់ជំនួយនិងសំណួរអំពីការប្រើគម្រោងនេះ សូមទំនាក់ទំនងមកយើងតាមការដាក់បញ្ហានៅក្នុង repo នេះ។

## គោលនយោបាយគាំទ្ររបស់ Microsoft
