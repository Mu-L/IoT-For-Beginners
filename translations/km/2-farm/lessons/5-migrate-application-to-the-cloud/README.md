## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹҶбһҺбһҫбһҳбһҠбһёбһҡбһ»бһҖбҹ’бһҒбһҮбһ¶бһҸбһ· бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бһҡбһ“бҹ’бһҸбһҹбҹҶбһҡбһ¶бһ”бҹӢ relay бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ IoT бһ“бҹ…бһӣбһҫбһ–бһ–бһҖбҹ” бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбһәбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҖбһјбһҠ server бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбһ”бһҹбҹӢ relay бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ–бһ–бһҖбҹ” бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһҡбһ”бҹҖбһ”бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ serverlessбҹ”

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–

* [бһҳбһ»бһҒбһ„бһ¶бһҡ serverless бһӮбһәбһўбҹ’бһңбһё?](#бһҳбһ»бһҒбһ„бһ¶бһҡ-serverless-бһӮбһәбһўбҹ’бһңбһё)
* [бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё serverless](#бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё-serverless)
* [бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбҹүбһәбһҳбҹүбҹ’бһҹбҹҠбһёбһңбһңбҹҒбһ“бһ•бҹ’бһ‘бҹғбһҖбҹ’бһ“бһ»бһ„ IoT Hub](#бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһ•бҹ’бһҹбһ¶бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ-iot-hub)
* [бһ•бҹ’бһүбһҫбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҡбһҹбһ»бҹҶбһ–бһ·бһ’бһёбһҖбһ¶бһҡвҖӢбһҸбҹ’бһҡбһ„бҹӢбһ–бһёбһҖбһјбһҠ serverless](#бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһ—бһ¶бһ–direct-method-бһ–бһёбһҖбһјбһҠ-serverless)
* [бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠ serverless бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһ–бһ–бһҖ](#бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҖбһјбһҠ-serverless-бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҳбҹҒбһғ)

## бһҳбһ»бһҒбһ„бһ¶бһҡ serverless бһӮбһәбһўбҹ’бһңбһё?

Serverless, бһ¬бһҖбҹҸбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡ serverless, бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһӣбһ»бһҖбһҸбһјбһ…бҹ—бһ“бҹғбһҖбһјбһҠбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһ–бһ–бһҖбһҮбһ¶бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ“бҹғбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹҶбһ–бһёбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ“бҹ„бҹҮбҹ” бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһҳбһҖбһ–бһёбһҡбһҝбһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ•бҹ’бһҹбҹҶбһЎбһҫбһ„бһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҡбһҹбһ»бҹҶбһңбҹҒбһ”бһҹбһ¶бһҷ, бһҹбһ¶бһҡвҖӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡ, бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһ¬бһҹбһ¶бһҡвҖӢбһҠбҹӮбһӣбһ•бҹ’бһүбһҫбһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ IoT бһҠбҹ„бһҷбһ§бһ”бһҖбһҡбһҺбҹҚ IoTбҹ”

бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһёбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҮбһ¶бһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡ serverless бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ”

> рҹҺ“ Serverless бһҖбҹҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҗбһ¶ Functions as a Service (FaaS) бһ–бҹ’бһҡбҹ„бҹҮбһҡбһ¶бһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё trigger event бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбҹ”

бһ‘бҹ„бҹҮбһ”бһёбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһӮбҹҒ Serverless бһҸбҹӮбһңбһ¶бһҖбҹҸбһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһ•бһ„бһҠбҹӮбһҡбҹ” бһҲбҹ’бһҳбҹ„бҹҮбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ бһўбҹ’бһ“бһҖбһҮбһ¶бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ–бҹ’бһҡбһҪбһҷбһҖбҹ’бһҡбҹ’бһҡбһҳбһҒбҹ’бһӣбһҪбһ“бһўбҹҶбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбҹӮбһӣбһ‘бһ¶бһҳбһ‘бҹ’бһҡбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹҒ бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбһәбһўбҹ’бһ“бһҖбһҖбҹҸбһ…бһ„бҹӢбһІбҹ’бһҷбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹҶбһ–бҹ„бҹҮбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбҹ” бһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһ–бһ–бһҖбһҳбһ¶бһ“ runtime serverless бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ•бҹ’бһҠбһӣбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ бһ”бһҺбҹ’бһҠбһ¶бһү бһҹбҹ’бһҸбһ»бһҖ CPU бһ…бһ„бһ…бһ¶бҹҶ бһ“бһ·бһ„бһўбҹ’бһңбһёбҹ—бһ‘бҹҖбһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҗбһ¶ бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҮбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бҹҒ бһ–бҹ’бһҡбҹ„бҹҮбһӮбҹ’бһҳбһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһ‘бҹҒбҹ” бһ•бһӣбһ·бһҸбһ•бһӣбһӮбһәбһўбҹ’бһ“бһҖбһ”бһ„бҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһ…бҹҶбһ“бһҪбһ“бһ…бһ„бһ…бһ¶бҹҶбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбҹ”

> рҹ’° Serverless бһҮбһ¶бһҡбһ”бҹҖбһ”бһҗбҹ’бһӣбҹғбһҗбҹ„бһҖбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбһӣбһҫбһ–бһ–бһҖбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бҹ…бһ–бҹҒбһӣбһҖбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһҹбһҡбһҹбҹҒбһҡ бһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһ–бһ–бһҖбһҳбһҪбһҷбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ»бһҒбһ„бһ¶бһҡ serverless бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһ¶бһҺ бҹЎ,бҹ бҹ бҹ ,бҹ бҹ бҹ  бһҠбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҒбҹӮбһҳбһ»бһ“бһ–бҹҒбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ“бҹ„бҹҮбһӮбҹҒбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ US$0.20 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бҹЎ,бҹ бҹ бҹ ,бҹ бҹ бҹ  бһҠбһ„бһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһ“бҹ…бһ–бҹҒбһӣбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ„бҹҮ бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бһ„бҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ‘бҹҒбҹ”

бһҮбһ¶бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ IoT бһҳбҹүбһјбһҠбҹӮбһӣ serverless бһӮбһәбһӣбҹ’бһўбһӮбһҪбһҡбһұбҹ’бһҷбһ”бҹ’бһҡбһҫбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһ…бҹҶбһ–бҹ„бҹҮбһҹбһ¶бһҡбһҠбҹӮбһӣбһ•бҹ’бһүбһҫбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ IoT бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһӣбһҫбһ–бһ–бһҖбҹ” бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһ¶бһҡвҖӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһүбһҫ бһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹӮбһ“бҹ…бһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

вң… бһ…бһ„бҹӢбһҡбһҖбһҳбһҫбһӣбһҖбһјбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһҖбһјбһҠ server бһҠбҹӮбһӣбһҹбҹ’бһҸбһ¶бһ”бҹӢбһҹбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ MQTTбҹ” бһҸбһҫбһңбһ¶бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһ–бһ–бһҖбһҮбһ¶бһҳбһҪбһҷ serverless бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…? бһҸбһҫбһўбҹ’бһ“бһҖбһӮбһ·бһҸбһҗбһ¶бһҖбһјбһҠбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Serverless?

> рҹ’Ғ бһҳбҹүбһјбһҠбҹӮбһӣ serverless бһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ‘бҹ…бһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ–бһ–бһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ•бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ serverless бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ“бҹ…бһӣбһҫбһ–бһ–бһҖбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғ serverless бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ„бҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҸбһ¶бһҳбһҹбҹҶбһҺбһҫбһҠбҹӮбһӣбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбһјбһ…бһҮбһ¶ бһҹбҹҶбһҺбһҪбһҡ бһ¬бһҖбҹҸбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ бһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбҹғбһӮбһ·бһҸбһҮбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҮбһҳбҹ’бһҡбһҫбһҹбһҸбҹӮбһҳбһҪбһҷбһ“бҹғбһҮбһҪбһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҸбһ¶бһҳбһӮбһјбһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҫбһҳбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһҸбһ·бһ…бһҮбһ¶бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ ЧһЧ•ЧЁЧӣЧ‘ЧӘ бһҠбҹӮбһӣбһ•бҹ’бһҹбҹҶбһҸбһ¶бһҡбһ¶бһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһ”бһ„бҹ’бһҡбһҪбһҳбһҮбһҪбһҡбһҷбҹүбһ¶бһ„бһҹбҹ’бһ‘бһҫбһҡбһӣбһ¶бһ“бҹ”

вң… бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҳбһҪбһҷбһҳбһ»бһҒбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһ“бһ·бһ„бһўбһ¶бһ“бһўбһҸбһ·бһҗбһ·бһҮбһ“бһўбҹҶбһ–бһё Azure Functions бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢ Microsoft Azure Functions](https://docs.microsoft.com/azure/azure-functions/functions-overview?WT.mc_id=academic-17441-jabenn)бҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҡбһҹбҹҒбһҡ Azure Functions бһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Azure Functions бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбҹ” Azure Functions бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶ Python, JavaScript, TypeScript, C#, F#, Java, бһ“бһ·бһ„ Powershellбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһҡбһ”бҹҖбһ”бһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Azure Functions бһҮбһ¶бһ—бһ¶бһҹбһ¶ Pythonбҹ”

> рҹ’Ғ Azure Functions бһҖбҹҸбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹҶбһҺбһҸбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ HTTP бһҡбһҪбһҳбһҳбһ¶бһ“бһ—бһ¶бһҹбһ¶бһ…бһ¶бһҹбҹӢбҹ—бһҠбһјбһ…бһҮбһ¶ COBOLбҹ”

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡ *trigger* бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҳбһҪбһҷ - бһҳбһ»бһҒбһ„бһ¶бһҡбһҸбһ”бһҸбһ‘бҹ…бһ“бһ№бһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“бһ…бҹ’бһҡбһҫбһ“ trigger бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Function бһҳбһҪбһҷ бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢ IoT Hub бһ бһҫбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһ“бһ·бһ„бһҹбҹ’бһҸбһ¶бһ”бҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбҹ”

### бһҖбһ¶бһҡбһ„бһ¶бһҡ - бһҠбҹҶбһЎбһҫбһ„бһ§бһ”бһҖбһҡбһҺбҹҚ Azure Functions

> бһ“бҹ…бһ–бҹҒбһӣбһ”бһ·бһ‘бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡ бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбһјбһҠ Azure Functions бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбһҫ Apple Silicon бһҮбһ¶бһҳбһҪбһҷбһӮбһҳбҹ’бһҡбҹ„бһ„ Python бһ‘бҹҒбҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ Mac бһ…бһ¶бһҹбҹӢбһҸбҹӮ Intel, бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһңбһёбһ“бһҠбһјбһҡ бһ¬бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһЎбһ·бһ“бһ»бһҖбһҹбҹҚбҹ”

бһҳбһ»бһҒбһ„бһ¶бһҡбһӣбҹ’бһўбһҳбһҪбһҷбһ“бҹғ Azure Functions бһӮбһә бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһ”бһ¶бһ“бҹ” Runtime бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹ…бһ–бһ–бһҖбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҸбһ”бһҸбҹ’бһҡбһјбһңбһҹбһ¶бһҡбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ… debug бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҒбһҺбҹҲбһ–бҹҒбһӣбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ•бһ„бһҠбҹӮбһҡбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ–бҹ’бһҡбһҳбһ…бһ·бһҸбҹ’бһҸбһ“бһ№бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһңбһ¶бһўбһ¶бһ…бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһ–бһ–бһҖбһ”бһ¶бһ“бҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚ Azure Functions бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҮбһ¶бһ„бһҮбһ¶ CLI бһҠбҹӮбһӣбһӮбҹҒбһ бҹ…бһҗбһ¶ Azure Functions Core Toolsбҹ”

1. бһҠбҹҶбһЎбһҫбһ„ Azure Functions core tools бһҠбҹ„бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһҜбһҖбһҹбһ¶бһҡбһўбҹҶбһ–бһё Azure Functions Core Tools](https://docs.microsoft.com/azure/azure-functions/functions-run-local?WT.mc_id=academic-17441-jabenn)

бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Azure Functions бһ‘бҹ…бһ–бһ–бһҖ бһңбһ¶бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҸбһ·бһ…бһҸбһҪбһ…бһ“бҹғбһ•бҹ’бһ‘бһ»бһҖбһ–бһ–бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбҹ’бһҹбһ¶бһҡбһҝбһ„бһҠбһјбһ…бһҮбһ¶ бһҜбһҖбһҹбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһ бҹ…бһ бҹ…бҹ” бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ•бҹ’бһ‘бһ»бһҖбһ–бһ–бһҖбһ•бһ„бһҠбҹӮбһҡ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ•бҹ’бһ‘бһ»бһҖбһ–бһ–бһҖбһ–бһ·бһҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһӣбҹӮбһ„бһ”бһ“бҹ’бһӣбҹҶбһ•бҹ’бһ‘бһ»бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһҗбһ¶ [Azurite](https://github.com/Azure/Azurite)бҹ” бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ…бһ•бҹ’бһ‘бһ»бһҖбһ–бһ–бһҖбҹ”

> рҹҺ“ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure бһ•бҹ’бһ‘бһ»бһҖбһҠбҹӮбһӣ Azure Functions бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбһәбһҮбһ¶бһӮбһҺбһ“бһё Azure Storageбҹ” бһӮбһҺбһ“бһёбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡ бһҜбһҖбһҹбһ¶бһҡбһҹбһ¶бһҡ data бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„ бһ¬ data бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһӮбһҺбһ“бһёбһ“бҹҒбҹҮбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһҡбһңбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбҹ”

1. Azurite бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Node.js бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҠбҹҶбһЎбһҫбһ„ Node.jsбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһҳбһҫбһӣбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһ“бҹ…бһӣбһҫ [бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ Node.js](https://nodejs.org/)бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Mac бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„бһ–бһё [Homebrew](https://formulae.brew.sh/formula/node) бһ•бһ„бһҠбҹӮбһҡбҹ”

1. бһҠбҹҶбһЎбһҫбһ„ Azurite бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ (`npm` бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбҹҶбһЎбһҫбһ„ Node.js):

    ```sh
    npm install -g azurite
    ```

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҜбһҖбһҹбһ¶бһҡбһҲбҹ’бһҳбҹ„бҹҮ `azurite` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Azurite бһ”бҹ’бһҡбһҫбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ:

    ```sh
    mkdir azurite
    ```

1. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ Azurite бһ бһҫбһҷбһ•бҹ’бһҠбһӣбҹӢбһҗбһҸбһ“бҹҒбҹҮбһҮбһ¶бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢ:

    ```sh
    azurite --location azurite
    ```

    бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһӣбҹӮбһ„бһ”бһ“бҹ’бһӣбҹҶбһ•бҹ’бһ‘бһ»бһҖ Azurite бһ“бһ№бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһ“бһ·бһ„бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ runtime Functions бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”

    ```output
    вһң  ~ azurite --location azurite

    Azurite Queue service is successfully listening at http://127.0.0.1:10001
    Azurite Table service is starting at http://127.0.0.1:10002
    Azurite Table service is successfully listening at http://127.0.0.1:10002
    ```

### бһҖбһ¶бһҡбһ„бһ¶бһҡ - бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„ Azure Functions

CLI бһҡбһ”бһҹбҹӢ Azure Functions бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһҗбҹ’бһҳбһёбҹ”

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ“бҹ„бҹҮ бһ бҹ…бһҗбһ¶ `soil-moisture-trigger`

    ```sh
    mkdir soil-moisture-trigger
    cd soil-moisture-trigger
    ```

1. бһ”бһ„бҹ’бһҖбһҫбһҸ virtual environment Python бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ“бҹҒбҹҮ:

    ```sh
    python3 -m venv .venv
    ```

1. бһ”бһҫбһҖбһҖбҹҶбһҺбһҸбҹӢ virtual environment:

    * бһ“бҹ…бһӣбһҫ Windows:
        * бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Command Prompt бһ¬ Command Prompt бһҸбһ¶бһҳбһҡбһҷбҹҲ Windows Terminal бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–

            ```cmd
            .venv\Scripts\activate.bat
            ```

        * бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ PowerShell бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–

            ```powershell
            .\.venv\Scripts\Activate.ps1
            ```

    * бһ“бҹ…бһӣбһҫ macOS бһ¬ Linux бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–

        ```cmd
        source ./.venv/bin/activate
        ```

    > рҹ’Ғ бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһёбһ‘бһёбһҸбһ¶бҹҶбһ„бһҠбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бһ„бҹ’бһҖбһҫбһҸ virtual environment бҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ `.venv` бһ‘бҹҒ бһӮбһҪбһҡбһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ activate бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҜбһҖбһҹбһ¶бһҡбһ–бһёбһҗбһҸбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ¶бҹ”

1. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ“бҹҒбҹҮбҹ–

    ```sh
    func init --worker-runtime python soil-moisture-trigger
    ```

    бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ 3 бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ–

    * `host.json` - бһҜбһҖбһҹбһ¶бһҡбһ•бҹ’бһ‘бҹғбһҸбҹҶбһҺбһҡ JSON бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһЎбһҫбһҷбҹ”

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһӮбһәбһҮбһ¶бһҹбҹӮбһӣбһҡбһ”бһҹбҹӢбһҖбһјбһҠ serverless бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ IoT hub бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһ•бҹ’бһҹбһ¶бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ IoT Hub бһ‘бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ“бһ№бһ„бһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһҹбһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ‘бҹ… IoT Hub бһ бһҫбһҷбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбһӣбҹӢбһңбһ¶бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһҹбһ¶бһҡбһ“бҹҒбҹҮ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡвҖӢбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… IoT Hubs *event hub compatible endpoint*бҹ”

IoT Hub бһӮбһәбһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһҪбһҷбһ‘бҹҖбһҸбһӣбһҫбһҹбҹҒбһңбһ¶ Azure бһҠбҹӮбһӣбһ”бһ¶бһ“бһ бҹ…бһҗбһ¶ Azure Event Hubsбҹ” Event Hubs бһӮбһәбһҮбһ¶вҖӢбһҹбҹҒбһңбһ¶вҖӢбһҠбҹӮбһӣвҖӢбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸвҖӢбһұбҹ’бһҷвҖӢбһўбҹ’бһ“бһҖвҖӢбһ•бҹ’бһүбһҫвҖӢбһ“бһ·бһ„вҖӢбһ‘бһ‘бһҪбһӣвҖӢбһҹбһ¶бһҡ бһ”бҹ’бһҡбһҫ IoT Hub бһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһҫбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбһӣбҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ IoTбҹ” бһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“бһҹбһ¶бһҡбһ–бһё IoT Hub бһӮбһәбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҠбһјбһ…бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Event Hubsбҹ”

вң… бһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ“бҹ’бһҸбһ·бһ…бҹ– бһўбһ¶бһ“бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹғ Event Hubs бһҖбҹ’бһ“бһ»бһ„ [Azure Event Hubs documentation](https://docs.microsoft.com/azure/event-hubs/event-hubs-about?WT.mc_id=academic-17441-jabenn) бҹ” бһҸбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ’бҹҖбһ”бһ“бһ№бһ„ IoT Hub бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… IoT Hub бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҖбһјбһ“бһҹбҹ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһ’бһ¶бһ“бһ¶бһ”бһ¶бһ“бһҗбһ¶бһҳбһ¶бһ“бһҸбҹӮбһҳбҹ’бһҸбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһ¶бһ“бҹ” бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮ бһҖбҹҸбһ–бҹҒбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“бһҹбһ¶бһҡбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ…бҹҶбһ“бһҪбһ“бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһјбһ“бһҹбҹ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ¶ бһҮбһ¶бһҳбһҪбһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹғ IoT Hub бһ•бһ„бһҠбҹӮбһҡбҹ”

> рҹ’Ғ бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ· **iothubowner** бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·бһ–бҹҒбһүбһӣбҹҒбһүбһ‘бҹ…бһҖбһјбһҠбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһңбһ¶бһӣбһҫ IoT Hubбҹ” бһҖбҹ’бһ“бһ»бһ„ ideally бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ·бһ‘бҹ’бһ’бһ·бһ‘бһ¶бһ”бһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ” бһ“бҹҒбҹҮбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

бһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҡбһҪбһ… бһҖбһјбһҠбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ‘бҹ… IoT Hub бһҳбһ·бһ“бһӮбһ·бһҸбһҗбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһңбһ¶бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһҹбһ¶бһҡбһҮбһ¶бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ‘бһ¶бһӣбҹӢбһҸбҹӮбҹ”

### бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳ - бһ‘бһ‘бһҪбһӣбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢ event hub compatible endpoint

1. бһ–бһё VS Code terminal бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ IoT Hubs Event Hub compatible endpointбҹ–

    ```sh
    az iot hub connection-string show --default-eventhub \
                                      --output table \
                                      --hub-name <hub_name>
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<hub_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ IoT Hub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code бһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бһ”бһ“бҹ’бһҗбҹӮбһҳбһҸбһҳбҹ’бһӣбҹғбһ”бһ“бҹ’бһҗбҹӮбһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖ `Values`бҹ–

    ```json
    "IOT_HUB_CONNECTION_STRING": "<connection string>"
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<connection string>` бһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҮбҹҶбһ бһ¶бһ“бһҳбһ»бһ“бҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹ’бһ”бҹҖбһҹбһҖбҹ’бһҡбҹ„бһҷбһҮбһҪбһҡбһҒбһ¶бһ„бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһҮбһ¶ JSON бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

### бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳ - бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһ•бҹ’бһҹбһ¶бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһ•бҹ’бһҹбһ¶бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбҹ”

1. бһ–бһё VS Code terminal бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҗбһҸ `soil-moisture-trigger`бҹ–

* `function.json` - бһ“бҹҒбҹҮбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбҹ” бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӮбһәбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖ `bindings`бҹ” бһңбһ¶бһҮбһ¶бһҹбһ¶бһҖбһҹбһҳбһ“бһ№бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҡбһңбһ¶бһ„ Azure Functions бһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ— Azureбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһҳбһ¶бһ“ input binding бһ‘бҹ… event hub - бһңбһ¶бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… event hub бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

    > рҹ’Ғ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“ output bindings бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҖбһ¶бһҡбһ…бҹҒбһүбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһүбһҫбһ‘бҹ…бһҹбҹҒбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳ output binding бһ‘бҹ…бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ бһҫбһҷбһ”бһүбҹ’бһҮбһјбһ“бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ IoT Hub бһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡ бһ бһҫбһҷбһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ‘бҹ…бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

  * `"type": "eventHubTrigger"` - бһ“бҹҒбҹҮбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҗбһ¶бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбһ”бһ“бһ№бһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ–бһё Event Hub

  * `"connection": ""` - бһ“бҹҒбҹҮбһҖбҹҶбһҺбһҸбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһ“бҹғбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ” бһ–бҹҒбһӣбһҡбһҸбҹӢбһӣбһҫбһҖбҹ’бһҡбҹ„бһҷбһңбһ·бһүбһңбһ¶бһ“бһ№бһ„бһўбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ“бҹҒбҹҮбһ–бһёбһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бҹ”

    > рҹ’Ғ бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһ·бһ“бһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ“бҹ…бһҜбһҖбһҹбһ¶бһҡ `function.json` бһ”бһ¶бһ“бһ‘бҹҒ бһңбһ¶бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһўбһ¶бһ“бһ–бһёбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбҹ” бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһҡбһҺбһёбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бҹҒбһүбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбҹ”

1. бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮ [бһҖбҹ„бһҹбһ·бһҖбһ¶бһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„ template Azure Functions](https://github.com/Azure/azure-functions-templates/issues/1250) бһҳбһ¶бһ“бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `cardinality`бҹ” бһҹбһјбһҳбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `function.json` бһ–бһё `many` бһ‘бҹ… `one`бҹ–

бһҳбһҖбһҠбһӣбҹӢбһ–бҹҒбһӣбһ“бҹҒбҹҮ Functions app бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҹбҹ’бһҠбһ¶бһ”бҹӢбһҹбһ¶бһҡбһ–бһё IoT Hub бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Event Hub compatible end pointбҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһүбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ IoTбҹ” бһңбһ¶бһӮбһәбһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҒбһ»бһҹбһ‘бҹ… IoT Hub бһҸбһ¶бһҳбһҡбһҷбҹҲ *Registry Manager*бҹ” Registry Manager бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҳбһҫбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҮбһ¶бһҳбһҪбһҷ IoT Hub бһ бһҫбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһ•бҹ’бһүбһҫбһҹбһ¶бһҡ cloud to device, бһҹбҹҶбһҺбһҫбһ—бһ¶бһ– direct method бһ¬бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– device twinбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһё бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– бһ¬бһӣбһ»бһ”бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһ–бһё IoT Hub бһ•бһ„бһҠбҹӮбһҡбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… Registry Manager бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҪбһҷбҹ”

### бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳ - бһ‘бһ‘бһҪбһӣбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢ Registry Manager

1. бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ–

    ```sh
    az iot hub connection-string show --policy-name service \
                                      --output table \
                                      --hub-name <hub_name>
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<hub_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ IoT Hub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

    бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ *ServiceConnect* бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ `--policy-name service`бҹ” бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһёбһҹбһ·бһ‘бҹ’бһ’бһ·бһҠбҹӮбһӣбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ“бҹ„бҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ ServiceConnect бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ“бһ·бһ„бһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бҹ”

    вң… бһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ“бҹ’бһҸбһ·бһ…бҹ– бһўбһ¶бһ“бһўбҹҶбһ–бһёбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбҹ’бһ“бһ»бһ„ [IoT Hub permissions documentation](https://docs.microsoft.com/azure/iot-hub/iot-hub-devguide-security#iot-hub-permissions?WT.mc_id=academic-17441-jabenn)

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code бһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бһ”бһ“бҹ’бһҗбҹӮбһҳбһҸбһҳбҹ’бһӣбҹғбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖ `Values`бҹ–

    ```json
    "REGISTRY_MANAGER_CONNECTION_STRING": "<connection string>"
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<connection string>` бһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҮбҹҶбһ бһ¶бһ“бһҳбһ»бһ“бҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹ’бһ”бҹҖбһҹбһҖбҹ’бһҡбҹ„бһҷбһҮбһҪбһҡбһҒбһ¶бһ„бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһҮбһ¶ JSON бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

### бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳ - бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһ—бһ¶бһ–direct method бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ

1. SDK бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Registry Manager бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҳбһ”бҹүбһ¶бһҖбҹҒбһҮбҹҚ Pipбҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бҹ…бһҜбһҖбһҹбһ¶бһҡ `requirements.txt` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹүбһ¶бһҖбҹҒбһҮбҹҚбһ“бҹҒбҹҮбҹ–

    ```sh
    azure-iot-hub
    ```

1. бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ VS Code terminal бһҳбһ¶бһ“ virtual environment бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бһҫбһҷ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһЎбһҫбһ„бһ”бҹүбһ¶бһҖбҹҒбһҮбҹҚ Pipбҹ–

    ```sh
    pip install -r requirements.txt
    ```

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бҹ…бһҜбһҖбһҹбһ¶бһҡ `__init__.py`бҹ–

    ```python
    import json
    import os
    from azure.iot.hub import IoTHubRegistryManager
    from azure.iot.hub.models import CloudToDeviceMethod
    ```

    бһ“бҹҒбҹҮбһ“бһ¶бҹҶбһ…бһјбһӣбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“вҖӢ бһ”бһјбһҖбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ Registry Manager бһ“бһ·бһ„бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһ—бһ¶бһ–direct methodбҹ”

1. бһҠбһҖбһҖбһјбһҠбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `main` бһ…бҹҒбһү бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһ‘бһ»бһҖбҹ”

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `main` бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```python
    body = json.loads(event.get_body().decode('utf-8'))
    device_id = event.iothub_metadata['connection-device-id']

    logging.info(f'Received message: {body} from {device_id}')
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһҷбһҖбһҹбһ¶бһ…бҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ¶бһҡбһҮбһ¶ JSON бһ•бҹ’бһүбһҫбһҠбҹ„бһҷбһ§бһ”бһҖбһҡбһҺбҹҚ IoTбҹ”

    бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ‘бһ‘бһҪбһӣ device ID бһ–бһё annotations бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһҮбһ¶бһҳбһҪбһҷбһҹбһ¶бһҡбҹ” бһҹбһ¶бһ…бҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҳбһ¶бһ“бһҹбһ¶бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһҮбһ¶бһ‘бғ”бғҡбғ”бғ‘бғҳ `iothub_metadata` бһңбһҸбҹ’бһҗбһ»бһҜбһҖбһҹбһ¶бһҡбһўбҹ’бһ“бһҖбһ•бҹ’бһүбһҫбһҡбһҹбһ¶бһҡ IoT Hub бһ–бһё device ID бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһүбһҫ бһ“бһ·бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһҹбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбҹ”

    бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ terminal бһ–бҹҒбһӣбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ Functions app бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бҹ”

1. бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮ бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```python
    soil_moisture = body['soil_moisture']

    if soil_moisture > 450:
        direct_method = CloudToDeviceMethod(method_name='relay_on', payload='{}')
    else:
        direct_method = CloudToDeviceMethod(method_name='relay_off', payload='{}')
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣбһ”бҹ’бһҡбҹҒбһ„бһҹбһҫбһҳбһҠбһёбһ–бһёбһҹбһ¶бһҡбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһңбһ¶бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹҶбһ“бһҪбһ“бһ”бҹ’бһҡбҹҒбһ„бһҹбһҫбһҳбһҠбһё бһ бһҫбһҷбһӮбһ·бһҸбһӣбһҫбһҸбһҳбҹ’бһӣбҹғ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһ‘бҹҮбһҮбҹҶбһ“бһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҫбһ—бһ¶бһ–direct method бһҡбһ”бһҹбҹӢ `relay_on` бһ¬ `relay_off` бһ‘бҹ… direct methodбҹ” бһҹбҹҶбһҺбһҫбһ—бһ¶бһ–direct method бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ» payload бһ‘бҹҒ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ”бһүбҹ’бһҮбһјбһ“бһҜбһҖбһҹбһ¶бһҡ JSON бһ‘бһ‘бҹҒбҹ”

1. бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```python
    logging.info(f'Sending direct method request for {direct_method.method_name} for device {device_id}')

    registry_manager_connection_string = os.environ['REGISTRY_MANAGER_CONNECTION_STRING']
    registry_manager = IoTHubRegistryManager(registry_manager_connection_string)
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ•бҹ’бһ‘бһ»бһҖ `REGISTRY_MANAGER_CONNECTION_STRING` бһ–бһёбһҜбһҖбһҹбһ¶бһҡ `local.settings.json`бҹ” бһҸбһҳбҹ’бһӣбҹғбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҮбһ¶ environment variables бһ бһҫбһҷбһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `os.environ` бһҠбҹӮбһӣбһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһ”бһ„бҹ’бһңбһ·бһӣбһҸбҹ’бһҡбһЎбһ”бҹӢбһңбһ¶бһӣбһҫбһҳбҹҒбһӮбһ»бһҺбһ“бҹғбһЎбһ¶бһ“бһ–бҹҗбһ‘бҹҗбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

    > рҹ’Ғ бһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһјбһҠбһ“бҹҒбҹҮбһҠбһ¶бһҖбҹӢбһӣбһҫ cloud бһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„ `local.settings.json` бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶ *Application Settings* бһ бһҫбһҷбһўбһ¶бһ…бһўбһ¶бһ“бһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бҹ”

    бһҖбһјбһҠбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһҸбҹ’бһҗбһ» Registry Manager helper бһҸбһ¶бһҳбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”

1. бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ```python
    registry_manager.invoke_device_method(device_id, direct_method)

    logging.info('Direct method request sent!')
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бҹ’бһҡбһ¶бһ”бҹӢ registry manager бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһ—бһ¶бһ–direct method бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһ‘бҹҒбһӣбһ·бһҳбҹүбҹҒбһҸбҹ’бһҡбһ“бҹҒбҹҮбҹ”
> рҹ’Ғ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ MQTT бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ”бһүбҹ’бһҮбһ¶бһҡбҹҒбһӣбһёбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹӮбһҳбһҪбһҷбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ (method request) бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹӮбһҳбһҪбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһәбһ“бһҹбҹҗбһҡбһҮбһ¶бһҸбһ·бһ бһјбһҳбҹүбҹ’бһҹбҹҗбһҡ (moisture sensors) бһ“бһ·бһ„бһҡбҹҒбһӣбһё бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

1. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Functions бһ бһҫбһҷбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҹбһ¶бһҡбҹ—бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бһ·бһ„бһҹбҹҶбһҺбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбҹ” бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҹбһәбһ“бһҹбҹҗбһҡбһҮбһ¶бһҸбһ·бһ бһјбһҳбҹүбҹ’бһҹбҹҗбһҡбһ…бһјбһӣбһ“бһ·бһ„бһ…бҹҒбһүбһ–бһёбһҠбһёбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҸбһҳбҹ’бһӣбҹғбһ”бһ„бҹ’бһңбһ·бһӣбһ”бҹ’бһҠбһјбһҡбһ“бһ·бһ„бһ–бһ“бҹ’бһӣбһҝбһ“бһҡбҹҒбһӣбһёбһ”бһҫбһҖбһ”бһ·бһ‘бҹ”

> рҹ’Ғ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһҖбһјбһҠбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ [code/functions](../../../../../2-farm/lessons/5-migrate-application-to-the-cloud/code/functions)бҹ”

## бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҖбһјбһҠ serverless бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҳбҹҒбһғ

### бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ - бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“бһҳбҹҒбһғ

    ```sh
                              --name <storage_name> 
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<storage_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Storage Account бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҲбҹ’бһҳбҹ„бҹҮбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹӮбһҳбһҪбһҷбһӣбһҫбһҹбһҖбһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ“бҹғ URL бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ…бһјбһӣбһҳбһҫбһӣ Storage Accountбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ“бһ¶бһҖбһҡбһҸбһјбһ… бһ“бһ·бһ„бһӣбҹҒбһҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҲбҹ’бһҳбҹ„бҹҮбһ“бҹҒбҹҮ бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҸбһҪбһўбһҖбҹ’бһҹбһҡ бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһ“бһ·бһ„бһңбһ¶бһҳбһ¶бһ“бһҖбҹҶбһҡбһ·бһҸ 24 бһҸбһҪбһўбһҖбҹ’бһҹбһҡбҹ” бһ”бҹ’бһҡбһҫбһўбҹ’бһңбһёбһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶ `sms` бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҠбҹҸбһҸбҹӮбһҳбһҪбһҷбһ“бҹ…бһ…бһ»бһ„ бһҠбһјбһ…бһҮбһ¶бһ–бһ¶бһҖбҹ’бһҷбһ…бҹғбһҠбһ“бҹ’бһҷ бһ¬бһҲбҹ’бһҳбҹ„бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

    бһҮбһҳбҹ’бһҡбһҫбһҹ `--sku Standard_LRS` бһҮбҹ’бһҡбһҫбһҹбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһҸбһҳбҹ’бһӣбҹғ бһҠбҹ„бһҷбһҮбҹ’бһҡбһҫбһҹбһӮбһҺбһ“бһёбһ‘бһјбһ‘бҹ…бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бһ”бһ”бҹҶбһ•бһ»бһҸбҹ” бһӮбҹ’бһҳбһ¶бһ“бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҹбҹҒбһңбһ¶бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Storage бһ‘бҹҒ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҸбһҳбҹ’бһӣбҹғбһҹбҹҒбһңбһ¶бһ’бҹҖбһ”бһ”бһ¶бһ“бһҗбһ¶бһҮбһ¶бһ‘бһ¶бһ” бһҠбҹ„бһҷ Storage бһҠбҹӮбһӣбһҗбҹ’бһӣбҹғбһ”бҹҶбһ•бһ»бһҸбһҳбһ¶бһ“бһҗбҹ’бһӣбҹғбһҸбһ·бһ…бһҮбһ¶бһ„ US$0.05 бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҒбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ¶бһӣбҹӢбһ…бҹҶбһҺбһ»бҹҮбһӮбһёбһЎбһјбһ”бҹғбһҸбҹӢбҹ”

    вң… бһўбһ¶бһ“бһҹбҹ’бһҸбһёбһ–бһёбһҸбһҳбҹ’бһӣбҹғбһ“бҹ…бһӣбһҫ [бһ‘бҹҶбһ–бҹҗбһҡбһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳ Azure Storage Account](https://azure.microsoft.com/pricing/details/storage/?WT.mc_id=academic-17441-jabenn)

1. бһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ Function App:

    ```sh
    az functionapp create --resource-group soil-moisture-sensor \
                          --runtime python \
                          --functions-version 3 \
                          --os-type Linux \
                          --consumption-plan-location <location> \
                          --storage-account <storage_name> \
                          --name <functions_app_name>
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<location>` бһҮбһ¶бһҳбһҪбһҷбһ‘бһёбһҸбһ¶бҹҶбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ Resource Group бһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ”

    бһҮбҹҶбһ“бһҪбһҹ `<storage_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮ Storage Account бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһҳбһ»бһ“бҹ”

    бһҮбҹҶбһ“бһҪбһҹ `<functions_app_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Functions App бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҲбҹ’бһҳбҹ„бҹҮбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢбһӣбһҫбһҹбһҖбһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғ URL бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ…бһјбһӣ Functions Appбҹ” бһ”бҹ’бһҡбһҫбһўбҹ’бһңбһёбһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶ `soil-moisture-sensor-` бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҸбҹӮбһҳбһҪбһҷбһ“бҹ…бһ…бһ»бһ„ бһҠбһјбһ…бһҮбһ¶бһ–бһ¶бһҖбҹ’бһҷбһ…бҹғбһҠбһ“бҹ’бһҷ бһ¬бһҲбҹ’бһҳбҹ„бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

    бһҮбһҳбҹ’бһҡбһҫбһҹ `--functions-version 3` бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбҹӮ Azure Functions бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбҹ” бһҖбҹҶбһҺбҹӮ 3 бһҮбһ¶бһҖбҹҶбһҺбҹӮбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ”

    бһҮбһҳбҹ’бһҡбһҫбһҹ `--os-type Linux` бһ”бҹ’бһҡбһ¶бһ”бҹӢRuntime Functions бһұбҹ’бһҷбһ”бҹ’бһҡбһҫ Linux бһҮбһ¶ OS бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ” Functions бһўбһ¶бһ…бһҡбҹҖбһ”бһ…бҹҶбһӣбһҫ Linux бһ¬ Windows бһ•бҹ’бһ‘бҹҒбһҒбһ»бҹҶбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Python бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ“бһ¶бҹҶбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһӣбһҫ Linux бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

### бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ - бһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ Functions App бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹҶбһҺбһҸбҹӢбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҒбҹ’бһҹбҹӮбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… IoT Hub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһ‘бҹ… Application Settings бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Functions App бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ… Azure бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

> рҹҺ“ бһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһӣokalбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһ бһҫбһҷбһҳбһ·бһ“бһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбҹ’бһҡбһӮбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһјбһҠ бһҠбһјбһ…бһҮбһ¶ GitHubбҹ” бһ–бҹҒбһӣбһ”бһ¶бһүбҹӢбһ‘бҹ…бһҳбҹҒбһғ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё (Application Settings) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбҹ” бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбһәбһҮбһ¶бһӮбҹҶбһ–бһёбһҸ/бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһғ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһ–бһёбһ бҹҒбһҸбһ»бһҖбһ¶бһҡбһҺбҹҚбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ бһ¬бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠ бһ¬бһҳбҹҒбһҡбҹҖбһ“бһ–бҹҒбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһјбһҠбһ‘бҹ… IoT Hubбҹ”

1. бһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢ `IOT_HUB_CONNECTION_STRING` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Function App Application Settingsбҹ–

    ```sh
    az functionapp config appsettings set --resource-group soil-moisture-sensor \
                                          --name <functions_app_name> \
                                          --settings "IOT_HUB_CONNECTION_STRING=<connection string>"
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<functions_app_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Functions App бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

    бһҮбҹҶбһ“бһҪбһҹ `<connection string>` бһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғ `IOT_HUB_CONNECTION_STRING` бһ–бһёбһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

1. бһ’бҹ’бһңбһҫбһҠбһјбһ…бһҮбҹҶбһ бһ¶бһ“бһҒбһ¶бһ„бһӣбһҫбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһҠбҹ„бһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғ `REGISTRY_MANAGER_CONNECTION_STRING` бһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһ–бһёбһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһүбҹ’бһҮбһё Application Settings бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ function appбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бһ‘бҹ…бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһҸбһҳбҹ’бһӣбҹғбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

> рҹ’Ғ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `AzureWebJobsStorage` бһҡбһҪбһ…бһҳбһҖбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `local.settings.json` бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹ…бһҸбһҳбҹ’бһӣбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҳбҹҒбһҡбҹҖбһ“storage emulatorбҹ” бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ Functions App бһўбҹ’бһ“бһҖбһ•бҹ’бһүбһҫ Storage Account бһҮбһ¶бһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһ“бһ·бһ„бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ“бҹҒбҹҮбҹ”

### бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ - бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ– Functions App бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҳбҹҒбһғ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ Functions App бһ”бҹ’бһҡбһ¶бһҖбһҠбһ—бһ¶бһ–бһҡбһҪбһ… бһҡбһ¶бһӣбҹӢ бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ‘бҹ…бһҳбҹҒбһғбһ”бһ¶бһ“бҹ”

1. бһҡбһҸбҹӢбһ”бһүбҹ’бһҮбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһ–бһёСӮРөСҖбһҳбһ·бһ“ VS Code бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ Functions App бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ:

    ```sh
    func azure functionapp publish <functions_app_name>
    ```

    бһҮбҹҶбһ“бһҪбһҹ `<functions_app_name>` бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Functions App бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҖбһјбһҠбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ бһҫбһҷбһ•бҹ’бһүбһҫбһ‘бҹ… Functions App бһҠбҹӮбһӣбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ– бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ” бһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Solo бһ бһҫбһҷбһ”бһүбҹ’бһ…бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһёбһҖбһ¶бһҡбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ– бһ“бһ·бһ„бһ”бһүбҹ’бһҮбһёбһҡбһ”бһҹбҹӢ function бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһ”бһүбҹ’бһҮбһёбһ“бһ№бһ„бһҳбһ¶бһ“бһҸбҹӮ trigger бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”
