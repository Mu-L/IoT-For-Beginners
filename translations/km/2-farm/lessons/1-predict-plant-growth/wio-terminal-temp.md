# бһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– - Wio Terminal

бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһ‘бҹ…бһҖбһ¶бһ“бҹӢ Wio Terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһ–бһёбһңбһ¶бҹ”

## бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ№бһ„

Wio Terminal бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚбһҗбһ¶бһҳбһ–бһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһӮбһәбһҮбһ¶ [бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбҹҶбһҺбһҫбһҳ бһ“бһ·бһ„бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– DHT11](https://www.seeedstudio.com/Grove-Temperature-Humidity-Sensor-DHT11.html) бһ•бҹ’бһҸбһӣбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһёбһҡбһҡбһҪбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҮбһ“бһҳбһҪбһҷбҹ” бһңбһ¶бһҳбһ¶бһ“бһ”бҹ’бһҡбһҮбһ¶бһ”бҹ’бһҡбһ·бһҷбһ—бһ¶бһ–бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҠбҹ„бһҷбһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹӢбһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– бһҹбҹҶбһҺбһҫбһҳ бһ“бһ·бһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҹбҹҶбһ–бһ¶бһ’бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһӮбһәбһҮбһ¶бһ—бһ¶бһӮбһҡбһҷбһңбһ·бһҖбһ¶бһҡбһҺбҹҚбһўбһ¶бһҖбһ¶бһҹбһ’бһҳбҹ’бһҳбһҸбһ¶бһ“бҹғбһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· (NTC) thermistor бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһ’бһјбһҡбһҹбҹ’бһ”бһҫбһҷбһ“бҹ…бһ–бҹҒбһӣбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҖбһҫбһ“бһЎбһҫбһ„бҹ”

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҢбһёбһҮбһёбһҗбһӣ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ ADC бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһүбҹ’бһүбһ¶бһҢбһёбһҮбһёбһҗбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– бһ“бһ·бһ„бһҹбҹҶбһҺбһҫбһҳбһҠбҹӮбһӣбһҳбҹүбҹҠбһёбһҖбҹ’бһҡбһјбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһӣбһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“бҹ”

### бҖҒбҖ»бҖӯбҖҗбҖәбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–

бһ§бһ”бһҖбһҡбһҺбҹҚ Grove temperature sensor бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ–бһҸбҹҚбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҢбһёбһҮбһёбһҗбһӣбһҡбһ”бһҹбҹӢ Wio Terminalбҹ”

#### бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳ - бһҒбҹ’бһҹбҹӮбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–

бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бҹ”

![бһ§бһ”бһҖбһҡбһҺбҹҚ Grove temperature sensor](../../../../../translated_images/km/grove-dht11.07f8eafceee17004.webp)

1. бһҠбһ¶бһҖбҹӢбһ…бһ»бһ„бһ“бҹғбһҒбҹ’бһҹбҹӮ Grove бһҳбһҪбһҷбһ…бһ»бһ„бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһҹбһ·бһҸбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбҹҶбһҺбһҫбһҳ бһ“бһ·бһ„бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бҹ” бһңбһ¶бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһ‘бһ·бһҹбһҸбҹӮбһҳбҹ’бһҸбһ„бҹ”

1. бһ“бҹ…бһ–бҹҒбһӣ Wio Terminal бһҳбһ·бһ“бһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ¬бһ”бҹ’бһҡбһ—бһ–бһҗбһ¶бһҳбһ–бһӣбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹғбһҒбҹ’бһҹбҹӮ Grove бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһҹбһ·бһҸ Grove бһ–бһёбһ…бҹҶбһ бҹҖбһ„бһҹбҹ’бһҸбһ¶бҹҶбһҶбҹ’бһ„бһ¶бһҷбһ–бһёбһ”бҹҠбһјбһҸбһ»бһ„бһ”бҹҶбһ—бҹ’бһӣбһәбҹ”

## бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ Wio Terminal бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”

### бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбһ¶бһҹбҹӢбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бҹ”

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҗбҹ’бһҳбһё Wio Terminal бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ PlatformIOбҹ” бһ бҹ…бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбһҗбһ¶ `temperature-sensor`бҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбҹҠбҹҒбһҡбһёбҹ”

    > вҡ пёҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбҹ„бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢ [бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„ PlatformIO бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ‘бһё 1 бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 1 бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ](../../../1-getting-started/lessons/1-introduction-to-iot/wio-terminal.md#create-a-platformio-project)бҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Grove Humidity and Temperature бһҡбһ”бһҹбҹӢ Seeed бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `platformio.ini` бһҡбһ”бһҹбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бҹ–

    ```ini
    lib_deps =
        seeed-studio/Grove Temperature And Humidity Sensor @ 1.0.1
    ```

    > вҡ пёҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбҹ„бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢ [бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ‘бҹ…бһӮбһҳбҹ’бһҡбҹ„бһ„ PlatformIO бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ‘бһё 1 бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 4 бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ](../../../1-getting-started/lessons/4-connect-internet/wio-terminal-mqtt.md#install-the-wifi-and-mqtt-arduino-libraries)бҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳ бһҖбһјбһҠ `#include` бһҒбһ¶бһ„бһӣбһҫбһҜбһҖбһҹбһ¶бһҡ бһ–бһёбһҖбҹ’бһҡбҹ„бһҳ `#include <Arduino.h>` бһҡбһҪбһ…бҹ–

    ```cpp
    #include <DHT.h>
    #include <SPI.h>
    ```

    бһ“бҹҒбҹҮбһ“бһ¶бҹҶбһ…бһјбһӣбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбҹ” бһҜбһҖбһҹбһ¶бһҡ `DHT.h` бһҳбһ¶бһ“бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ бһҫбһҷбһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳ `SPI.h` бһҮбһҪбһҷбһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһјбһҠбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбҹ”

1. бһҳбһ»бһ“бһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһҹбһјбһҳбһ”бҹ’бһҡбһҖбһ¶бһҹбһ§бһ”бһҖбһҡбһҺбҹҚ DHTбҹ–

    ```cpp
    DHT dht(D0, DHT11);
    ```

    бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ¶бһҹбһңбһҸбҹ’бһҗбһ» DHT бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ§бһ”бһҖбһҡбһҺбҹҚ **D**igital **H**umidity бһ“бһ·бһ„ **T**emperatureбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ…бҹ’бһҡбһҖ `D0` бһҠбҹ„бһҷбһ…бҹҶбһ бҹҖбһ„бһҹбҹ’бһҠбһ¶бҹҶбһ“бҹғбһҮбҹҶбһ–бһјбһҖ Grove бһҡбһ”бһҹбҹӢ Wio Terminalбҹ” бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ‘бһёбһ–бһёбһҡбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ *DHT11* - бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ•бһ„бһҠбҹӮбһҡбҹ”

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбҹҠбҹҒбһҡбһёбҹ–

    ```cpp
    void setup()
    {
        Serial.begin(9600);
    
        while (!Serial)
            ; // бһҡбһ„бҹӢбһ…бһ¶бҹҶ Serial бһІбҹ’бһҷбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ
    
        delay(1000);
    }
    ```

1. бһ“бҹ…бһ…бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `setup` бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё `delay` бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ бҹ…бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ§бһ”бһҖбһҡбһҺбҹҚ DHTбҹ–

    ```cpp
    dht.begin();
    ```

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `loop` бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бҹҒбһүбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһ‘бҹ…бһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбҹ–

    ```cpp
    void loop()
    {
        float temp_hum_val[2] = {0};
        dht.readTempAndHumidity(temp_hum_val);
        Serial.print("Temperature: ");
        Serial.print(temp_hum_val[1]);
        Serial.println ("В°C");
    
        delay(10000);
    }
    ```

    бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҖбһ¶бһҹбһўбһ¶бһҡбһ¶бһҷбһ‘бһ‘бҹҒ 2 бһ‘бҹҶбһ“бҹҒбһҡ бһ“бһ·бһ„бһ•бҹ’бһүбһҫбһ‘бҹ…бһ бҹ… `readTempAndHumidity` бһ“бҹ…бһӣбһҫбһңбһҸбҹ’бһҗбһ» `DHT`бҹ” бһ бҹ…бһ“бҹҒбҹҮбһ”бҹҶбһ–бҹҒбһүбһўбһ¶бһҡбһ¶бһҷбһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғ 2 бһҖбҹҶбһ–бһ»бһ„ - бһҹбҹҶбһҺбһҫбһҳбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ“бҹ…бһҡбһ”бһҹбҹӢбһ’бһ¶бһҸбһ»бһҖбһ“бҹ’бһӣбһ„ 0 (бһ…бһ¶бҹҶбһҗбһ¶бһўбһ¶бһҡбһ¶бһҷ C++ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҹбһјбһ“бҹ’бһҷ бһҠбһјбһ…бһ“бҹҒбҹҮбһ’бһ¶бһҸбһ»бһҒбҹ’бһҹбҹӮ 0 бһӮбһәбһ’бһ¶бһҸбһ»бһҒбҹ’бһҹбҹӮ "бһҠбҹҶбһ”бһјбһ„") бһ бһҫбһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ’бһ¶бһҸбһ»бһҖбһ“бҹ’бһӣбһ„ 1бҹ”

    бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһ–бһёбһ’бһ¶бһҸбһ»бһҖбһ“бҹ’бһӣбһ„ 1 бһ“бҹғбһўбһ¶бһҡбһ¶бһҷ бһ бһҫбһҷбһ”бһүбҹ’бһ…бҹҒбһүбһ‘бҹ…бһ…бҹ’бһҡбһҖбһҹбҹҠбҹҒбһҡбһёбҹ”

    > рҹҮәрҹҮё бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһҮбһ¶бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбҹҒбҹ’бһӣбһҹбҹ’бһҷ бһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһҳбҹҒбһҡбһ·бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢ Fahrenheit бһ…бҹӮбһҖбһҸбһҳбҹ’бһӣбҹғ Celsius бһҠбҹ„бһҷ 5, бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһӮбһ»бһҺ 9, бһ”бһјбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ 32бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһўбһ¶бһ“бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– 20В°C бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶ ((20/5)*9) + 32 = 68В°Fбҹ”

1. бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠбһ‘бҹ…бһҖбһ¶бһ“бҹӢ Wio Terminalбҹ”
