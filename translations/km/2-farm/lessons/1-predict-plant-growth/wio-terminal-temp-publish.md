# бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ…бҹҒбһүбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– - Wio Terminal

бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ…бҹҒбһүбһҸбһҳбҹ’бһӣбҹғбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ¶бһ”бҹӢбһҷбһҖбһҠбҹ„бһҷ Wio Terminal бһҸбһ¶бһҳбһҡбһҷбҹҲ MQTT бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһҡбҹ„бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһҖбҹҶбһҺбһҫбһ“бһҖбһҳбҹ’бһҠбҹ…бһҠбһё (GDD)бҹ”

## бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ…бҹҒбһүбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“ бһңбһ¶бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ…бҹҒбһүбһҸбһ¶бһҳ MQTT бһ‘бҹ…бһҖбһјбһҠ 'бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ' бһҳбһҪбһҷбһҠбҹӮбһӣбһ“бһ№бһ„бһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ“бһ·бһ„бһ•бҹ’бһ‘бһ»бһҖбһңбһ¶бһІбҹ’бһҷбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһҡ GDDбҹ” бһҳбҹүбһ·бһ…бҹ’бһҡбһјбһҖбһ»бһ„бһҸаҘҚаӨ°аҘӢаӨІбһӣбһәбһҳбһ·бһ“бһўбһ¶бһ“бһҳбҹүбҹ„бһ„бһ–бһёбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһ бһҫбһҷбһҸбһ¶бһҳбһҠбһ¶бһ“бһҳбҹүбҹ„бһ„бһҠбҹ„бһҷбһ“бһ¶бһЎбһ·бһҖбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸбһҮбһ¶бһҳбһҪбһҷбһҡбһ¶бһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹ„бҹҮбһЎбһҫбһҷ бһўаёёаёӣаёҒаёЈаё“а№ҢбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһ”бҹӮбһ”бһ“бҹҒбҹҮ бһ”бһҫбһҹбһ·бһ“бһҮбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҳбһ·бһ“бһ”бһүбҹ’бһ…бһјбһ“бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҳбһҖбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹүбһҫбһ“бһҹбҹҗбһҡбһЎбһҫбһҷ бһҸбҹӮбһҖбһјбһҠбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһҖбҹ’бһҡбҹ„бһҷбһ–бҹҒбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҹбһ¶бһҡбҹ”

### бһҸбҹҶбһҺбһ¶бһ„

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ…бҹҒбһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бҹ”

1. бһ”бһҫбһҖбһӮбҹҶбһҡбһј `temperature-sensor` Wio Terminal

1. бһ’бҹ’бһңбһҫбһЎбһҫбһ„бһңбһ·бһүбһҮбҹҶбһ бһ¶бһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 4 бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT бһ“бһ·бһ„бһ”бһүбҹ’бһҮбһјбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ telemetry бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫ Mosquitto broker бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһҠбһҠбҹӮбһӣбҹ”

    бһҮбҹҶбһ бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ“бҹҒбҹҮбһҳбһ¶бһ“бҹ–

    - бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Seeed WiFi бһ“бһ·бһ„ MQTT бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `.ini`
    - бһ”бһ“бҹ’бһҗбҹӮбһҳбһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бһ·бһ„бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… WiFi
    - бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT broker
    - бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ…бҹҒбһү telemetry

    > вҡ пёҸ бһҹбһјбһҳбһҷбҹ„бһ„бһ‘бҹ… [бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT](../../../1-getting-started/lessons/4-connect-internet/wio-terminal-mqtt.md) бһ“бһ·бһ„ [бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһҮбһјбһ“ telemetry](../../../1-getting-started/lessons/4-connect-internet/wio-terminal-telemetry.md) бһ–бһёбһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 4 бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ”

1. бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһІбҹ’бһҷбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ `CLIENT_NAME` бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `config.h` бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһӮбҹҶбһҡбһјбһ“бҹҒбҹҮбҹ–

    ```cpp
    const string CLIENT_NAME = ID + "temperature_sensor_client";
    ```

1. бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ telemetry бһҮбҹҶбһ“бһҪбһҹбһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһ–бһ“бҹ’бһӣбһә бһҹбһјбһҳбһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ¶бһ“бһ–бһёбһҹа№ҮаёҷбһҹбҹҗбһҡбһҠбһёбһўбҹҒбһ…бһ’бһё (DHT sensor) бһҖбҹ’бһ“бһ»бһ„бһӣбһҖбҹ’бһҒбһҺбҹҲбһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ JSON бһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ `temperature` бһҠбҹ„бһҷбһ”бҹ’бһҠбһјбһҡ function `loop` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `main.cpp`бҹ–

    ```cpp
    float temp_hum_val[2] = {0};
    dht.readTempAndHumidity(temp_hum_val);

    DynamicJsonDocument doc(1024);
    doc["temperature"] = temp_hum_val[1];
    ```

1. бһҸбһҳбҹ’бһӣбҹғбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҒбҹ’бһӣбһ¶бҹҶбһ„бһ‘бҹҒ - бһңбһ¶бһҳбһ·бһ“бһ”бҹ’бһҡбҹӮбһҖбҹ’бһӣбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһё бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҖбҹҶбһҺбһҸбҹӢ `delay` бһҖбҹ’бһ“бһ»бһ„ function `loop` бһ‘бҹ… бҹЎбҹ  бһ“бһ¶бһ‘бһёбҹ–

    ```cpp
    delay(10 * 60 * 1000);
    ```

    > рҹ’Ғ Function `delay` бһ‘бһ‘бһҪбһӣбһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһҮбһ¶бһҜбһҖбһҸбһ¶бһҳбһёбһӣбһёбһңбһ·бһ“бһ¶бһ‘бһё бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ’бҹ’бһңбһҫвҖӢбһІбҹ’бһҷвҖӢбһңбһ¶вҖӢбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ¶бһ“ бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҮбһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡбһҳбһҪбһҷбҹ” 1,000ms бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһңбһ·бһ“бһ¶бһ‘бһё, 60s бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ“бһ¶бһ‘бһё бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ 10 x (60s бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ“бһ¶бһ‘бһё) x (1000ms бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһңбһ·бһ“бһ¶бһ‘бһё) бһ•бҹ’бһҠбһӣбҹӢбһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣ бҹЎбҹ  бһ“бһ¶бһ‘бһёбҹ”

1. бһўбһ¶бһ”бҹӢбһЎбһјбһҠбһ“бҹҒбҹҮбһ‘бҹ… Wio Terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһ“бһёбһ‘бҹҗбһҡбһҠбһ¶бһҹбһ»бһёбһҮбһ¶бһӣбһҜбһҖбһҹбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҖбҹҶбһ–бһ»бһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһүбһҫбһ‘бҹ… MQTT brokerбҹ”
