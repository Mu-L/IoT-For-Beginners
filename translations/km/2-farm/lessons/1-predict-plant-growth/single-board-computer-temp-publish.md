# бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– - бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһҖбһ“бһ·бһ„ Raspberry Pi

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҠбҹӮбһӣбһҡбһҖбһғбһҫбһүбһҠбҹ„бһҷ Raspberry Pi бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһҖбһҸбһ¶бһҳбһҡбһҷбҹҲ MQTT бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺ GDDбҹ”

## бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–

бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ¶бһ“бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҡбһҪбһ… бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһңбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲ MQTT бһ‘бҹ…бһҖбһјбһҠ 'бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ' бһҳбһҪбһҷбһҠбҹӮбһӣбһ“бһ№бһ„бһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ“бҹ„бҹҮ бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҡбһҪбһ…бһҡбһҪбһҳбһҡбһҪбһ…бһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ бҹҒбһҸбһ»бһҖбһ¶бһҡбһҺбҹҚ GDDбҹ”

### бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… - бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бҹ”

1. бһ”бһҫбһҖбһӮбҹҶбһҡбһјбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё `temperature-sensor` бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһңбһ¶бһ“бҹ…бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһ¶бһ“бһ”бһҫбһҖ

1. бһ’бҹ’бһңбһҫбһЎбһҫбһ„бһңбһ·бһүбһҮбҹҶбһ бһ¶бһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“ 4 бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT бһ“бһ·бһ„бһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ Mosquitto бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһҠбһҠбҹӮбһӣбҹ”

    бһҮбҹҶбһ бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ“бҹҒбҹҮбһҳбһ¶бһ“:

    - бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһүбҹ’бһ…бһ”бҹӢ pip MQTT
    - бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MQTT
    - бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„

    > вҡ пёҸ бһҹбһјбһҳбһҷбҹ„бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢ [бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… MQTT](../../../1-getting-started/lessons/4-connect-internet/single-board-computer-mqtt.md) бһ“бһ·бһ„ [бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„](../../../1-getting-started/lessons/4-connect-internet/single-board-computer-telemetry.md) бһ–бһёбһҳбҹҒбһҡбҹҖбһ“ 4 бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ”

1. бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһІбҹ’бһҷ `client_name` бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ“бһјбһңбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбҹ–

    ```python
    client_name = id + 'temperature_sensor_client'
    ```

1. бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһ–бһ“бҹ’бһӣбһә бһҹбһјбһҳбһ•бҹ’бһүбһҫбһҸбһҳбҹ’бһӣбҹғбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ¶бһ“бһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹӮбһ“ DHT бһҖбҹ’бһ“бһ»бһ„бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ JSON бһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ `temperature`бҹ–

    ```python
    _, temp = sensor.read()
    telemetry = json.dumps({'temperature' : temp})
    ```

1. бһҸбһҳбҹ’бһӣбҹғбһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ¶бһ“бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢ - бһңбһ¶бһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһёбһ“бҹ„бҹҮбһ‘бҹҒ бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһҹбһјбһҳбһҖбҹҶбһҺбһҸбҹӢ `time.sleep` бһ‘бҹ… бҹЎбҹ  бһ“бһ¶бһ‘бһёбҹ–

    ```cpp
    time.sleep(10 * 60);
    ```

    > рҹ’Ғ бһҳбһ»бһҒбһ„бһ¶бһҡ `sleep` бһ‘бһ‘бһҪбһӣбһҷбһҖбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҮбһ¶бһ‘ Sekunden бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбһ¶бһ“бһ”бһ¶бһ“бһ„бһ¶бһҷ бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһҮбһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҖбһ·бһҡбһ·бһҷбһ¶бһҹбһҳбҹ’бһ”бһ‘бһ¶бҹ” бҹҰбҹ  бһҹбһ·бһ“бһ¶бһ‘бһёбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ“бһ¶бһ‘бһё бһ‘бһҫбһ” бҹЎбҹ  x (бҹҰбҹ  бһҹбһ·бһ“бһ¶бһ‘бһёбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ“бһ¶бһ‘бһё) бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣ бҹЎбҹ  бһ“бһ¶бһ‘бһёбҹ”

1. бһҡбһҸбҹӢбһҖбһјбһҠбһҠбһјбһ…бһҮбһ¶бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбһҸбҹӢбһҖбһјбһҠбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһ“бһ“бҹғбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ IoT бһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһ… бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё CounterFit бһҖбҹҶбһ–бһ»бһ„бһҡбһҸбҹӢ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹӮбһ“бһҹбҹҶбһҺбһҫбһҳбһ”бһҡбһ·бһҷбһ¶бһҷ бһ“бһ·бһ„бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһӣбһҫбһ–бһ·бһ“бһҒбҹ’бһҳбҹӮбһҡбһ–бһёбһҺбһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”
