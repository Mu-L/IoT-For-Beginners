# កសិកម្មជាមួយ IoT

នៅពេលដែលប្រជាជនកើនឡើង ការទាមទារដល់វិស័យកសិកម្មក៏រីកចម្រើនដែរ។ បរិមាណដីដែលមានកំណត់មិនប្រែប្រួលឡើយ ប៉ុន្តែអាកាសធាតុប្រែប្រួល - ដែលផ្តល់បញ្ហាបន្ថែមទៅកសិករមួយចំនួន ជាពិសេសគឺកសិករជិត 2 ពាន់លាននាក់ [កសិករអាស្រ័យលើខ្លួនឯង](https://wikipedia.org/wiki/Subsistence_agriculture) ដែលពឹងផ្អែកលើការដាំដូលដើម្បី អាចញុំា និងផ្គត់ផ្គង់អាហារឲ្យគ្រួសាររបស់ពួកគេ។ IoT អាចជួយកសិករទទួលបានការសម្រេចចិត្តឆ្លាតវៃជាងមុនលើអ្វីដែលត្រូវដាំនិងពេលវេលាចម្អិនវា កង្វះបរិមាណការងារដោយដៃ និងរកឃើញនិងដោះស្រាយបញ្ហាសត្វល្អិត។

នៅក្នុងមេរៀនទាំង 6 នេះ អ្នកនឹងរៀនពីរបៀបអនុវត្តអ៊ិនធឺណិតវត្ថុ ដើម្បីធ្វើឱ្យការដាំដុះកសិកម្មកាន់តែប្រសើរនិងអូតូម៉ាទិក។

> 💁 មេរៀនទាំងនេះនឹងប្រើប្រាស់ធនធានមេឃមួយចំនួន។ ប្រសិនបើអ្នកមិនបានបញ្ចប់មេរៀនទាំងអស់ក្នុងគម្រោងនេះ សូមប្រាកដថាអ្នកបាន [សម្អាតគម្រោងរបស់អ្នក](../clean-up.md)។
