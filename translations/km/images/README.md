# រូបភាព

រូបភាពនៅក្នុងថត [icons](../../../images/icons) នេះគឺមកពី [Noun Project](https://thenounproject.com) ហើយត្រូវការការទទួលស្គាល់អ្នកផ្តល់។ រូបភាពនីមួយៗបង្ហាញពីការទទួលស្គាល់ដែលត្រូវការ។ រូបភាពទាំងនេះគួរត្រូវបានប្រើសម្រាប់គំនូសតាងណាមួយដែលត្រូវការពួកវាដើម្បីរក្សាទំរង់រូបភាពឲ្យស្របតាមគ្នា។

ឯកសារ [Diagrams.sketch](../../../images/Diagrams.sketch) គឺជាឯកសារ [Sketch](https://www.sketch.com) ដែលមានគំនូសតាងទាំងអស់ដែលបានប្រើនៅក្នុងមេរៀន និងគម្រោង។ ប្រសិនបើអ្នកចូលរួមក្នុងការបកប្រែ សូមបកប្រែផងដែរដោយឯកសារនេះដោយ either ដកស្រង់និងបន្ទាន់សម័យឯកសារ ឬផ្តល់បញ្ជីពាក្យដែលអ្នកត្រូវការបកប្រែជាបញ្ហាមួយនៅលើ GitHub។

រូបភាពដែលនៅសល់ត្រូវបានប្រើនៅក្នុងមេរៀន និងគម្រោងទាំងអស់។
