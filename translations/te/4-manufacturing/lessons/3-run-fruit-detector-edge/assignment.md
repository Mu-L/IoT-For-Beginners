# ఎడ్జ్‌పై ఇతర సేవలను నడపండి

## సూచనలు

ఎడ్జ్‌పై నడపగలిగేది కేవలం ఇమేజ్ క్లాసిఫయర్లే కాదు, కంటైనర్‌లో ప్యాకేజ్ చేయగలిగే ఏదైనా ఐఓటి ఎడ్జ్ డివైసు సమూహంలో అమర్చవచ్చు. మీరు ముందు పాఠాల్లో సృష్టించిన ట్రిగ్గర్స్ వంటి అదృష్టకోడ్ Azure Functionsగా నడపవచ్చు, అందువలన అవి ఐఓటి ఎడ్జ్‌లో కంటైనర్లలో నడపబడతాయి.

మునుపటి పాఠాల్లో ఒకదాన్ని ఎంచుకుని Azure Functions యాప్‌ను ఐఓటి ఎడ్జ్ కంటైనర్‌లో నడిపేందుకు ప్రయత్నించండి. మీరు దాన్ని ఎలా చేయాలో చూపించే గైడ్‌ను [Tutorial: Deploy Azure Functions as IoT Edge modules on Microsoft docs](https://docs.microsoft.com/azure/iot-edge/tutorial-deploy-function?WT.mc_id=academic-17441-jabenn&view=iotedge-2020-11) లో కనుగొనవచ్చు.
