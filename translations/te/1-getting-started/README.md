# IoT ప్రారంభించడం

కరిక్యులంలోని ఈ విభాగంలో, మీరు ఇంటర్నెట్ ఆఫ్ థింగ్స్‌ను పరిచయం చేయబడతారు మరియు క్లౌడ్‌కు కనెక్ట్ అయ్యే మీ తొలి 'హెల్లో వరల్డ్' IoT ప్రాజెక్ట్ నిర్మాణం సహా ప్రాథమిక ধারণలను నేర్చుకుంటారు. ఈ ప్రాజెక్ట్ ఒక నైట్‌లైట్, ఇది సెన్సార్ ద్వారా కొలవబడిన లైట్ స్థాయిలు తక్కువగా ఉన్నప్పుడు వెలిగుతుంది.

![లైట్ స్థాయి మారుతున్నప్పుడు WIO-కి కనెక్ట్ చేయబడిన LED ఆన్ మరియు ఆఫ్ అవుతుంది](../../../images/wio-running-assignment-1-1.gif)
