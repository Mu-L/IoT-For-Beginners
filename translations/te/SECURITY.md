# భద్రత

మైక్రోసాఫ్ట్ మా సాఫ్ట్‌వేర్ ఉత్పత్తులు మరియు సేవల భద్రతను గంభీరంగా తీసుకుంటుంది, దీనిలో మా GitHub సంస్థల ద్వారా నిర్వహించబడుతున్న అన్ని సోర్స్ కోడ్ రిపోజిటరీలు ఉన్నాయి, అవి [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet), [Xamarin](https://github.com/xamarin), మరియు [మా GitHub సంస్థలు](https://opensource.microsoft.com/)ని కలిగి ఉన్నాయి.

మీకు ఏదైనా మైక్రోసాఫ్ట్-సంబంధిత రిపోజిటరీలో [మైక్రోసాఫ్ట్ భద్రతా లోపం నిర్వచనం](https://docs.microsoft.com/en-us/previous-versions/tn-archive/cc751383(v=technet.10))కి అనుగుణంగా భద్రతా లోపం కనుక్కొన్నట్లు భావిస్తే, దయచేసి క్రింద వివరించిన విధంగా మాకు నివేదించండి.

## భద్రతా సమస్యల నివేదిక

**దయచేసి భద్రతా లోపాలను పబ్లిక్ GitHub ఇష్యూల ద్వారా నివేదించవద్దు.**

దాని బదులు, దయచేసి మైక్రోసాఫ్ట్ భద్రత ప్రతిస్పందన కేంద్రానికి (MSRC) [https://msrc.microsoft.com/create-report](https://msrc.microsoft.com/create-report) వద్ద నివేదించండి.

లాగిన్ చేయకుండా పంప్ చేయాలని ఇష్టపడ్డట్లయితే, [secure@microsoft.com](mailto:secure@microsoft.com) ఈ-మెయిల్‌కు పంపండి. సాధ్యమైతే, మా PGP కీతో మీ సందేశాన్ని సంకేతీకరించండి; దాన్ని దయచేసి [Microsoft Security Response Center PGP Key పేజీ](https://www.microsoft.com/en-us/msrc/pgp-key-msrc) నుండి డౌన్లోడ్ చేసుకోండి.

మీకు 24 గంటలలో స్పందన అందాలి. ఎటువంటి కారణంగా అందకపోతే, దయచేసి ఇమెయిల్ ద్వారా ఫాలో అప్ చేయండి, తద్వారా మేము మీ మొదటి సందేశం అందుకున్నట్లే నిర్ధారించుకోగలమని. అదనపు సమాచారం [microsoft.com/msrc](https://www.microsoft.com/msrc) వద్ద అందుబాటులో ఉంది.

దయచేసి క్రింద సూచించిన సమాచారాన్ని (మీరు అందించగలంతవరకు) చేర్చండి, తద్వారా సమస్య స్వభావం మరియు పరిధిని మేము మెరుగ్గా అర్థం చేసుకోగలము:

  * సమస్య రకం (ఉదాహరణకి బఫర్ ఓవర్‌ఫ్లో, SQL ఇంజెక్షన్, క్రాస్-సైట్ స్క్రిప్టింగ్, మొదలయినవి)
  * సమస్య ప్రదర్శనకు సంబంధించిన సోర్స్ ఫైల్‌ల పూర్తి మార్గాలు
  * ప్రభావిత సోర్స్ కోడ్ స్థానం (ట్యాగ్/ శాఖ/ కమిట్ లేదా ప్రత్యక్ష URL)
  * సమస్యను పునరుత్పత్తి చేయడానికి అవసరమైన ప్రత్యేక ఆకృతీకరణలు
  * సమస్యను పునరుత్పత్తి చేసే దశల వారీ సూచనలు
  * ప్రూఫ్-ఆఫ్-కాన్సెప్ట్ లేదా ఎక్స్‌పాయిట్ కోడ్ (అవసరమైతే)
  * సమస్య ప్రభావం, దానిని ఎట్టిపరిస్థితుల్లో దాడివాడు ఎలా ఉపయోగించవచ్చో సహితం

ఈ సమాచారం మీ నివేదనను త్వరగా పరిశీలించడంలో సహాయపడుతుంది.

మీరు బగ్ బౌంటీ కోసం నివేదిస్తున్నట్లయితే, పూర్తి నివేదికలు ఎక్కువ బౌంటీ అవార్డుకు దోహదపడతాయి. మా [Microsoft Bug Bounty ప్రోగ్రామ్](https://microsoft.com/msrc/bounty) పేజీని సందర్శించి మా కార్యకలాపాల గురించి మరింత తెలుసుకోండి.

## ఇష్టమయ్యే భాషలు

మేము అన్ని సంభాషణలు ఇంగ్లీష్‌లో ఉండటం ఇష్టపడతాము.

## విధానము

మైక్రోసాఫ్ట్ [సహకార భద్రతా లోపం ప్రకటణ](https://www.microsoft.com/en-us/msrc/cvd) సూత్రాన్ని అనుసరిస్తుంది.
