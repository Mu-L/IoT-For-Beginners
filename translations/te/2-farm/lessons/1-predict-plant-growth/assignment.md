# Jupyter నోట్‌బుక్ ఉపయోగించి GDD డేటా ప్రదర్శించండి

## సూచనలు

ఈ పాఠంలో మీరు IoT సెన్సార్ ఉపయోగించి GDD డేటా సేకరించారు. మంచి GDD డేటా పొందడానికి, మీరు అనేక రోజులు డేటా సేకరించాలి. ఉష్ణోగ్రత డేటాను ప్రదర్శించడానికి మరియు GDD ను లెక్కించడానికి మీరు [Jupyter Notebooks](https://jupyter.org) వంటి సాధనాలను ఉపయోగించి డేటాను విశ్లేషించవచ్చు.

కొన్ని రోజులకోసం డేటా సేకరణ ప్రారంభించండి. మీ IoT పరికరం నడుస్తున్నప్పుడల్లా మీ సర్వర్ కోడ్ కూడా నడుస్తుండేలా చూడాలి, ఇది మీరు పవర్ మేనేజ్‌మెంట్ సెట్టింగులను సరిచేయడం ద్వారా లేదా [ఈ keep system active Python స్క్రిప్ట్](https://github.com/jaqsparow/keep-system-active) లాంటి దీన్ని నడిపించడం ద్వారా సాధించవచ్చు.

ఉష్ణోగ్రత డేటా వచ్చినప్పుడు, ఈ రిపోజ్‌లోని Jupyter నోట్‌బుక్ ఉపయోగించి దాన్ని ప్రదర్శించి GDD ను లెక్కించవచ్చు. Jupyter నోట్‌బుక్స్ కోడ్ మరియు సూచనలను *cells* అని పిలవబడే బ్లాక్స్‌లో కలిపి చూపిస్తాయి, సాధారణంగా Python కోడ్ ఉంటాయి. మీరు సూచనలను చదవవచ్చు, ఆపై ఒక్కొక్క కోడ్ బ్లాక్‌ను నడపవచ్చు. మీరు కోడ్‌ను కూడా సవరించవచ్చు. ఉదాహరణకు, ఈ నోట్‌బుక్‌లో మీరు మీ మొక్క కోసం GDD లెక్కించేందుకు ఉపయోగించే బేస్ ఉష్ణోగ్రతను సవరించవచ్చు.

1. `gdd-calculation` అనే ఫోల్డర్‌ని సృష్టించండి

1. [gdd.ipynb](./code-notebook/gdd.ipynb) ఫైల్‌ను డౌన్లోడ్ చేసి `gdd-calculation` ఫోల్డర్‌లోకాకు పెట్టండి.

1. MQTT సర్వర్ సృష్టించిన `temperature.csv` ఫైల్‌ను కాపీ చేయండి

1. `gdd-calculation` ఫోల్డర్‌లో కొత్త Python వర్చువల్ ఎన్‌వైਰਨ్మెంట్ సృష్టించండి.

1. Jupyter నోట్‌బుక్స్ మరియు డేటా నిర్వహణ, ప్లాటింగ్‌కు అవసరమైన లైబ్రరిలను ఇన్‌స్టాల్ చేయండి:

    ```sh
    pip install --upgrade pip
    pip install pandas
    pip install matplotlib
    pip install jupyter
    ```

1. Jupyterలో నోట్‌బుక్ నడపండి:

    ```sh
    jupyter notebook gdd.ipynb
    ```

    Jupyter ఆరంభమై బ్రౌజర్‌లో నోట్‌బుక్ తెరుస్తుంది. కొలవబడిన ఉష్ణోగ్రతలను ప్రదర్శించడానికి మరియు GDD లెక్కించడానికి నోట్‌బుక్‌లోని సూచనలను అనుసరించండి.
