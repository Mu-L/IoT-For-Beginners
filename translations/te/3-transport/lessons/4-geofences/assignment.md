# Twilio ఉపయోగించి నోటిఫికేషన్లు పంపండి

## సూచనలు

మీ కోడ్ ఇప్పటి వరకు మీరు జియోఫెన్స్ కీ దూరాన్ని మాత్రమే లాగ్ చేసారు. ఈ అసైన్‌మెంట్‌లో మీరు GPS సమన్వయాలు జియోఫెన్స్ క్లోజ్ లో ఉన్నప్పుడు ఒక నోటిఫికేషన్, అంటే టెక్స్ట్ మెసేజ్ లేదా ఇమెయిల్, జోడించాలి.

Azure Functions లో అనేక బైండింగ్స్ ఎంపికలు ఉన్నాయి, వాటిలో Twilio వంటి మూడవ పార్టీ సర్వీసులకు బైండింగ్స్ ఉన్నాయి, ఇది ఒక కమ్యూనికేషన్స్ ప్లాట్‌ఫాం.

* [Twilio.com](https://www.twilio.com) లో ఉచిత ఖాతాకు సైన్ అప్ అవ్వండి
* [Microsoft docs Twilio binding for Azure Functions పేజీ](https://docs.microsoft.com/azure/azure-functions/functions-bindings-twilio?WT.mc_id=academic-17441-jabenn&tabs=python)లో Azure Functions ని Twilio SMS కి బైండ్ చేసే డాక్యుమెంటేషన్ చదవండి.
* [Microsoft docs Azure Functions SendGrid bindings పేజీ](https://docs.microsoft.com/azure/azure-functions/functions-bindings-sendgrid?WT.mc_id=academic-17441-jabenn&tabs=python)లో Azure Functions ని Twilio SendGrid కి బైండ్ చేసి ఇమెయిల్స్ పంపే డాక్యుమెంటేషన్ చదవండి.
* జియోఫెన్స్ లో ఉనికిని లేదా బయట ఉన్న GPS సమన్వయాలకు నోటిఫికేషన్ అందించాలని మీ Functions యాప్‌కు బైండింగ్ జోడించండి - రెండింటినీ కాదు.
