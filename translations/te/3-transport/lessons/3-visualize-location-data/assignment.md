# మీ యాప్‌ను ప్రదర్శించండి

## సూచనలు

మీ యాప్‌ను ప్రపంచంతో పంచుకునేందుకు మీరు గిత్‌హబ్ పేజీలను ఉపయోగించడం లేదా అనేక సర్వీస్ ప్రొవైడర్లలో ఏదో ఒకదాన్ని ఉపయోగించడం వంటి అనేక మార్గాలు ఉన్నాయి. దీనిని చేయడానికి ఒక అద్భుతమైన మార్గం Azure Static Web Apps ను ఉపయోగించడం. ఈ అసైన్‌మెంట్‌లో, మీ వెబ్ యాప్‌ను నిర్మించి [ఈ సూచనలను](https://github.com/Azure/static-web-apps-cli) అనుసరించి లేదా [ఈ వీడియోలను](https://www.youtube.com/watch?v=ADVGIXciYn8&list=PLlrxD0HtieHgMPeBaDQFx9yNuFxx6S1VG&index=3) చూడటం ద్వారా క్లౌడ్‌లో డిప్లాయ్ చేయండి.
Azure Static Web Apps ఉపయోగించడంతో లాభం ఏమంటే మీరు ఏ API కీలు పోర్టల్లో దాచేయవచ్చు, కాబట్టి ఈ అవకాశాన్ని వాడుకుని మీ subscriptionKey ని ఒక వేరియబుల్‌గా మార్చి క్లౌడ్‌లో స్టోర్ చేసుకోండి.
