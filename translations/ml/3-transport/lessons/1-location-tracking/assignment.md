# മറ്റു GPS ഡാറ്റ അന്വേഷണ ചെയ്തുക

## നിർദ്ദേശങ്ങൾ

നിങ്ങളുടെ GPS സെൻസറിൽ നിന്ന് വരുന്ന NMEA വാക്യങ്ങൾക്ക് സ്ഥലം കൂടാതെ മറ്റ് ഡാറ്റയും ഉണ്ടായിരിക്കുന്നു. ചേർന്ന ഡാറ്റയെ അന്വേഷിച്ച് നിങ്ങളുടെ IoT ഉപകരണത്തിൽ ഉപയോഗിക്കുക.

ഉദാഹരണത്തിന് - നിങ്ങൾക്ക് പ്രస్తుత തിയ്യതി, സമയം ലഭിക്കാമോ? നിങ്ങൾ മൈക്രോകൺട്രോളർ ഉപയോഗിക്കുന്നുവെങ്കിൽ, NTP സിഗ്നലുകൾ ഉപയോഗിച്ച് നിശ്ചയിച്ച പോലെ GPS ഡാറ്റ ഉപയോഗിച്ച് ക്ലോക്ക് ക്രമീകരിക്കാമോ? ഉയരം (കടൽതലത്തിലേയ്ക്ക് നിങ്ങളുടെ ഉയരം) അല്ലെങ്കിൽ നിലവിലെ വേഗം ലഭിക്കാമോ?

നിങ്ങൾ വിർച്വൽ IoT ഉപകരണം ഉപയോഗിക്കുന്നുവെങ്കിൽ, [nmeagen.org](https://www.nmeagen.org) ഉപകരണങ്ങൾ ഉപയോഗിച്ച് സൃഷ്ടിക്കുന്ന NMEA വാക്യങ്ങൾ അയച്ചു ഇതിൽ നിന്നുള്ള ചില ഡാറ്റ ലഭ്യമാക്കാം.
