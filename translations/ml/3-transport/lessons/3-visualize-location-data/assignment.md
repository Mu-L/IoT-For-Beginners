# നിങ്ങളുടെ ആപ്പ് വിന്യസിക്കുക

## നിര്‍ദ്ദേശങ്ങള്‍

നിങ്ങളുടെ ആപ്പ് ലോകത്തോടു പങ്കുവെക്കാനുള്ള നിരവധി വഴികളുണ്ട്, അതില്‍ GitHub പേജുകള്‍ ഉപയോഗിക്കുന്നതോ അല്ലെങ്കില്‍ നിരവധി സേവനദായകരില്‍ ഒന്നിനെ ഉപയോഗിക്കുന്നതോ ഉള്‍പ്പെടുന്നു. ഇതിനായി വളരെ നല്ലൊരു മാര്‍ഗമാണ് Azure Static Web Apps ഉപയോഗിക്കുന്നത്. ഈ അസൈൻമെന്റിൽ, നിങ്ങളുടെ വെബ് ആപ്പ് നിർമ്മിച്ച് ഈ [നിര്‍ദ്ദേശങ്ങള്‍](https://github.com/Azure/static-web-apps-cli) പിന്തുടർന്ന് അല്ലെങ്കിൽ ഈ [വിഡിയോകള്‍](https://www.youtube.com/watch?v=ADVGIXciYn8&list=PLlrxD0HtieHgMPeBaDQFx9yNuFxx6S1VG&index=3) കണ്ടു ക്ലൗഡിലേക്ക് വിന്യസിക്കുക.  
Azure Static Web Apps ഉപയോഗിക്കുന്നതിന് ഒരു പ്രയോജനം എന്നത് നിങ്ങളുടെ API കീകൾ പോര്‍ട്ടലിൽ മറച്ചു വെക്കാനാകുന്നതാണ്, അതുകൊണ്ട് നിങ്ങളുടെ subscriptionKey പേരായി മാറ്റി ക്ലൗഡിൽ സൂക്ഷിക്കാൻ ഈ അവസരം ഉപയോഗിക്കൂ.
