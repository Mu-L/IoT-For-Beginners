# Twilio ഉപയോഗിച്ച് അറിയിപ്പുകൾ അയയ്ക്കുക

## നിർദ്ദേശങ്ങൾ

ഇപ്പോൾവരെ നിങ്ങളുടെ കോഡിൽ, ജിയോഫെൻസിലേക്ക് ദൂരം മാത്രം ലോഗ് ചെയ്തിട്ടുണ്ട്. ഈ അസിൻമെന്റിൽ ജിപിഎസ് സംയോജിതങ്ങൾ ജിയോഫെൻസിനുള്ളിൽ ഉണ്ടായപ്പോൾ ടെക്സ്റ്റ് മെസേജ് അല്ലെങ്കിൽ ഇമെയിൽ എന്നൊരു അറിയിപ്പ് ചേർക്കേണ്ടതുണ്ട്.

Azure Functions ന് പല ബൈൻഡിംഗുകൾ ഉണ്ട്, Twilio പോലുള്ള മൂന്നാം കേരള സേവനങ്ങളോടും ഉൾപ്പെടെ, ഒരു കമ്മ്യൂണിക്കേഷൻ പ്ലാറ്റ്ഫോം.

* [Twilio.com](https://www.twilio.com) - ല്ലോ സൗജന്യ അക്കൗണ്ടിന് സൈൻ അപ്പ് ചെയ്യുക
* Twilio SMS-ന്റെ Azure Functions ബൈൻഡിംഗിനെക്കുറിച്ചുള്ള ഡോക്യുമെന്റേഷൻ [Microsoft docs Twilio binding for Azure Functions page](https://docs.microsoft.com/azure/azure-functions/functions-bindings-twilio?WT.mc_id=academic-17441-jabenn&tabs=python) ചെക്ക് ചെയ്യുക.
* Twilio SendGrid ഉപയോഗിച്ച് ഇമെയിലുകൾ അയയ്ക്കാൻ Azure Functions ബൈൻഡിങ് ഡോക്യുമെന്റേഷൻ [Microsoft docs Azure Functions SendGrid bindings page](https://docs.microsoft.com/azure/azure-functions/functions-bindings-sendgrid?WT.mc_id=academic-17441-jabenn&tabs=python) ൽ വായിക്കുക.
* ജിപിഎസ് സംയോജിതങ്ങൾ ജിയോഫെൻസിനുള്ളോ പുറത്തോ ഉള്ളപ്പോൾ അറിയിപ്പ് ലഭിക്കുന്നവിധം നിങ്ങളുടെ Functions ആപ്പിൽ ബൈൻഡിങ് ചേർക്കുക – രണ്ടും അല്ല.

| -------- | --------- | -------- | ----------------- |
| ഫങ്ഷൻസ് ബൈൻഡിങ്ങുകൾ ക്രമീകരിച്ച് ഇമെയിൽ അല്ലെങ്കിൽ SMS ലഭിക്കുക | ജിയോഫെൻസിനുള്ളിലോ പുറത്തോ ഉള്ളപ്പോൾ ഇമെയിൽ അല്ലെങ്കിൽ SMS ലഭിക്കുന്ന വിധത്തിൽ ഫങ്ഷൻസ് ബൈൻഡിങ്ങുകൾ ക്രമീകരിക്കാൻ കഴിഞ്ഞു, പക്ഷേ രണ്ടും അല്ല | ബൈൻഡിങ്ങുകൾ ക്രമീകരിക്കാൻ കഴിഞ്ഞു, എന്നാൽ ഇമെയിൽ അല്ലെങ്കിൽ SMS അയക്കാൻ സാധിക്കാനായില്ല, അല്ലെങ്കിൽ കോഓർഡിനേറ്റുകൾ ജിയോഫെൻസിനുള്ളിലുമുള്ളപ്പോൾ ഒറ്റപ്പോഴേ അയക്കാൻ കഴിയുകയുമായിരുന്നുവെങ്കിൽ | ബൈൻഡിങ്ങുകൾ ക്രമീകരിക്കാൻ കഴിയാതെ ഇമെയീൽ അല്ലെങ്കിൽ SMS അയക്കാൻ സാധിക്കാതെ പോയിരിക്കുന്നു |
