# മുതിർന്നവർക്കുള്ള IoT - ഒരു പാഠ്യപദ്ധതി

Microsoft-ലെ Azure ക്ലൗഡ് അഭിഭാഷಕರು IoT അടിസ്ഥാനങ്ങളേക്കുറിച്ചുള്ള 12 ആഴ്ച, 24 പാഠങ്ങൾ അടങ്ങിയ ഒരു പാഠ്യപദ്ധതി ഒരുക്കിയത് സന്തോഷകരമാണ്. ഓരോ പാഠവും പ്രാക്ടീസ്-ക്വിസുകളും, പാഠം പൂർത്തിയാക്കാനുള്ള എഴുത്തൂരുന്ന നിർദ്ദേശങ്ങളും, പരിഹാരവും, അസൈൻമെന്റുമാണ് ഉൾക്കൊള്ളുന്നത്. ഞങ്ങളുടെ പ്രോജക്ട് അടിസ്ഥാനത്തിലുള്ള പഠനരീതിയിലൂടെ നിങ്ങൾ നിർമ്മിക്കുമ്പോൾ പഠിക്കും, ഇത് പുതിയ കഴിവുകൾ ഉറപ്പിക്കാൻ തെളിയിച്ച രീതിയാണ്.

ഈ പ്രോജക്ടുകൾ ഭക്ഷ്യപദാർത്ഥങ്ങളുടെ സ്ഥലം മുതൽ മേശയിലേക്ക് വരെയുള്ള യാത്രയെയാണ് ഉൾക്കൊള്ളിക്കുന്നത്. ഇത് കൃഷി, ലജിസ്റ്റിക്സ്, നിർമ്മാണം, റീറ്റെയിൽ, ഉപഭോക്താവ് എന്നീ IoT ഉപകരണങ്ങൾക്ക് ജനപ്രിയമായ വ്യാവസായിക മേഖലകളെയാണ് ഉൾക്കൊള്ളിക്കുന്നത്.

## പഠനരീതികൾ

ഈ പാഠ്യപദ്ധതി രൂപീകരിക്കുമ്പോൾ രണ്ട് അടിസ്ഥാന പഠന തത്വങ്ങൾ തിരഞ്ഞെടുക്കപ്പെട്ടിരിക്കുന്നു: ഇത് പ്രോജക്ട് അടിസ്ഥാനമുള്ളതും, ხშირമായി ക്വിസ് ഉൾക്കൊള്ളുന്നതുമായതും ആകണം. സീരീസിന്റെ അവസാനം വിദ്യാർത്ഥികൾ ഒരു പ്ലാന്റ് നിരീക്ഷണവും വെള്ളം നൽകൽ സംവിധാനവും, വാഹന ട്രാക്കറും, ഭക്ഷണ പരിശോധനയ്ക്കുള്ള ഒരു സ്മാർട്ട് ഫാക്ടറിയും, ശബ്ദനിയന്ത്രിതമായ പാചക ടൈമറിനും രൂപകല്‌പന ചെയ്ത് പഠിച്ചിരിക്കുമെന്നും ഉറപ്പുണ്ട്. ഇതിന്റെ കൂടെ, ഡിവൈസ് കോഡ് എഴുതൽ, ക്ലൗഡുമായി കണക്റ്റ് ചെയ്യൽ, ടെലിമെട്രി വിശകലനം, എജിൽ AI ഓടിക്കൽ തുടങ്ങിയ IoT അടിസ്ഥാനങ്ങൾ പഠിക്കും.

വിഷയങ്ങൾ പ്രോജക്ടുകളുമായി പൊരുത്തപ്പെടുത്തിക്കൊണ്ടിരിക്കുന്നത് വിദ്യാർത്ഥികൾക്ക് കൂടുതൽ ആകർഷകവുമായ പഠനം നൽകുകയും ആശയങ്ങൾ കൂടുതൽ സ്ഥിരമാകാൻ സഹായിക്കുകയും ചെയ്യും.

ക്ലാസ് തുടങ്ങുന്നതിന് മുമ്പ് താഴ്ന്ന സമ്മർദ്ദമുള്ള ക്വിസ് വിദ്യാർത്ഥിയുടെ പഠനനോക്കത്തിന് രൂപം നല്കുന്നു, ക്ലാസ് കഴിഞ്ഞ് ഒരു രണ്ടാം ക്വിസ് അതിന്റെ തുടർച്ചയായ അറിവ് ഉറപ്പാക്കുന്നു. ഈ പാഠ്യപദ്ധതി ഫ്ലെക്സിബിളും രസകരവുമാണ്, മുഴുവനായോ ഭാഗികമായോ പഠിക്കാവുന്നതാണ്. പ്രോജക്ടുകൾ ചെറിയതായിത്തുടങ്ങി 12 ആഴ്ചാകാലയളവിന്റെ അവസാനം progressively സങ്കീർണ്ണമാകും.

ഓരോ പ്രോജക്ടും വിദ്യാർത്ഥികൾക്കും ഹോബിക്കാർക്കും ലഭ്യമായ യഥാർത്ഥ-ലോക ഹാർഡ്‌വെയറിനെയാണ് അടിസ്ഥാനമാക്കുന്നത്. ഓരോ പ്രോജക്ടും ബന്ധപ്പെട്ട വ്യവസായ മേഖലയിൽ കണ്ണു പുലർന്ന്, അനുയോജ്യമായ പശ്ചാത്തല അറിവ് നൽകുന്നു. ഒരു വിജയകരമായ ഡെവലപ്പറായി മാറാൻ പ്രശ്ന പരിഹാരത്തിനായി നിങ്ങൾ പരിഗണിക്കുന്ന മേഖലയെ മനസ്സിലാക്കുന്നത് സഹായിക്കുന്നു; ഇത്തരത്തിലുള്ള പശ്ചാത്തല അറിവ് വിദ്യാർത്ഥികൾക്ക് അവരുടെ IoT പരിഹാരങ്ങളെക്കുറിച്ചും പഠനങ്ങളെക്കുറിച്ചും ആഴത്തിൽ ചിന്തിക്കാൻ അനുവദിക്കുന്നതാണ്. അവർ നിർമ്മിക്കുന്ന പരിഹാരങ്ങളുടെ "എന്തിന്" എന്ന കാരണം പഠിക്കുകയും അന്തിമ ഉപഭോക്താവിന്റെ പ്രാധാന്യം അറിയുകയും ചെയ്യും.

## ഹാർഡ്‌വെയർ
ഞങ്ങൾക്ക് ഇഷ്ടാനുസൃതത, പ്രോഗ്രാമിംഗ് ഭാഷ പാലനം അല്ലെങ്കിൽ ഇഷ്ടങ്ങൾ, പഠന ലക്ഷ്യങ്ങൾ, ലഭ്യത എന്നിവയുടെ അടിസ്ഥാനത്തിൽ ഉപയോഗിക്കാവുന്ന രണ്ട് ഐഒടി ഹാർഡ്‌വെയർ ഓപ്ഷനുകൾ ഉണ്ട്. ഹാർഡ്‌വെയർ ലഭ്യമല്ലാത്തവർക്കോ, വാങ്ങുന്നതിന് മുന്‍പ് കൂടുതൽ പഠിക്കാൻ ആഗ്രഹിക്കുന്നവർക്കോ വേണ്ടി 'വർച്വൽ ഹാർഡ്‌വെയർ' പതിപ്പ് ഞങ്ങൾ നൽകിയിട്ടുണ്ട്. നിങ്ങൾക്ക് കൂടുതൽ വായിച്ച്, ഞങ്ങളുടെ സുഹൃത്തുക്കളായ സീഡ് സ്റ്റുഡിയോയിൽ നിന്നും സമ്പൂർണ്ണ കിറ്റുകൾ വാങ്ങാനുള്ള ലിങ്കുകൾ ഉൾപ്പെടെ ഒരു 'ഷോപ്പിംഗ് ലിസ്റ്റ്' [ഹാർഡ്‌വെയർ പേജ്](./hardware.md)യിൽ കാണാം.

>
