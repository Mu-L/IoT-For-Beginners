# പിന്തുണ

## പ്രശ്നങ്ങൾ രജിസ്റ്റർ ചെയ്യാനും സഹായം നേടാനും എങ്ങനെ  

ഈ പ്രോജക്റ്റ് ബഗുകളും ഫീച്ചർ അഭ്യർത്ഥനകളും ട്രാക്ക് ചെയ്യാൻ GitHub Issues ഉപയോഗിക്കുന്നു. പുനരാവൃതികളെ ഒഴിവാക്കാൻ മുമ്പുള്ള ലിസ്റ്റിൽ ഉള്ള പ്രശ്നങ്ങൾ പരിശോധിക്കുക. പുതിയ പ്രശ്നങ്ങൾക്ക്, നിങ്ങളുടെ ബഗ് അല്ലെങ്കിൽ ഫീച്ചർ അഭ്യർത്ഥനയെ ഒരു പുതിയ Issue ആയി രജിസ്റ്റർ ചെയ്യുക.

ഈ പ്രോജക്റ്റ് ഉപയോഗിക്കുന്നതിനുള്ള സഹായം അല്ലെങ്കിൽ ചോദ്യങ്ങൾക്കായി, ദയവായി ഈ റപ്പോയിൽ ഒരു പ്രശ്നം ഉയർത്തി ഞങ്ങളെ ബന്ധപ്പെടുക.

## Microsoft Support നയം  

ഈ **PROJECT or PRODUCT** നെ കിട്ടുന്ന പിന്തുണ മുകളിൽ കൊടുത്തിരിക്കുന്ന സ്രോതസ്സുകളിലാക്കി പരിമിതമാണ്.
