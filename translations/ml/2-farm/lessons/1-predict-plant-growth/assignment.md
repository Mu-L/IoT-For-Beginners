# Jupyter നോട്ട്ബുക്ക് ഉപയോഗിച്ച് GDD ഡാറ്റ ദൃശ്യവൽക്കരിക്കുക

## നിർദ്ദേശങ്ങൾ

ഈ പാഠത്തിൽ നിങ്ങൾ IoT സെൻസർ ഉപയോഗിച്ച് GDD ഡാറ്റ ശേഖരിച്ചു. നല്ല GDD ഡാറ്റ ലഭിക്കാൻ, നിങ്ങൾക്ക് പല ദിവസങ്ങളിലായും ഡാറ്റ ശേഖരിക്കേണ്ടതുണ്ട്. താപനില ഡാറ്റ ദൃശ്യവൽക്കരിക്കാനും GDD കണക്കാക്കാനുമുള്ള സഹായത്തിന് [Jupyter Notebooks](https://jupyter.org) പോലുള്ള ഉപകരണങ്ങൾ ഉപയോഗിക്കാം.

കുറച്ച് ദിവസങ്ങളിലായി ഡാറ്റ ശേഖരിക്കുക. നിങ്ങളുടെ IoT ഉപകരണം പ്രവർത്തിക്കുന്ന സമയത്ത് നിങ്ങളുടെ സെർവർ കോഡ് സ്ഥിരമായി പ്രവർത്തിക്കുന്നുണ്ടോ എന്ന് ഉറപ്പാക്കേണ്ടതാണ്, നിങ്ങളുടെ പവർ മാനേജ്‌മെന്റ് ക്രമീകരണങ്ങൾ ക്രമീകരിച്ചോ അല്ലെങ്കിൽ [ഈ സിസ്റ്റം പ്രവർത്തനക്ഷമമാക്കിയ Python സ്ക്രിപ്റ്റ്](https://github.com/jaqsparow/keep-system-active) പോലൊരു പ്രക്രിയ പ്രവർത്തിപ്പിച്ചോ.

താപനില ഡാറ്റ ലഭിച്ചതിനുശേഷം, ഈ റീപ്പോയിലെ Jupyter നോട്ട്ബുക്ക് ഉപയോഗിച്ച് അതിനു ദൃശ്യവൽക്കരണം നടത്തുകയും GDD കണക്കാക്കുകയും ചെയ്യാം. Jupyter നോട്ട്ബുക്കുകളിൽ *സെല്ലുകൾ* എന്ന് വിളിക്കുന്ന ബ്ലോകുകളിൽ കോഡ്, നിർദ്ദേശങ്ങൾ എന്നിവ ചേർത്തിരിക്കുന്നതാണ്, പലപ്പോഴും Python കോഡും. നിങ്ങൾ നിർദ്ദേശങ്ങൾ വായിച്ചുകൊണ്ട് ഓരോ കോഡ് ബ്ലോക്കും 실행ിക്കാം. കൂടാതെ കോഡ് തിരുത്താനും കഴിയും. ഉദാഹരണത്തിന്, ഈ നോട്ട്ബുക്കിൽ നിങ്ങളുടെ ചെടി GDD കണക്കാക്കാൻ അടിസ്ഥാന താപനില തിരുത്താൻ കഴിയും.

1. `gdd-calculation` എന്ന പേരിൽ ഒരു ഫോൾഡർ സൃഷ്‌ടിക്കുക

1. [gdd.ipynb](./code-notebook/gdd.ipynb) ഫയൽ ഡൗൺലോഡ് ചെയ്ത് `gdd-calculation` ഫോൾഡറിലേക്കു് പകർത്തുക.

1. MQTT സെർവർ സൃഷ്‌ടിച്ച `temperature.csv` ഫയൽ പകർത്തുക

1. `gdd-calculation` ഫോൾഡറിലെ പുതിയ Python വെർച്ച്വൽ എൻവയോൺമെന്റ് സൃഷ്‌ടിക്കുക.

1. Jupyter നോട്ട്ബുക്കുകൾക്കായി മാർഗ്ഗരേഖകൾ, ഡാറ്റ കൈകാര്യം ചെയ്യുന്നതിനും പ്ലോട്ട് ചെയ്യുന്നതിനുമുള്ള ലൈബ്രററികൾ ഉൾപ്പെടെയുള്ള ചില പിപ് പാക്കേജുകൾ ഇൻസ്റ്റാൾ ചെയ്യുക:

    ```sh
    pip install --upgrade pip
    pip install pandas
    pip install matplotlib
    pip install jupyter
    ```

1. Jupyter-ൽ നോട്ട്ബുക്ക് നടത്തുക:

    ```sh
    jupyter notebook gdd.ipynb
    ```

    Jupyter ആരംഭിച്ച് നിങ്ങളുടെ ബ്രൗസറിൽ നോട്ട്ബുക്ക് തുറക്കും. താപനില അളവുകളുടെ ദൃശ്യവൽകരണത്തിനും വളർച്ചാ ഡിഗ്രി ഡേസ് കണക്കാക്കുന്നതിനുമായുള്ള നിർദ്ദേശങ്ങൾ പിന്തുടരുക.
