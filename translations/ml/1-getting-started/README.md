# IoT കൊണ്ട് ആരംഭിക്കൽ

ഈ പഠനക്രമത്തിന്റെ ഈ ഭാഗത്തിൽ നിങ്ങൾക്ക് ഇന്റർനെറ്റ് ഓഫ് തിങ്‌സ് പരിചയപ്പെടുകയും, പ്രവൃത്തി ഘടന ഉൾപ്പെടെ അടിസ്ഥാന തത്വങ്ങൾ പഠിക്കുകയും, ക്ലൗഡുമായി ബന്ധിപ്പിക്കുന്ന നിങ്ങളുടെ ആദ്യ 'ഹെലോ വേൾഡ്' IoT പ്രോജക്ട് നിർമ്മിക്കാനും കഴിയും. ഇത് ഒരു നൈറ്റ് ലൈറ്റാണ്, സെൻസർ അളക്കുന്ന വെളിച്ചത്തിന്റെ നില താഴ്ന്നപ്പോൾ പ്രകാശിക്കാൻ തുടങ്ങുന്നത്.

![വെളിച്ചത്തിന്റെ തലത്തിലുണ്ടാകുന്ന മാറ്റം അനുസരിച്ച് ഓണാക്കുകയും ഓഫ് ചെയ്യുകയും ചെയ്യുന്ന WIOയോട് കണക്റ്റ് ചെയ്ത LED](../../../images/wio-running-assignment-1-1.gif)
