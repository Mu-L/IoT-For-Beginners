# അധ്യാപകർക്കായി

ഈ പാഠ്യപദ്ധതി നിങ്ങൾගේ ക്ലാസ്സിൽ ഉപയോഗിക്കാൻ ആഗ്രഹമാണോ? ദയവായി കൊണ്ട് സ്വതന്ത്രരാകൂ!

വാസ്തവത്തിൽ, ഇത് GitHub Classrom ഉപയോഗിച്ച് GitHub-ലേ തന്നെ ഉപയോഗിക്കാം.

അതിനായി, ഈ റിപ്പോ ഫോർക്ക് ചെയ്യുക. ഓരോ പാഠത്തിനും ഒരു റിപ്പോ സൃഷ്ടിക്കേണ്ടതിനാൽ, ഓരോ ഫോൾഡറും പ്രത്യേകം റിപ്പോയാക്കി വേർതിരിക്കേണ്ടതുണ്ട്. അങ്ങനെ, [GitHub Classroom](https://classroom.github.com/classrooms) ഓരോ പാഠവും വ്യത്യസ്തമായി കൈവരുത്താൻ കഴിയും.

ഈ [പൂർണ്ണ നിർദ്ദേശങ്ങൾ](https://github.blog/2020-03-18-set-up-your-digital-classroom-with-github-classroom/) നിങ്ങളുടെ ക്ലാസ്സ് എങ്ങനെ സജ്ജീകരിക്കാമെന്നുള്ള ഒരു ആശയം നൽകും.

## ശിപാർശചെയ്ത പഠന മോഡൽ

ഈ പാഠ്യപദ്ധതി പഠിപ്പിക്കുമ്പോൾ ശിപാർശചെയ്ത പഠന മോഡലിനെക്കുറിച്ച് കൂടുതൽ വായിക്കാൻ ഞങ്ങളുടെ [ശിപാർശാ പഠന മോഡൽ ഗൈഡ്](recommended-learning-model.md) പരിശോധിക്കാം.

## നിലവിലെ രൂപത്തിൽ റിപ്പോ ഉപയോഗിക്കൽ

GitHub Classroom ഉപയോഗിക്കാതെ ഇപ്പോഴുള്ള രൂപത്തിൽ ഈ റിപ്പോ ഉപയോഗിക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, അതും ചെയ്യാനാകും. നിങ്ങൾക്കും പഠാർത്ഥികൾക്കും ഒരുമിച്ച് ഏത് പാഠം പഠിക്കാമെന്ന് അറിയിപ്പു നൽകേണ്ടതുണ്ട്.

ഓൺലൈൻ ഫോർമാറ്റിൽ (Zoom, Teams, അല്ലെങ്കിൽ മറ്റു) നിങ്ങൾ ക്വിസുകൾക്കായി ബ്രേക്ക് ഔട്ട് റൂമുകൾ ഉണ്ടാക്കാനും, പഠിക്കാൻ തയ്യാറെടുക്കുന്നതിനായി മന്ററിങ്ങ് ചെയ്യാനും കഴിയും. തുടർന്ന് ക്വിസുകൾക്കായി വിദ്യാർത്ഥികളെ വെച്ചു സ്‌നേഹപൂർവ്വം ക്ഷണിച്ച് 'issues' ആയി അർപ്പിക്കാൻ പറയാം. നിങ്ങൾക്ക് ആഗ്രഹമെങ്കിൽ, അസൈൻമെന്റുകൾക്കും കൂടി ഒരേപോലെ ഓപ്പണായി സഹകരിക്കാൻ സാധിക്കും.

കൂടുതൽ സ്വകാര്യമായ ഫോർമാറ്റ് തിരഞ്ഞെടുക്കുന്നതാണെങ്കിൽ, ഓരോ പാഠവും വിദ്യാർത്ഥികൾ തങ്ങളുടെ സ്വകാര്യ GitHub റിപ്പോയാക്കി ഫോർക്ക് ചെയ്തു, നിങ്ങൾക്ക് ആക്‌സസ് നൽകും എന്നും ആവശ്യപ്പെടാം. അങ്ങനെ അവർ സ്വകാര്യമായി ക്വിസുകളും അസൈൻമെന്റുകളും പൂർത്തിയാക്കി, issues വഴി ക്ലാസ് റിപോയിലേക്ക് സമർപ്പിക്കാൻ കഴിയും.

ഓൺലൈൻ ക്ലാസ് ഫോർമാറ്റിൽ ഇത് നടത്താനുള്ള നിരവധി മാർഗ്ഗങ്ങളുണ്ട്. നിങ്ങള്ക്ക് ഏറ്റവും നല്ലത് എന്താണെന്ന് ദയവായി അറിയിക്കൂ!

## നിങ്ങളുടെ അഭിപ്രായങ്ങൾ ഞങ്ങളോട് പങ്കുവെക്കൂ!

ഈ പാഠ്യപദ്ധതി നിങ്ങൾക്കും നിങ്ങളുടെ വിദ്യാർത്ഥികൾക്കും എളുപ്പത്തിൽ ചേരുവാൻ ഞങ്ങൾ ആഗ്രഹിക്കുന്നു. ദയവായി [പ്രതിക്രിയ](https://forms.microsoft.com/Pages/ResponsePage.aspx?id=v4j5cvGGr0GRqy180BHbR2humCsRZhxNuI79cm6n0hRUQzRVVU9VVlU5UlFLWTRLWlkyQUxORTg5WS4u) നൽകുക.
