# Contributing

ഈ പ്രോജക്ട് സംഭാവനകളെയും നിർദ്ദേശങ്ങളെയും സ്വാഗതം ചെയ്യുന്നു. അധികാരമുള്ള ഒരു സംഭാവനാ ലൈസൻസ് കരാർ (CLA) നു നിങ്ങൾ സമ്മതിച്ചിരിക്കണം എന്നും നിങ്ങൾക്ക് ആ അവകാശം ഉണ്ടെന്നും യഥാർത്ഥത്തിൽ ഞങ്ങൾക്കു നിങ്ങളുടെ സംഭാവന ഉപയോഗിക്കാനുള്ള അവകാശങ്ങൾ നൽകാറുണ്ടെന്ന് പ്രഖ്യാപിക്കുന്ന ഒരു CLA-വിനോട് അംഗീകാരം നൽകേണ്ടിവരും. വിശദാംശങ്ങൾക്ക്  https://cla.microsoft.com സന്ദർശിക്കുക.

> പ്രധാനമായത്: ഈ റിപ്പോസിറ്ററിയിലുള്ള എഴുത്ത് വിവർത്തനം ചെയ്യുമ്പോൾ, ദയവായി യന്ത്രം വിവർത്തനം ഉപയോഗിക്കരുത്. സമൂഹത്തിലൂടെ വിവർത്തനങ്ങൾ പരിശോധിക്കും, അതിനാൽ നിങ്ങളുടെ പ്രാവീണ്യമുള്ള ഭാഷകളിലായിട്ടു മാത്രമേ വിവർത്തനങ്ങൾക്ക് സന്നദ്ധത നൽകൂ.

നീங்கள் ഒരു പുൾ റിക്വസ്റ്റ് സമർപ്പിക്കുമ്പോൾ, CLA-ബോട്ട് ഓട്ടോമാറ്റിക്കായി നിങ്ങൾക്ക് CLA നൽകേണ്ടതുണ്ടോ എന്ന് നിർണയിച്ച് PR-നെ അനുയോജ്യമായി അലങ്കരിക്കും (ഉദാ: ലേബൽ, അഭിപ്രായം). ബോട്ടിന്റെ നിർദ്ദേശങ്ങൾ അനുസരിക്കുക. CLA ഉപയോഗിക്കുന്ന എല്ലാ റിപ്പോസിറ്ററികളിലേക്കും ഇതു ഒരുമാത്രമായാണ് ചെയ്യേണ്ടത്.

ഈ പ്രോജക്ട് [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) സ്വീകരിച്ചിരിക്കുന്നു. കൂടുതൽ വിവരങ്ങൾക്ക് [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) വായിക്കുക അല്ലെങ്കിൽ അധിക ചോദ്യങ്ങൾക്കോ അഭിപ്രായങ്ങൾക്കോ [opencode@microsoft.com](mailto:opencode@microsoft.com) ബന്ധപ്പെടുക.
