# വർഗ്ഗീകരണ ഫലങ്ങൾക്ക് പ്രതികരിക്കുക

## നിർദ്ദേശങ്ങൾ

നിങ്ങളുടെ ഉപകരണം ചിത്രങ്ങൾ വർഗ്ഗീകരിച്ചിട്ടുണ്ട്, കൂടാതെ പ്രവചനങ്ങളുടെ മൂല്യങ്ങളും ഉണ്ടായിരിക്കുന്നു. ഉപകരണം ഈ വിവരങ്ങൾ ഉപയോഗിച്ച് ഒന്നെങ്കിലും ചെയ്യാൻ കഴിയാം - ഇത് മറ്റൊരു സംവിധാനങ്ങൾ പ്രോസസ്സ് ചെയ്യുന്നതിനായി IoT ഹബിലേക്ക് അയയ്ക്കാം, അല്ലെങ്കിൽ പഴം പകുത്തിട്ടില്ലാത്തപ്പോൾ പ്രകാശിക്കാൻ LED പോലുള്ള ആക്ടുവേറ്റർ നിയന്ത്രിക്കാം.

നിങ്ങളുടെ ഉപകരണത്തിൽ നിങ്ങളുടെ ഇഷ്ടാനുസൃതമായ രീതിയിൽ പ്രതികരിക്കാൻ കോഡ് ചേർക്കുക - IoT ഹബിലേക്ക് ഡാറ്റ അയയ്ക്കുക, ആക്ടുവേറ്റർ നിയന്ത്രിക്കുക, അല്ലെങ്കിൽ ഇരു പ്രവർത്തനങ്ങളും സംയോജിപ്പിച്ച്, Serverless കോഡ് ഉപയോഗിച്ച് പഴം പഴുകയാണോ എന്ന് നിർണ്ണയിച്ച് ആക്ടുവേറ്റർ നിയന്ത്രിക്കാൻ ഒരു കമാൻഡ് തിരിച്ച് അയയ്ക്കുക.
