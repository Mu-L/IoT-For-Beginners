# എഡ്ജിൽ മറ്റ് സേവനങ്ങൾ പ്രവർത്തിപ്പിക്കുക

## നിർദ്ദേശങ്ങൾ

എഡ്ജിൽ പ്രവർത്തിപ്പിക്കാൻ സാധിക്കുന്നത് ഇമേജ് ക്ലാസിഫയർമാർ മാത്രം അല്ല, കണ്ടെയ്‌നറിൽ പാക്കേജ് ചെയ്യാൻ സാധിക്കുന്ന ഏതുതരം പ്രോഗ്രാമും ഒരു IoT എഡ്ജ് ഡിവൈസിലേക്ക് ഡിപ്ലോയുചെയ്യാൻ കഴിയും. നിങ്ങൾ മുമ്പത്തെ പാഠങ്ങളിൽ നിർമ്മിച്ച ട്രിഗറുകൾ പോലുള്ള Azure Functions ആപ്സ് സർവറ്ലസ്സ് കോഡായി പ്രവർത്തിക്കുകയും കണ്ടെയ്‌നറുകളിൽ പ്രവർത്തിക്കുകയും അതുവഴി IoT എഡ്ജിൽ പ്രവർത്തിക്കുകയുമാണ്.

മുമ്പത്തെ പാഠങ്ങളിൽ ഒരെണ്ണം തിരഞ്ഞെടുക്കുക, Azure Functions ആപ് ഒരു IoT എഡ്ജ് കണ്ടെയ്‌നറിൽ പ്രവർത്തിപ്പിക്കാൻ ശ്രമിക്കുക. വ്യത്യസ്തമായ Functions ആപ് പ്രോജക്റ്റ് ഉപയോഗിച്ച് ഇത് എങ്ങനെ ചെയ്യാമെന്ന് കാണിക്കുന്ന മാർഗ്ഗനിർദേശം [Tutorial: Deploy Azure Functions as IoT Edge modules on Microsoft docs](https://docs.microsoft.com/azure/iot-edge/tutorial-deploy-function?WT.mc_id=academic-17441-jabenn&view=iotedge-2020-11) ല് ലഭ്യമാണ്.
