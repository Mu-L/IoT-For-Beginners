# ഒരു പഴം ഗുണനിലവാരം കണ്ടെത്തുന്ന ഉപകരണം നിർമ്മിക്കുക

## നിർദ്ദേശങ്ങൾ

പഴത്തിന്റെ ഗുണനിലവാരം കണ്ടെത്തുന്ന ഉപകരണം നിർമ്മിക്കുക!

ഇല്ലാതെ വരുംവരെ നിങ്ങൾ പഠിച്ചിട്ടുള്ള എല്ലാ കാര്യങ്ങളും ഉപയോഗിച്ച് പ്രോട്ടോട്ടൈപ്പ് പഴം ഗുണനിലവാരം കണ്ടെത്തുന്ന ഉപകരണം നിർമ്മിക്കുക. സമീപതയെ അടിസ്ഥാനമാക്കി ചിത്രം വർഗ്ഗീകരണം പ്രേരിപ്പിക്കുക, അജ്ഞാതമേഖലയിൽ പ്രവർത്തിക്കുന്ന AI മോഡൽ ഉപയോഗിച്ച്, വർഗ്ഗീകരണ ഫലം സംഭരണത്തിൽ സൂക്ഷിക്കുക, കൂടാതെ പഴത്തിന്റെ പാകം നിലയെ അടിസ്ഥാനമാക്കി ഒരു LED നിയന്ത്രിക്കുക.

ഇതിനായി ഇതുവരെ പഠിച്ച എല്ലാ പാഠങ്ങളിൽ എഴുതിയ കോഡ് ഉപയോഗിച്ച് ഇത് ഒന്നിച്ച് കൂട്ടി രൂപപ്പെടുത്താനാകും.

## റൂബ്രിക്

| മാനദണ്ഡം | ഉദാത്തമായത് | യതാർത്ഥമായത് | മെച്ചപ്പെടുത്തേണ്ടിവരുന്നു |
| -------- | --------- | -------- | ----------------- |
| എല്ലാ സേവനങ്ങളും ക്രമീകരിക്കുക | IoT ഹബ്, Azure functions അപ്ലിക്കേഷൻ, Azure storage സജ്ജമാക്കാൻ കഴിഞ്ഞു | IoT ഹബ് സജ്ജീകരിക്കാൻ കഴിഞ്ഞു, പക്ഷേ Azure functions ആപ്പ് അല്ലെങ്കിൽ Azure storage സജ്ജീകരിക്കാൻ കഴിഞ്ഞില്ല | യാതൊരു ഇന്റർനെറ്റ് IoT സേവനങ്ങളും സജ്ജമാക്കാൻ കഴിഞ്ഞില്ല |
| സമീപത നിരീക്ഷിച്ച് ഒരു വസ്തു നിർദ്ദേശിച്ച ദൂരത്തിൽ കഴിയുന്ന പക്ഷം IoT ഹബിലേക്ക് ഡാറ്റ അയച്ച്, കമാൻഡ് വഴി ക്യാമറ യാത്രാക്കുറിപ്പിക്കുക | ഒരു വസ്തു അടുത്ത് ഉണ്ടെങ്കിൽ ദൂരം അളച്ച് IoT ഹബിലേക്ക് സന്ദേശം അയച്ചു, ക്യാമറ പ്രവർത്തിപ്പിക്കാൻ കമാൻഡ് അയയ്ക്കാൻ കഴിഞ്ഞു | സമീപത അളച്ച് IoT ഹബിലേക്ക് അയച്ചു, പക്ഷേ ക്യാമറയെ കമാൻഡ് അയയ്ക്കാൻ കഴിഞ്ഞില്ല | ദൂരം അളച്ച് IoT ഹബിലേക്ക് സന്ദേശം അയയ്ക്കാനോ കമാൻഡ് പ്രേരിപ്പിക്കാനോ കഴിഞ്ഞില്ല |
| ചിത്രം പകര്‍ത്തി, അതിനെ വർഗ്ഗീകരിച്ച് ഫലം IoT ഹബിലേക്ക് അയയ്‌ക്കുക | ചിത്രം പകര്‍ത്തി, അജ്ഞാതമേഖല ഉപകരണത്തില്‍ ഉപയോഗിച്ച് വർഗ്ഗീകരിച്ച് ഫലം IoT ഹബിലേക്ക് അയയ്ക്കാൻ കഴിഞ്ഞു | ചിത്രം വർഗ്ഗീകരിക്കാനായി ശ്രമിച്ചു, പക്ഷേ അജ്ഞാതമേഖല ഉപകരണത്തിൽ അല്ല, അല്ലെങ്കിൽ ഫലം IoT ഹബിലേക്ക് അയയ്ക്കാൻ കഴിഞ്ഞില്ല | ചിത്രം വർഗ്ഗീകരിക്കാൻ കഴിഞ്ഞില്ല |
| വർഗ്ഗീകരണ ഫലങ്ങളുടെ അടിസ്ഥാനത്തിൽ കമാൻഡ് വഴി ഡിവൈസിൽ LED ഓണോ ഓഫോ ആക്കുക | പഴം പകുത്തുകൽ അല്ലെങ്കിൽ പാകമല്ലെങ്കിൽ കമാൻഡ് വഴി LED ഓണാക്കാൻ കഴിഞ്ഞു | കമാൻഡ് ഡിവൈസിലേക്ക് അയച്ചെങ്കിലും LED നിയന്ത്രിക്കാൻ കഴിഞ്ഞില്ല | LED നിയന്ത്രിക്കാനായി കമാൻഡ് അയക്കാൻ കഴിഞ്ഞില്ല |
