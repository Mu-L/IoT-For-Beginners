# ಶಿಕ್ಷಕರಿಗಾಗಿ

ನೀವು ನಿಮ್ಮ ತರಗತಿಯಲ್ಲಿ ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ಬಳಸಲು ಇಚ್ಛಿಸುತ್ತೀರೀ? ದಯವಿಟ್ಟು ಸ್ವತಃ ಅನುಭವಿಸಿ!

ನಿಜಕ್ಕೆ, ನೀವು GitHub ಗ್ರಹಣದಲ್ಲಿ GitHub Classroom ಬಳಸಿ ಇದನ್ನು ಬಳಸಬಹುದು.

ಅದಕ್ಕಾಗಿ, ಈ ರೆಪೊವನ್ನು ಫೋರ್ಕ್ ಮಾಡಿ. ಪ್ರತಿ ಪಾಠಕ್ಕೆ ಬೇರೆ ರೆಪೊ ಸೃಷ್ಟಿಸುವ ಅಗತ್ಯವಿದೆ, ಆದ್ದರಿಂದ ಪ್ರತಿ ಫೋಲ್ಡರ್ ಅನ್ನು ಪ್ರತ್ಯೇಕ ರೆಪೊಗೆ ತೆಗೆದುಹಾಕಬೇಕು. ಆ ಮೂಲಕ, [GitHub Classroom](https://classroom.github.com/classrooms) ಪ್ರತಿ ಪಾಠವನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಹಿಡುಕೊಳ್ಳಬಹುದು.

ಈ [ಸಮಗ್ರ ಸೂಚನೆಗಳು](https://github.blog/2020-03-18-set-up-your-digital-classroom-with-github-classroom/) ನಿಮ್ಮ ತರಗತಿಯನ್ನು ಹೇಗೆ ಸಿದ್ಧಮಾಡಿಕೊಳ್ಳುವುದು ಎಂಬ ಆಲೋಚನೆಯನ್ನು ನೀಡುತ್ತವೆ.

## ಶಿಫಾರಸು ಮಾಡಿದ ಅಧ್ಯಯನ ಮಾದರಿ

ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ಬೋಧಿಸುವಾಗ ಶಿಫಾರಸು ಮಾಡಲಾದ ಅಧ್ಯಯನ ಮಾದರಿಯನ್ನು ನಮ್ಮ [ಶಿಫಾರಸು ಮಾಡಿದ ಅಧ್ಯಯನ ಮಾದರಿ ಮಾರ್ಗದರ್ಶಿಕೆ](recommended-learning-model.md) ನಲ್ಲಿ ಹೆಚ್ಚು ಓದಲು পারবেন.

## ರೆಪೊವನ್ನು ಇದೇನೂ ಬದಲಾಯಿಸದೆ ಬಳಸುವುದು

ನೀವು GitHub Classroom ಅನ್ನು ಬಳಸದೆ, ಇದಾಗಿರುವಂತೆ ಈ ರೆಪೊವನ್ನು ಬಳಸಲು ಇಚ್ಛಿಸಿದರೆ, ಅದು ಕೂಡ ಸಾಧ್ಯ. ನೀವು ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಯಾವ ಪಾಠವನ್ನು ಒಟ್ಟಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸಬೇಕೆಂದು ಸಂವಹನ ಮಾಡಬೇಕು.

ಆನ್ಲೈನ್ ಮಾದರಿಯಲ್ಲಿ (Zoom, Teams ಅಥವಾ ಇತರ) ನೀವು ಪ್ರಶ್ನೋತ್ತರಗಳಿಗಾಗಿ ಬ್ರೇಕ್‌ಅ웃್ ರೂಮ್ಗಳನ್ನು ರೂಪಿಸಿ, ವಿದ್ಯಾರ್ಥಿಗಳನ್ನು ಕಲಿಕಾ ಸಿದ್ಧತೆಗೆ ಸಹಾಯ ಮಾಡಬಹುದು. ನಂತರ ವಿದ್ಯಾರ್ಥಿಗಳನ್ನು ಪ್ರಶ್ನೊತ್ತರಗಳಿಗೆ ക്ഷಣಿಸಿ, ಅವರ ಉತ್ತರಗಳನ್ನು ನಿಗದಿತ ಸಮಯದಲ್ಲಿ 'ಇಶ್ಯೂಗಳ' ರೂಪದಲ್ಲಿ ಸಲ್ಲಿಸಲು ಹೇಳಬಹುದು. ವಿವರಣೆಗಳಲ್ಲಿ ಕೂಡ ಇದ್ದಂತೆ, ನೀವು ವಿದ್ಯಾರ್ಥಿಗಳು ಸಹಯೋಗದೊಂದಿಗೆ ಹೊರಗೆ ಕೆಲಸ ಮಾಡಲು ಇಚ್ಛಿಸಿದರೆ, ಅದಕ್ಕೆ ಇದೇ ರೀತಿಯಾಗಿ ಮಾಡಬಹುದು.

ನೀವು ಖಾಸಗಿ ಸ್ವರೂಪವನ್ನು ಇಚ್ಛಿಸಿದರೆ, ನಿಮ್ಮ ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪಠ್ಯಕ್ರಮವನ್ನು ಪ್ರತಿ ಪಾಠಕ್ಕೆ ಪ್ರತ್ಯೇಕವಾಗಿ ಅವರ ಖಾಸಗಿ GitHub ರೆಪೊಗಳು ಆಗಿ ಫೋರ್ಕ್ ಮಾಡಲು ಕೇಳಿ, ಕೊನೆಯಲ್ಲಿ ನಿಮಗೆ ಪ್ರವೇಶ ತಿಳಿಸಿ. ನಂತರ ಅವರು ಪ್ರಶ್ನೋತ್ತರಗಳು ಮತ್ತು ಹ dávώνas ಸಲ್ಲಿಸಬಹುದು ಖಾಸಗಿ ರೀತಿಯಲ್ಲಿ ಮತ್ತು ನಿಮ್ಮ ತರಗತಿ ರೆಪೊದಲ್ಲಿ ಇಶ್ಯೂಗಳ ಮೂಲಕ ನಿಮಗೆ ಕಳುಹಿಸಬಹುದು.

ಆನ್ಲೈನ್ ತರಗತಿ ಸ್ವರೂಪದಲ್ಲಿ ಇದನ್ನು ಕೆಲಸ ಮಾಡಿಸಲು ಅನೇಕ ಮಾರ್ಗಗಳಿವೆ. ನಿಮ್ಮ ಕೈಗೆ ಯಾವುದು ಸೂಕ್ತವಾಗಿದೆ ಎಂದು ನಮಗೆ ತಿಳಿಸಿ!

## ದಯವಿಟ್ಟು ನಿಮ್ಮ ಅಭಿಪ್ರಾಯ ನೀಡಿರಿ!

ನಾವು ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ನಿಮ್ಮ ಮತ್ತು ನಿಮ್ಮ ವಿದ್ಯಾರ್ಥಿಗಳಿಗಾಗಿ ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಮಾಡಲು ಬಯಸುತ್ತೇವೆ. ದಯವಿಟ್ಟು ನಮಗೆ [ಪ್ರತಿಕ್ರಿಯೆ](https://forms.microsoft.com/Pages/ResponsePage.aspx?id=v4j5cvGGr0GRqy180BHbR2humCsRZhxNuI79cm6n0hRUQzRVVU9VVlU5UlFLWTRLWlkyQUxORTg5WS4u) ನೀಡಿ.
