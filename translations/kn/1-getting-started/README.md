# আইওটি সাথে ಪ್ರಾರಂಭ

ಈ ಪಾಠಕ್ರಮದ ವಿಭಾಗದಲ್ಲಿ, ನೀವು ಇಂಟರ್ನೆಟ್ ಆಫ್ ಥಿಂಗ್ಸ್ (IoT) ಪರಿಚಯವಾಗುತ್ತೀರಿ ಮತ್ತು ಮೇಘದೊಂದಿಗೆ ಸಂಪರ್ಕಿಸುವ ನಿಮ್ಮ ಮೊದಲ 'ಹೆಲೋ ವರ್ಲ್ಡ್' ಐಓಟಿ ಯೋಜನೆಯನ್ನು ನಿರ್ಮಿಸುವ ಸಾಮಾನ್ಯ ಕಲ್ಪನೆಗಳನ್ನು ಕಲಿಯುತ್ತೀರಿ. ಈ ಯೋಜನೆ ಒಂದು ರಾತ್ರಿ ಲೈಟ್ ಆಗಿದ್ದು, ಸೆನ್ಸರ್ ಮೂಲಕ ಅಳತೆ ಮಾಡಲಾದ ಬೆಳಕು ಮಟ್ಟಗಳು ಕಡಿಮೆಯಾಗುವಾಗ ಬೆಳಗುತ್ತದೆ.

![ಬೆಳಕು ಮಟ್ಟ ಬದಲಿಸುವಂತೆ ಆನ್ ಮತ್ತು ಆಫ್ ಆಗುತ್ತಿರುವ WIO ಗೆ ಸಂಪರ್ಕಿಸಿದ ಎಲ್‌ಇಡಿ](../../../images/wio-running-assignment-1-1.gif)
