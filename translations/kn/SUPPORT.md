# ಬೆಂಬಲ

## ಸಮಸ್ಯೆಗಳನ್ನು файл ಮಾಡುವ ವಿಧಾನ ಮತ್ತು ಸಹಾಯ ಪಡೆಯುವುದು  

ಈ ಯೋಜನೆ ದೋಷಗಳು ಮತ್ತು ವೈಶಿಷ್ಟ್ಯ ವಿನಂತಿಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು GitHub Issues ಅನ್ನು ಬಳಸುತ್ತದೆ. ನಕಲು ತಪ್ಪಿಸಲು ದಯವಿಟ್ಟು ಹೊಸ ಸಮಸ್ಯೆಗಳನ್ನು файл ಮಾಡುವ ಮುನ್ನ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸಮಸ್ಯೆಗಳನ್ನು ಹುಡುಕಿ. ಹೊಸ ಸಮಸ್ಯೆಗಳಿಗಾಗಿ, ನಿಮ್ಮ ದೋಷ ಅಥವಾ ವೈಶಿಷ್ಟ್ಯ ವಿನಂತಿಯನ್ನು ಹೊಸ ಸಮಸ್ಯೆಯಾಗಿ файл ಮಾಡಿ.

ಈ ಯೋಜನೆಯನ್ನು ಬಳಸುವ ಸಂಬಂಧ ಸಹಾಯ ಮತ್ತು ಪ್ರಶ್ನೆಗಳಿಗಾಗಿ, ದಯವಿಟ್ಟು ಈ ರೆಪೊದಲ್ಲಿ ಸಮಸ್ಯೆಯನ್ನು ಎತ್ತಿ ನಮ್ಮನ್ನು ಸಂಪರ್ಕಿಸಿ.
