# ಚಿತ್ರಗಳು

[icons](../../../images/icons) ಫೋಲ್ಡರ್‌ನಲ್ಲಿರುವ ಚಿತ್ರಗಳು [Noun Project](https://thenounproject.com) ನಿಂದ ಇವೆ ಮತ್ತು ಅವುಗಳಿಗಾಗಿ ಸ್ವೀಕಾರ ಸೂಚನೆ ಅಗತ್ಯವಿದೆ. ಪ್ರತಿ ಚಿತ್ರವು ಅಗತ್ಯವಿರುವ ಸ್ವೀಕಾರ ಸೂಚನೆಯನ್ನು ಸೂಚಿಸುತ್ತದೆ. ದೃಶ್ಯವನ್ನು ಸಮಾನವಾಗಿ ಇರಿಸಲು ಯಾವುದೇ ನೋಂದಿಗೆ ಆ ಚಿತ್ರಗಳನ್ನು ಬಳಸಬೇಕು.

[Diagrams.sketch](../../../images/Diagrams.sketch) ಫೈಲ್ ಒಂದು [Sketch](https://www.sketch.com) ದಾಖಲೆ ಆಗಿದ್ದು, ಪಾಠಗಳು ಮತ್ತು ಪ್ರಾಜೆಕ್ಟ್ಗಳಲ್ಲಿ ಬಳಸಲಾಗುವ ಎಲ್ಲಾ ಡಯಾಗ್ರಾಮ್‌ಗಳನ್ನು ಒಳಗೊಂಡಿದೆ. ನೀವು ಅನುವಾದದೊಡನೆ ಇದ್ದರೆ, ದಯವಿಟ್ಟು ಈ ಡಾಕ್ಯುಮೆಂಟನ್ನು ಪ್ರತಿಲಿಪಿ ಮಾಡಿ ಮತ್ತು ನವೀಕರಿಸುವ ಮೂಲಕ ಅಥವಾ ನೀವು ಅನುವಾದಿಸಬೇಕಾಗಿರುವ ಪದಗಳ ಪಟ್ಟಿಯನ್ನು GitHub ಸಮಸ್ಯೆಯನ್ನು ರೂಪಿಸುವ ಮೂಲಕ ಅನುವಾದಿಸಿ.

ಮತ್ತಿ ಚಿತ್ರಗಳನ್ನು ಪಾಠಗಳು ಮತ್ತು ಪ್ರಾಜೆಕ್ಟ್‌ಗಳಲ್ಲೆಲ್ಲ ಬಳಸಲಾಗುತ್ತದೆ.
