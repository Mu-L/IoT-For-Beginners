# ವರ್ಗೀಕರಣ ಫಲಿತಾಂಶಗಳಿಗೆ ಪ್ರತಿಕ್ರಿಯಿಸು

## ಸೂಚನೆಗಳು

ನಿಮ್ಮ ಸಾಧನವು ಚಿತ್ರಗಳನ್ನು ವರ್ಗೀಕರಿಸಿದೆ ಮತ್ತು ಭವಿಷ್ಯವಾಣಿಗಳ ಮೌಲ್ಯಗಳನ್ನು ಹೊಂದಿದೆ. ನಿಮ್ಮ ಸಾಧನವು ಈ ಮಾಹಿತಿಯನ್ನು ಬಳಸಿ ಏನಾದರೂ ಮಾಡಬಹುದು - ಅದನ್ನು ಇತರ ವ್ಯವಸ್ಥೆಗಳ ಪ್ರಕ್ರಿಯೆಗಾಗಿ IoT ಹಬ್‌ಗೆ ಕಳುಹಿಸಬಹುದು, ಅಥವಾ ಹಣ್ಣು ಪಾಕದಿಲ್ಲದಾಗ ಬೆಳಕಿಗಾಗಿ LED ಅನ್ನು ನಿಯಂತ್ರಿಸುವ ಒಂದು ತಂತ್ರಾಂಶವನ್ನು ಹುದ್ದೆಗೊಳಿಸಬಹುದು.

ನಿಮ್ಮ ಆಯ್ಕೆಯಾದ ರೀತಿಯಲ್ಲಿ ಪ್ರತಿಕ್ರಿಯಿಸುವುದಕ್ಕಾಗಿ ನಿಮ್ಮ ಸಾಧನಕ್ಕೆ ಕೋಡ್ ಸೇರಿಸಿ - ಅಥವಾ ಡೇಟಾವನ್ನು IoT ಹಬ್‌ಗೆ ಕಳುಹಿಸಿ, ಅಥವಾ ಒಂದು ಅಕ್ಟಿಯೇಟರನ್ನು ನಿಯಂತ್ರಿಸಿ, ಅಥವಾ ಎರಡನ್ನೂ ಸಂಯೋಜಿಸಿ ಹಾಗೂ ಹಣ್ಣು ಪಾಕವಾಗಿದೆಯೇ ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ನಿರ್ಧರಿಸುವ ಕೆಲವು ಸರ್ವರ್‌ರಹಿತ ಕೋಡ್ ಸಹಿತ IoT ಹಬ್‌ಗೆ ಡೇಟಾವನ್ನು ಕಳುಹಿಸಿ ಮತ್ತು ಆಕ್ಟಿಯೇಟರ್ ನಿಯಂತ್ರಣಕ್ಕೆ ಆದೇಶವನ್ನು ಹಿಂದಿರುಗಿಸಿ.
