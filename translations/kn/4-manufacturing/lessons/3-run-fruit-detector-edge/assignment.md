# ಎಡ್ಜ್‌ನಲ್ಲಿ ಇತರ ಸೇವೆಗಳನ್ನು ನಿರ್ವಹಿಸಿ

## ಸೂಚನೆಗಳು

ಎಡ್ಜ್‌ನಲ್ಲಿ ನಿರ್ವಹಿಸಬಹುದಾದದ್ದು ಕೇವಲ ಚಿತ್ರ ವರ್ಗೀಕರಣಗಳಷ್ಟೇ ಅಲ್ಲ, ಯಾವುದೇ ಕಂಟೈನರ್‌ನಲ್ಲಿ ಸಾಧಿಸಿಕೊಂಡುಬರುವ ಏನಾದರೂ ಐಒಟಿ ಎಡ್ಜ್ ಸಾಧನಕ್ಕೆ ನಿಯೋಜಿಸಬಹುದಾಗಿದೆ. ಮುಂಚಿನ ಪಾಠಗಳಲ್ಲಿ ನೀವು ಸೃಷ್ಟಿಸಿದ ಟ್ರಿಗ್ಗರ್‌ಗಳಂತಹ ಅಜುರ್ ಫಂಕ್ಷನ್‌ಗಳಾಗಿ ನಿರ್ವಹಿಸುವ ಸರ್ವರ್‌ಲೇಸ್ ಕೋಡ್‌ಗಳನ್ನು ಕಂಟೈನರ್‌ಗಳಲ್ಲಿ ಕಾರ್ಯಗತಗೊಳಿಸಬಹುದು, ಹಾಗಾಗಿ ಅವು ಐಒಟಿ ಎಡ್ಜ್‌ನಲ್ಲಿ ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತವೆ.

ಹಿಂದಿನ ಪಾಠಗಳಲ್ಲಿ ಒಂದನ್ನು ಆಯ್ಕೆಮಾಡಿ ಅಜುರ್ ಫಂಕ್ಷನ್ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಐಒಟಿ ಎಡ್ಜ್ ಕಂಟೈನರ್‌ನಲ್ಲಿ ಚಾಲನೆ ಮಾಡಲು ಪ್ರಯತ್ನಿಸಿ. ಬೇರೆ ಫಂಕ್ಷನ್ ಅಪ್ಲಿಕೇಶನ್ ಪ್ರಾಜೆಕ್ಟ್ ಬಳಸಿಕೊಂಡು ಇದನ್ನು ಹೇಗೆ ಮಾಡುವುದೆಂದು ತೋರಿಸುವ ಮಾರ್ಗದರ್ಶಿಕೆಯನ್ನು ನೀವು [Tutorial: Deploy Azure Functions as IoT Edge modules on Microsoft docs](https://docs.microsoft.com/azure/iot-edge/tutorial-deploy-function?WT.mc_id=academic-17441-jabenn&view=iotedge-2020-11) ನಲ್ಲಿ ಹುಡುಕಬಹುದು.
