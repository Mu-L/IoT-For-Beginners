# ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ನಿಯೋಜಿಸಿ

## ಸೂಚನೆಗಳು

ನೀವು ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಜಗತ್ತಿನೊಂದಿಗೆ ಹಂಚಿಕೊಳ್ಳಲು ಅನೇಕ ಮಾರ್ಗಗಳಿವೆ, ಅವುಗಳಲ್ಲಿ GitHub ಪುಟಗಳನ್ನು ಬಳಸುವುದು ಅಥವಾ ಅನೇಕ ಸೇವಾ ಒದಗಿಸುವವರಲ್ಲಿ ಒಂದುವನ್ನು ಬಳಸುವುದು ಸೇರಿದೆ. ಅದನ್ನು ಮಾಡಲು ಅತ್ಯುತ್ತಮ ಮಾರ್ಗವೆಂದರೆ Azure Static Web Apps ಅನ್ನು ಬಳಸುವುದು. ಈ ಕಾರ್ಯದಲ್ಲಿ, ನಿಮ್ಮ ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ರಚಿಸಿ ಮತ್ತು [ಈ ಸೂಚನೆಗಳನ್ನು](https://github.com/Azure/static-web-apps-cli) ಅನುಸರಿಸಿ ಅಥವಾ [ಈ ವೀಡಿಯோகಳನ್ನು](https://www.youtube.com/watch?v=ADVGIXciYn8&list=PLlrxD0HtieHgMPeBaDQFx9yNuFxx6S1VG&index=3) ವೀಕ್ಷಿಸಿ ಮುಗಿಸಲು ಬದಲಾಗಿಸಿ ಕ್ಲೌಡ್‌ನಲ್ಲಿ ನಿಯೋಜಿಸಿ.
Azure Static Web Apps ಬಳಕೆಯ ಒಂದು ಪ್ರಯೋಜನವೆಂದರೆ ನೀವು ಪೋರ್ಟಲ್‌ನಲ್ಲಿಯೇ ಯಾವುದೇ API ಕೀಲಿಗಳನ್ನು ಮರೆಮಾಡಬಹುದು, ಆದ್ದರಿಂದ ನಿಮ್ಮ subscriptionKey ಅನ್ನು ವರ್ಚುವಲ್ ರೂಪದಲ್ಲಿ ಪರಿವರ್ತಿಸಿ ಮತ್ತು ಅದನ್ನು ಕ್ಲೌಡ್‌ನಲ್ಲಿ ವಹಿಸಿ.
