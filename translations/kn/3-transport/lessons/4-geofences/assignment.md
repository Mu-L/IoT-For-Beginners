# Twilio ಬಳಸಿ ಸೂಚನೆಗಳನ್ನು ಕಳುಹಿಸಿ

## ಸೂಚನೆಗಳು

ನೀವು ಇದುವರೆಗೆ ನಿಮ್ಮ ಕೋಡ್‌ನಲ್ಲಿ ಕೇವಲ ಜಿಯೋಫೆನ್ಸ್‌ಗೆ ದೂರವನ್ನು ಲಾಗ್ ಮಾಡಿದ್ದೀರಿ. ಈ ಕಾರ್ಯದಲ್ಲಿ, ಜಿಪಿಎಸ್ ನ ನಿರ್ವಚನಗಳು ಜಿಯೋಫೆನ್ಸ್ ಒಳಗಾಗುವಾಗ ಅಥವಾ ಹೊರಗಾಗುವಾಗ ಪಠ್ಯ ಸಂದೇಶ ಅಥವಾ ಇಮೇಲ್ ಮೂಲಕ ಸೂಚನೆ ಸೇರಿಸಬೇಕಾಗುತ್ತದೆ.

ಆಜುರ್ ಫಂಕ್ಷನ್‌ಗಳು ಹಲವಾರು ಬಿಂದಿಂಗ್ಸ್ ಆಯ್ಕೆಗಳನ್ನು ಹೊಂದಿವೆ, Third-party ಸೇವೆಗಳಾದ Twilio, ಒಂದು ಸಂವಹನ ವೇದಿಕೆ, ಸೇರಿದಂತೆ.

* [Twilio.com](https://www.twilio.com) ನಲ್ಲಿ ಉಚಿತ ಖಾತೆಗೆ ಸಹಿ ಕೊಡಿ
* Twilio SMS ಗೆ ಆಜುರ್ ಫಂಕ್ಷನ್ಸ್ ಬಿಂದಿಂಗ್ ಮಾಡುವುದು ಕುರಿತಂತೆ ಡಾಕ್ಯುಮೆಂಟೇಷನ್ ನೋಡಿ [Microsoft docs Twilio binding for Azure Functions page](https://docs.microsoft.com/azure/azure-functions/functions-bindings-twilio?WT.mc_id=academic-17441-jabenn&tabs=python)
* ಇಮೇಲ್ ಕಳುಹಿಸಲು Twilio SendGrid ಗೆ ಆಜುರ್ ಫಂಕ್ಷನ್ಸ್ ಬಿಂದಿಂಗ್ ಅನ್ನು ನೋಡಿ [Microsoft docs Azure Functions SendGrid bindings page](https://docs.microsoft.com/azure/azure-functions/functions-bindings-sendgrid?WT.mc_id=academic-17441-jabenn&tabs=python)
* ನಿಮ್ಮ ಫಂಕ್ಷನ್ ಅಪ್ಲಿಕೇಶನ್‌ಗೆ ಬಿಂದಿಂಗನ್ನು ಸೇರಿಸಿ, ಜಿಪಿಎಸ್ ನಿರ್ವಚನಗಳು ಜಿಯೋಫೆನ್ಸ್ ಒಳಗೆ ಅಥವಾ ಹೊರಗೆ ಇದ್ದಾಗ, ಎರಡೂ ಅಲ್ಲದೆ ಒಂದರಿಗೊಂದನ್ನು ಸೂಚನೆ ನೀಡಲು

| -------- | --------- | -------- | ----------------- |
