## ಅಧ್ಯಯನ ವಿಧಾನಶಾಸ್ತ್ರ

ನಾವು ಈ ಪಾಠಕ್ರಮವನ್ನು ರಚಿಸುವಾಗ ಎರಡು ಅಧ್ಯಯನ ತತ್ವಗಳನ್ನು ಆರಿಸಿಕೊಂಡಿದ್ದೇವೆ: ಪ್ರಾಜೆಕ್ಟ್ ಆಧಾರಿತವಾಗಿರುವುದು ಮತ್ತು ನಿಯಮಿತ ಪ್ರಶ್ನಾಪತ್ರಿಕೆಗಳನ್ನು ಒಳಗೊಂಡಿರುವುದು. ಸರಣಿಯ ಕೊನೆಯಲ್ಲಿ, ವಿದ್ಯಾರ್ಥಿಗಳು ಒಂದು ಸಸ್ಯಗಳುಂಡುವ ಮತ್ತು ನೀರು ನೀಡುವ ವ್ಯವಸ್ಥೆ, ವಾಹನ ತ್ರ್ಯಾಕರ್, ಅಡುಗೆಗಾಗಿನ ಸ್ವಯಂ ಚಾಲಿತ ಟೈಮರ್, ಮತ್ತು ಆಹಾರದ ತಪಾಸಣೆಗಾಗಿ ಸ್ಮಾರ್ಟ್ ಫ್ಯಾಕ್ಟರಿ ವ್ಯವಸ್ಥೆ ನಿರ್ಮಿಸುತ್ತಾರೆ ಮತ್ತು IoT ಮೂಲಭೂತಗಳನ್ನು ಕಲಿಯುತ್ತಾರೆ.

ಪ್ರಾಜೆಕ್ಟ್‌ಗಳಿಗೆ ಹೊಂದಿಕೆ ಮಾಡಿರುವ ವಿಷಯವು ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಹೆಚ್ಚು ಆಸಕ್ತಿದಾಯಕವಾಗಿದ್ದು, ಕಲಿಕೆಯ ಗುರಿಗಳನ್ನು ಸುಗಮಗೊಳಿಸುತ್ತದೆ.

ಮತುಗಾಲಿನ ಪ್ರಶ್ನಾಪತ್ರಿಕೆ ತರಗತಿಗೆ ಮುಂಚಿತವಾಗಿ ವಿದ್ಯಾರ್ಥಿಯ ಗಮನವನ್ನು ಸೆಳೆಯಲು, ತರಗತಿಯ ನಂತರದ ಪ್ರಶ್ನಾಪತ್ರಿಕೆ ವಿಷಯದ ಲಕ್ಷ್ಯವನ್ನು ಉಳಿಸುವುದಕ್ಕೆ ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಈ ಪಾಠಕ್ರಮವು ಬಗೆಯಲ್ಲದ ಮತ್ತು ಖುಷಿಯಾಯಕವಾಗಿದ್ದು, ಸಂಪೂರ್ಣ ಅಥವಾ ಭಾಗವಾಗಿ ತೆಗೆದುಕೊಳ್ಳಬಹುದು. ಪ್ರಾಜೆಕ್ಟುಗಳು ಸಣ್ಣದಾಗಿ ಪ್ರಾರಂಭವಾಗುತ್ತವೆ ಮತ್ತು 12 ವಾರಗಳ ವೇಳಾಪಟ್ಟಿಯ ಕೊನೆಯಲ್ಲಿ ಹೆಚ್ಚು ಸಂಕೀರ್ಣವಾಗುತ್ತವೆ.

ಪ್ರತಿ ಪ್ರಾಜೆಕ್ಟ್ ನೈಜ ಪ್ರಪಂಚದಲ್ಲಿ ವಿದ್ಯಾರ್ಥಿಗಳು ಮತ್ತು ಹವ್ಯಾಸಿಗಳಿಗಾಗಿ ಲಭ್ಯವಿರುವ ಹಾರ್ಡ್ವೇರ್ ಆಧರಿಸಿರುವುದು. ಪ್ರತಿ ಪ್ರಾಜೆಕ್ಟ್ ನಿರ್ದಿಷ್ಟ ಪ್ರಾಜೆಕ್ಟ್ ಕ್ಷೇತ್ರ ತಿಳಿಸಿಕೊಂಡುಕೊಳ್ಳಲಾಗಿ, ಸಂಬಂಧಿತ ಹಿನ್ನೆಲೆ ಜ್ಞಾನವನ್ನು ನೀಡುತ್ತದೆ. ಯಶಸ್ವಿ ಡೆವಲಪರ್ ಆಗಲು ನೀವು ಪರಿಹರಿಸುತ್ತಿರುವ ಕ್ಷೇತ್ರವನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಬೇಕಾಗುತ್ತದೆ. ಈ ಹಿನ್ನೆಲೆ ಜ್ಞಾನ ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಅವರ IoT ಪರಿಹಾರಗಳನ್ನು ಮತ್ತು ಕಲಿಕೆಗಳನ್ನು ನೈಜ ಪ್ರಪಂಚದ ಸಮಸ್ಯೆಗಳ ಸನ್ನಿವೇಶದಲ್ಲಿ ಯೋಚಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. ವಿದ್ಯಾರ್ಥಿಗಳು ಅವರು ನಿರ್ಮಿಸುತ್ತಿರುವ ಪರಿಹಾರಗಳ 'ಯಾಕೆ' ಸಹ ಅನ್ವೇಷಣೆ ಮಾಡುತ್ತಾರೆ ಮತ್ತು ಅಂತಿಮ ಬಳಕೆದಾರರನ್ನು ಮೆಚ್ಚುತ್ತದೆ.

## ಹಾರ್ಡ್ವೇರ್
ನಾವು ವೈಯಕ್ತಿಕ ಇಚ್ಛೆ, ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷಾ ಜ್ಞಾನ ಅಥವಾ ಇಚ್ಛೆಗಳು, ಕಲಿಕೆಯ ಉದ್ದೇಶಗಳು ಮತ್ತು ಲಭ್ಯತೆಯ ಮೇಲೆ ಅವಲಂಬಿಸಿ ಯೋಜನೆಗಳಿಗೆ ಬಳಸಲು ಎರಡು ಆಯ್ಕೆಯ ಐಒಟಿ ಹಾರ್ಡ್ವೇರ್ ಹೊಂದಿದ್ದೇವೆ. hardware ಲಭ್ಯವಿಲ್ಲದವರಿಗೆ ಅಥವಾ ಖರೀದಿಗೆ ಎದುರು ನೋಡುವ ಮೊದಲು ಇನ್ನಷ್ಟು ಕಲಿಯಲು ಬಯಸುವವರಿಗೆ ನಾವು 'ವರ್ಚುವಲ್ ಹಾರ್ಡ್ವೇರ್' ಆವೃತ್ತಿಯನ್ನು ಕೂಡ ಒದಗಿಸಿದ್ದೇವೆ. ನೀವು ಇನ್ನಷ್ಟು ಓದಬಹುದು ಮತ್ತು [hardware ಪುಟದಲ್ಲಿ](./hardware.md) 'ಶಾಪಿಂಗ್ ಪಟ್ಟಿ' ಅನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು, ಇದರಲ್ಲಿ Seeed Studio ಅವರಿಂದ ಪೂರ್ಣ ಕಿಟ್ಟುಗಳನ್ನು ಕೊಳ್ಳಲು ಲಿಂಕ್ಸ್ ಸಹ ಸೇರಿವೆ.

> 💁 ನಮ್ಮ [ನೈತಿಕ ಮಾರ್ಗಸೂಚಿ](CODE_OF_CONDUCT.md), [ಸಹಕಾರ](CONTRIBUTING.md), ಮತ್ತು [ಅನುವಾದ](TRANSLATIONS.md) ಮಾರ್ಗಸೂಚಿಗಳನ್ನು ಪರಿಶೀಲಿಸಿ. ನಿಮ್ಮ ರಚನಾತ್ಮಕ ಪ್ರತಿಕ್ರಿಯೆಗಳಿಗೆ ನಾವು ಸ್ವಾಗತ ಮಾಡುತ್ತೇವೆ!
>
> 🔧 ಸಮಸ್ಯೆಗಳು ಎದುರಾದರೆ? ಸಾಮಾನ್ಯ ಸಮಸ್ಯೆಗಳ ಪರಿಹಾರಕ್ಕಾಗಿ ನಮ್ಮ [ಟ್ರಬಲ್‌ಶೂಟಿಂಗ್ ಮಾರ್ಗದರ್ಶಿ](TROUBLESHOOTING.md) ಅನ್ನು ನೋಡಿ.
