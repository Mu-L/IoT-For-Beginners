# Jupyter ನೋಟುಬುಕ್ ಬಳಸಿ GDD ಡೇಟಾವನ್ನು ದೃಶ್ಯೀಕರಿಸಿ

## ಸೂಚನೆಗಳು

ಈ ಪಾಠದಲ್ಲಿ ನೀವು IoT ಸೆನ್ಸಾರ್ ಬಳಸಿ GDD ಡೇಟಾವನ್ನು ಸಂಗ್ರಹಿಸಿದ್ದೀರಿ. ಉತ್ತಮ GDD ಡೇಟಾವನ್ನು ಪಡೆಯಲು, ನೀವು ಅನೇಕ ದಿನಗಳ ಡೇಟಾವನ್ನು ಸಂಗ್ರಹಿಸಬೇಕಾಗುತ್ತದೆ. ತಾಪಮಾನ ಡೇಟಾವನ್ನು ದೃಶ್ಯೀಕರಿಸಲು ಮತ್ತು GDD ಅನ್ನು ಲೆಕ್ಕಿಸಲು [Jupyter Notebooks](https://jupyter.org) ತರುತ್ತದೆ.

ಕೆಲವು ದಿನಗಳ ಡೇಟಾವನ್ನು ಸಂಗ್ರಹಿಸುವುದರಿಂದ ಪ್ರಾರಂಭಿಸಿ. ನಿಮ್ಮ IoT ಉಪಕರಣ ಚಲಿಸುತ್ತಿರುವಾಗ ನಿಮ್ಮ ಸರ್ವರ್ ಕೋಡ್ ನಿರಂತರವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸಬೇಕು, ಇದಕ್ಕಾಗಿ ನಿಮ್ಮ ಪವರ್ ಮ್ಯಾನೇಜ್ಮೆಂಟ್ ಸೆಟ್ಟಿಂಗ್ಗಳನ್ನು ಸರಿಹೊಂದಿಸುವುದು ಅಥವಾ [ಈ ಕಾರ್ಯಕ್ಷಮತೆಯ Python ಸ್ಕ್ರಿಪ್ಟ್](https://github.com/jaqsparow/keep-system-active) ಅನ್ನು ಚಲಾಯಿಸಬಹುದು.

ತಾಪಮಾನ ಡೇಟಾವನ್ನು ಪಡೆದ ನಂತರ, ನೀವು ಈ ರೆಪೋನಲ್ಲಿ ಇರುವ Jupyter ನೋಟುಬುಕ್ ಬಳಸಿ ಅದನ್ನು ದೃಶ್ಯೀಕರಿಸಿ ಹಾಗೂ GDD ಲೆಕ್ಕಿಸಬಹುದು. Jupyter ನೋಟುಬುಕ್‌ಗಳು ಕೋಡ್ ಮತ್ತು ಸೂಚನೆಗಳನ್ನು *ಸೆಲ್ಸ್* ಎನ್ನುವ ಬ್ಲಾಕ್‌ಗಳಲ್ಲಿ ಮಿಶ್ರಣ ಮಾಡುತ್ತವೆ, ಹೆಚ್ಚಿನ ಬಾರಿ Python ಕೋಡ್. ನೀವು ಸೂಚನೆಗಳನ್ನು ಓದಿ ಪ್ರತಿ ಕೋಡ್ ಬ್ಲಾಕ್ ಅನ್ನು ಕ್ರಮವಾಗಿ ಕಾರ್ಯಗತಗೊಳಿಸಬಹುದು. ಕೋಡ್ ಅನ್ನು גם ಸಂಪಾದಿಸಬಹುದು. ಉದಾಹರಣೆಗೆ, ಈ ನೋಟುಬುಕ್‌ನಲ್ಲಿ ನಿಮ್ಮ ಸಸ್ಯದ GDD ಲೆಕ್ಕಿಸಲು ಬಳಸುವ ಮೂಲ ತಾಪಮಾನವನ್ನು ಸಂಪಾದಿಸಬಹುದು.

1. `gdd-calculation` ಎಂಬ ಫೋಲ್ಡರ್ ರಚಿಸಿ

1. [gdd.ipynb](./code-notebook/gdd.ipynb) ಫೈಲ್ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ ಅದನ್ನು `gdd-calculation` ಫೋಲ್ಡರ್‌ಗೆ ನಕಲಿಸಿ

1. MQTT ಸರ್ವರ್ ಸೃಷ್ಟಿಸಿದ `temperature.csv` ಫೈಲ್ ನಕಲಿಸಿ

1. `gdd-calculation` ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಹೊಸ Python ವರ್ಚುವಲ್ ಎನ್ವಿರಾನ್‌ಮೆಂಟ್ ರಚಿಸಿ

1. Jupyter ನೋಟುಬುಕ್‌ಗಾಗಿ ಮತ್ತು ಡೇಟಾವನ್ನು ನಿರ್ವಹಿಸಲು, ಮಾರ್ಪಡಿ ಮಾಡಲು ಬೇಕಾದ ಲೈಬ್ರರಿಗಳನ್ನು ಸರಿಯಾಗಿ ಸೂಚಿಸಿರುವ ಪಿಪ್ ಪ್ಯಾಕೇಜುಗಳನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ:

    ```sh
    pip install --upgrade pip
    pip install pandas
    pip install matplotlib
    pip install jupyter
    ```

1. Jupyter ನಲ್ಲಿ ನೋಟುಬುಕ್ ಅನ್ನು ಚಾಲನೆ ಮಾಡಿ:

    ```sh
    jupyter notebook gdd.ipynb
    ```

    Jupyter ಪ್ರಾರಂಭವಾಗುತ್ತದೆ ಮತ್ತು ನಿಮ್ಮ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ನೋಟುಬುಕ್ ತೆರೆಯುತ್ತದೆ. ನೋಟುಬುಕ್‌ನ ಸೂಚನೆಗಳನ್ನು ಅನುಸರಿಸಿ ತಾಪಮಾನಗಳನ್ನು ದೃಶ್ಯೀಕರಿಸಿ, ಫಳವಂತೆ ಜವಾಬ್ದಾರಿಯಾಗಿರುವ ಹೆಚ್ಚುವರಿ ಗಿಡದ ದಿನಗಳನ್ನು ಲೆಕ್ಕಿಸಿ.
