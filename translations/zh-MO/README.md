# IoT 初學者課程大綱

微軟 Azure 雲端倡導者很高興提供一套為期 12 週、包含 24 節課程的物聯網基礎課程。每節課包括課前及課後測驗、完成課程的書面說明、解決方案、作業等。我們採用以專案為基礎的教學方法，讓你在實作中學習，這是新技能能夠「紮根」的有效方式。

這些專案涵蓋食物從農場到餐桌的整個旅程。包括農業、物流、製造、零售到消費者，全部都是物聯網裝置常見的產業領域。

## 教學法

我們在建構本課程時選擇了兩個教學原則：確保以專案為基底，並包含頻繁的小測驗。完成此系列課程後，學生將自建植物監控與澆水系統、車輛追蹤器、智慧工廠以追蹤與檢查食品，以及語音控制烹調計時器，並學會物聯網基礎，包括如何編寫裝置程式碼、連接雲端、分析遙測資料及在邊緣執行 AI。

確保課程內容符合專案導向，更能讓學生投入學習，提升概念記憶度。

此外，課前的低壓力小測幫助學生日前建立學習主題的動機，課後小測則加深記憶。本課程設計彈性且富趣味，可全程或部分修習。專案循序漸進，從簡入深，於 12 週週期結束時達到較高複雜度。

每個專案皆以學生與業餘愛好者可取得的真實硬體為基礎。專案涵蓋特定領域，提供相關背景知識。成為成功開發者需理解問題領域，此背景知識有助學生思考物聯網解決方案與學習內容，並理解他們所面對的真實問題。學生學會所建置解決方案的緣由，並對最終使用者產生認識。

## 硬體
