# IoT for Beginners - A Curriculum

Azure Cloud Advocates for Microsoft happy to offer 12-week, 24-lesson curriculum wey dey teach IoT basics. Every lesson get pre- and post-lesson quizzes, written instructions to finish the lesson, solution, assignment and more. Our project-based style dey allow you learn as you dey build, na proven way for new skills to 'stick'.

The projects cover di journey of food from farm go table. This one include farming, logistics, manufacturing, retail and consumer - all na popular industry areas for IoT devices.

## Pedagogy

We choose two pedagoji principles while we dey build dis curriculum: to make sure say e get project-based style and e get frequent quizzes. By di time you finish this series, you go don build plant monitoring and watering system, vehicle tracker, smart factory wey dey track and check food, and voice-controlled cooking timer, and go don learn the basics of Internet of Things including how to write device code, connect to cloud, analyze telemetry and run AI for edge.

By making sure say di content match projects, di process go dey more interesting for students and e go help better retention of concepts.

Also, low-stakes quiz before class dey set the student mindset to learn topic, while second quiz after class dey make sure say dem remember well. This curriculum design na flexible and fun and you fit take all or part of am. The projects start small and get more hard as you reach di end of di 12 week cycle.

Each project base on real-world hardware wey students and hobbyists fit get. Each project dey look into im specific domain, providing correct background knowledge. To be successful developer e good to sabi di domain wey you dey solve problem; providing dis background knowledge dey help students reason about their IoT solutions and learnings with respect to real-world problem wey dem fit face as IoT developer. Students go learn the 'why' behind the solutions wey dem dey build, and go appreciate di end user well well.

## Each lesson get:

- sketchnote
- optional supplemental video
- pre-lesson warmup quiz
- written lesson
- for project-based lessons, step-by-step guides on how to build the project
- knowledge checks
- challenge
- supplemental reading
- assignment
- [post-lesson quiz](https://ff-quizzes.netlify.app/en/)

> **Note about quizzes**: All quizzes dey for quiz-app folder, we get 48 quizzes total with three questions each. Dem link dem from inside lessons but quiz app fit run locally or e fit deploy for Azure; follow instruction for `quiz-app` folder. Dem dey slowly dey localize dem.
